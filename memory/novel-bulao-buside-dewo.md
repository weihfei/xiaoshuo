## 十二、小说团队创作流程（已固化）

### 团队成员列表

| 角色 | 职能 |
|------|--------|------|
| **大纲规划师** | 制定整体结构、章节规划 |
| **世界观架构师** | 构建世界观、设定、法则 |
| **角色塑造师** | 创建人物、性格、对话风格 |
| **情节设计师** | 设计具体情节、冲突、高潮 |
| **初稿写手** | 根据大纲写正文 |
| **编辑/校对** | 错别字、语病、逻辑漏洞检查 |
| **修订者** | 深度逻辑审查 |
| **润色师** | 提升文笔、统一风格 |
| **类型 specialist** | 专业审查 |
| **版权顾问** | 查重、侵权检测 |

### 小说创作标准顺序

```
1. 世界观架构师 → 构建世界观、设定、法则
     ↓
2. 角色塑造师 → 创建人物、性格、对话风格
     ↓
3. 大纲规划师 → 制定整体结构、章节规划
     ↓
4. 情节设计师 → 设计具体情节、冲突、高潮
     ↓
5. 初稿写手 → 根据大纲写正文
     ↓
 ┌───────────────────────────┘
 ↓
6. 编辑/校对 + 7. 修订者（可并行）
 ↓
8. 润色师 + 9. 类型 specialist（可并行）
 ↓
10. 版权顾问
 ↓
 ✅ 完成
```

**注意：世界观架构师和角色塑造师必须在大纲规划师前面，先有世界和人物，才能规划故事走向。**

### 审查流程（已写章节）

| 阶段 | 团队 | 任务 |
|------|------|------|
| 1 | **编辑/校对** | 错别字、语病检查 |
| 2 | **修订者** | 深度逻辑审查 |
| 3 | **润色师** | 文笔提升（可选）
| 4 | **类型 specialist** | 专业审查（可选）

### 重要原则

- ❌ 禁止使用非小说团队角色
- ✅ 创作新章节：按顺序 1→5
- ✅ 审查已有章节：6→7 并行
- ✅ 每次小说任务第一反应：调用团队，而不是自己硬扛

## 十三、重要创作经验（来自老板指导）

### 核心原则（老板强调）：
1. **设定是铁律，永远不会改变**
2. **小说内容必须符合设定，而不是反过来改设定**
3. 如果设定与情节冲突 → **重写情节，绝不改设定**

### 需要避免的问题：
- 不要习惯性改角色设定迁就情节
- 章节内部逻辑必须贯通，不能前面说A后面说B
- 需要检查：章节内部逻辑 + 文笔一致性 + 设定一致性
