# 第15章 Context - 熵魔教徒，山谷伏击

## 1. 本章结尾关键情节
队伍进入碎星谷后，果然遭遇了早就埋伏好的熵魔教徒袭击。对方由投靠熵魔的内核级高手黑风老魔带队，一两百人将队伍团团包围。玄真长老对阵黑风老魔，王翔在旁边观察，发现了黑风老魔左肋下旧伤漏洞，抓住黑风老魔出大招后的间隙，一剑刺中要害，帮助玄真长老击杀了黑风老魔。剩余的熵魔教徒要么被击杀，要么投降后被清理。战斗结束，队伍虽然获胜，但都受了点轻伤，王翔被黑风老魔掌风扫中，受了点内伤。众人经过短暂休息，继续朝着碎星谷深处前进，寻找上古神殿入口。

## 2. 人物当前状态（位置、伤势、修为）
- **王翔**：碎星谷中部的山道上，前进中，被黑风老魔掌风扫中受了点内伤，没有生命危险，比特级巅峰修为，能正常战斗。
- **玄真长老**：队伍领头位置，无大碍，内核级三段修为，击杀黑风老魔后消耗了不少算力，状态基本保持完好。
- **赵凯**：队伍前排，胳膊被砍了一刀，伤口不大但被熵魔煞气感染，已经清理处理过，不影响行动，比特级巅峰修为。
- **钱磊**：队伍右侧断后，大腿受了刀伤，已经处理包扎，比特级巅峰修为。
- **孙博**：队伍前方开路，无伤势，只是有点脱力，休息后已经恢复，比特级巅峰修为。
- **所有队员**：都保持了战斗力，虽然有伤亡但全员都在，士气还在，继续向碎星谷深处前进。
- **熵魔教黑风老魔**：已经被击杀，一代魔头陨落。

## 3. 关键物品位置
- 排序剑：王翔手中，刚刚刺中黑风老魔，沾了黑血，已经擦拭干净。
- 云纹软甲：王翔穿在身上，帮他挡了一部分掌力，不然内伤会更重。
- 黑色镰刀（黑风老魔兵器）：掉落在伏击战场，被玄真长老收走，打算回去之后熔掉重新炼制。
- 空间稳定符：每人都还剩一张，贴身存放。
- 传讯符：王翔贴身存放，暂时还没联系叶清语，等稳定下来再报平安。
- 三枚代码钥匙：还是王翔灵魂一枚，九尾、零音各一枚，在开源盟等待大部队汇合。

## 4. 下一章开头承接点
一行人继续深入碎星谷，根据凌云长老笔记上的描述，开始寻找上古神殿入口的大致方位。王翔利用源代码视野，扫描周围的地质代码，终于找到了笔记上描述的"三块巨石拱卫"的标志性地点，拼合地图后确认入口就在这附近地下。
