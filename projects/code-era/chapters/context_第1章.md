# 《代码纪元》第1章 启动上下文

## 本章规划（来自大纲）

**第1章** ：猝死！穿越到代码世界

- 开篇：地球，996加班深夜，主角王翔改最后一个BUG，心脏骤停猝死
- 苏醒：在比特神域黑铁聚落破木屋中醒来，发现自己附身到叫"万相"的少年身上
- 震惊：弄清楚状况，发现这个世界万物皆代码，自己能直接看到底层源代码
- 结尾：原身被零号帮打成重伤，寿命（进程倒计时）只剩几个小时，主角必须想办法活下去

**预期字数**：3000-4000字
**节奏要求**：开篇快节奏，三页之内进入主题，金手指在第一章揭晓，结尾留下危机悬念（零号帮寻仇+寿命倒计时）

## 相关设定

### 主角当前状态
- 灵魂：地球全栈程序员王翔（28岁）
- 身体：比特神域黑铁聚落少年万相（18岁），低级比特战士
- 伤势：被零号帮头目打成重伤，核心代码受损，进程剩余时间不足一天
- 所处位置：黑铁聚落边缘破木屋
- 装备：一把原身留下的新手铁剑

### 世界观关键信息
- 比特神域：万物皆代码，所有东西都有底层源代码
- 普通人（原住民）：只能调用API，看不到源代码
- 主角金手指：能直接看到源代码，可以修改优化

### 需要记住的关键事实
1. 主角是穿越者，来自地球，是全栈程序员
2. 原身父母被零号帮害死，原身自己也被打死
3. 主角能看到这个世界的底层源代码，这是独一无二的金手指
4. 当前危机：重伤，寿命倒计时，零号帮可能还会来
5. 第一章结束时，主角应该已经发现金手指，并找到初步修复损伤的方法

## 前情提要
这是故事的开头，没有前情。开篇要讲清楚穿越过程和新世界设定。
