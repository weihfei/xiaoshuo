# 《代码纪元》第2章 启动上下文

## 本章规划（来自大纲）

**第2章**：动态规划退敌，收获第一桶金

- 疤脸上门挑战，疤脸是比特战士9段，比张三强很多
- 疤脸不信主角一个重伤少年能杀张三，上来就猛攻
- 主角利用动态规划思想设计战术，分步消耗，找到疤脸旧伤弱点
- 主角用排序剑刺中弱点，反杀疤脸
- 解决零号帮，夺回自家矿地，获得第一桶金
- 结尾：黑铁聚落村民震惊，主角开始正式在比特站稳脚跟
- 埋下伏笔：疤脸身上有一块奇怪的代码碎片，主角收起来

**预期字数**：约1万字
**节奏要求**：本章是第一个小高潮，战斗要精彩，爽点要足，反杀恶霸，主角正式立足

## 相关设定

### 主角当前状态
- 姓名：万相（王翔，地球灵魂穿越）
- 境界：比特战士1段（但能量效率提升10倍，实际战斗力相当于比特战士5段以上）
- 装备：排序剑（绿名，动态伤害，排序斩特效）
- 伤势：基本修复，剩余战斗力九成以上
- 位置：黑铁聚落边缘破木屋门口
- 刚刚杀了张三，打跑李四，正在等疤脸上门

### 疤脸设定
- 身份：零号帮头目，黑铁聚落一霸
- 境界：比特战士9段
- 装备：精铁开山刀（蓝色品质）
- 弱点：早年和人争夺矿地被砍过一刀，胸口核心代码有旧伤，存在内存泄漏漏洞
- 性格：残暴，轻敌，看不起主角，认为他就是个濒死小鬼

### 需要记住的关键事实
1. 主角能看源代码，找弱点，改代码
2. 主角能量效率是原住民的十倍，同等境界实力碾压
3. 原身父母被疤脸逼死，此仇必须报
4. 本章结束必须解决疤脸，夺回矿地，主角在黑铁聚落站稳脚跟
5. 要留下一个小伏笔：疤脸身上奇怪的代码碎片

## 前情提要
第一章主角穿越苏醒，发现金手指，修复身体，重构出排序剑，杀了张三，打跑李四，疤脸怒气冲冲找上门来复仇。
