# 《代码纪元》第3章 启动上下文

## 本章规划（来自大纲）

**第3章**：优化基础功法，准备出发

- 疤脸死后，黑铁聚落村民纷纷来祝贺，村长过来确认现状，承认矿地归主角
- 主角有空整理基础功法，把《黑铁基础诀》从头到尾重构一遍，效率提升十倍
- 村长告诉主角，下个月二进制宗要在青岩城招比特学徒，问他要不要去报名
- 主角决定去，村民你一言我一语凑了点报名费，主角感激
- 整理行装，把矿地暂时委托给村里打理，出发前往青岩城
- 结尾：踏上路途，第一次走出黑铁聚落，见识更广阔的比特神域

**预期字数**：约1万字
**节奏要求**：战前休整，提升实力，铺垫后续，节奏舒缓但要有进展，引出下一段剧情（学徒考核）

## 相关设定

### 主角当前状态
- 姓名：万相（王翔）
- 境界：比特战士二段（刚升的，能量效率是原住民十倍，实际战斗力≈比特战士七段）
- 装备：排序剑（绿）+ 疤脸的精铁胸甲（蓝）+ 精铁开山刀（蓝，备用）
- 物品：12块低纯度比特结晶，68铜板，未知代码碎片
- 位置：黑铁聚落破木屋
- 刚杀死疤脸，解决零号帮，村民拥戴

### 需要记住的关键事实
1. 主角会重构代码，能优化原住民的烂代码功法
2. 重构后《黑铁基础诀》效率提升十倍，修炼速度更快
3. 二进制宗招生是主角走出新手村的第一个机会，必须去
4. 矿地委托给村里，主角没有后顾之忧，可以安心出发
5. 本章结尾主角踏上旅途，去青岩城参加考核

## 前情提要
第二章主角反杀疤脸，解决零号帮，报了血海深仇，夺回矿地，收获第一桶金，升到比特二段。
