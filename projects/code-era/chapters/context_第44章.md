## 写作要求

1. **保持人物一致性：**
   - 万相：理性、果断，说话简洁，喜欢用程序员术语
   - 叶清语：温柔、细心，话不多但总能说到点子上，喜欢关心主角
   - 九尾：灵动、俏皮，喜欢说话，偶尔调侃，战斗起来很凶猛
   - 零音：话少、冷静，原则性强，对敌人狠，对主角温柔

2. **女主互动要求：**
   - 本章至少安排两个自然的女主互动细节
   - 已经在情节设计中安排了：
     1. 石门打开时，叶清语抓主角衣袖，主角护着她
     2. 零音运行调试传承压力大，主角悄悄分担算力，零音感激眼神
   - 保持自然，不要刻意撒糖，符合人物性格

3. **主线伏笔要求：**
   - 本章要点明内鬼身份，确认北冥毒就是放熵魔的人
   - 提及三把钥匙，呼应主线集齐钥匙打开神殿的设定
   - 距离README只有一步之遥，保持读者期待感

4. **反派仇恨要求：**
   - 点明北冥毒过往恶行，尤其是害死零音师父这件事
   - 拉满仇恨值，为后续决战做铺垫

5. **金手指运用：**
   - 主角运用进程杀死（SIGKILL）这一操作系统知识杀死熵魔残魂
   - 解释要通俗易懂，符合这个世界的说法，不要太生硬
   - 体现降维打击，原住民不懂这个，主角会，所以能赢

6. **字数要求：**
   - 控制在9000-11000字之间，平均每章约1万字

## 需要特别注意的设定

1. **创造者神殿四层结构：** 对应软件开发四个阶段：
   - 第一层：编译试炼 ✅ 已过
   - 第二层：调试试炼 ✅ 已过
   - 第三层：链接试炼 ✅ 已过
   - 第四层：运行试炼 ➡️ 本章
   这个对应关系要明确，让读者能理解

2. **力量层级：** 本章主角突破到内核二级初期，这个是合理进度：
   - 第一层编译战后：内核一级初期
   - 第二层调试战后：内核一级中期
   - 第三层链接战后：内核一级巅峰
   - 第四层运行战后：内核二级初期
   节奏正确，每过一关提升一级，符合预期

3. **三把钥匙：**
   - 第一把：主角灵魂中的根域名碎片（来自穿越时携带）
   - 第二把：九尾身上的狐族传承钥匙（上古狐族守护）
   - 第三把：零音手中的杀毒权限钥匙（创造者留下的杀毒进程权限）
   本章开门需要三把钥匙依次激活，这个过程要写清楚

4. **熵魔等级：** 本章出现的是熵魔残魂，但已经达到内核级，比第三层的熵魔领主（字节级巅峰）强，符合层层递进的节奏
