3. **第三部分：死战与击杀**（约3000字）

## 必须遵守的规则

1. **设定是铁律：** 严格按照世界观和人物设定来，不准改设定迁就情节
2. **战斗逻辑：** 主角赢在找到漏洞利用域名劫持，不是硬实力超越，不能写成主角现在就能正面硬刚域主，逻辑要通顺
3. **女主分工：** 叶清语医疗后备，九尾堵路骚扰，零音正面牵制，主角主攻，分工明确，不要抢戏
4. **爽文节奏：** 战斗节奏要快，高潮迭起，劫持升级 → 一剑击杀 → 收获暴涨，一波接一波
5. **程序员设定：** 域名劫持漏洞解释要自然，符合安全常识，不要太晦涩也不要错得离谱
6. **一致性：** 承接第47章结尾，四人都有消耗，主角赢在智取不是硬拼，符合之前情节
7. **字数控制：** 控制在1万字左右，不要太少，也不要太啰嗦
