1. **第一部分：入城与欢迎**（约2000字）

2. **第二部分：论功行赏与整编**（约3000字）
   - 回到城主府，论功行赏，赵凯钱磊孙博各升其职，三位女主各有赏赐
   - 开始整编，老弟兄编三个主力师团，新来五万人编五个新编师团，打乱原有编制
   - 黑石不服，想要保留原有编制让儿子当师团长，当众质问主角
   - 主角拿出通敌证据，零音当场斩杀黑石，杀鸡儆猴
   - 当众宣布不收内奸，所有人震惊，整编顺利进行

3. **第三部分：突破内核级**（约3000字）
   - 众人退去，主角拿出域主核心准备闭关突破，三位女主分工护法

   - 吸收到七成，北冥毒残留神魂和熵魔病毒爆发反噬

4. **第四部分：定策与未来**（约2000字）
   - 突破消息传开，全城再次沸腾
   - 核心骨干开会讨论战略，赵凯提议快速扩张，零音提议稳扎稳打准备围剿
   - 主角综合决策：一边接收周边小城统一混沌域，一边加固城墙训练新军准备围剿
