# 《代码纪元》第4章 启动上下文

## 本章规划

**第4章**：资质测试，震惊全场

- 王翔走进测试场地，考官让他把手放在测试水晶上，激活资质
- 其他考生都在100单位以下，优秀的也才两三百
- 王翔把手放上去，测试水晶直接亮到九百多，接近顶级资质
- 考官震惊，问王翔是哪个聚落的，怎么这么高
- 主考官过来，记住了王翔名字，直接给他过第一关
- 所有考生都震惊，议论纷纷，没想到黑铁聚落出来的小鬼有这么好资质
- 有人嫉妒，有人佩服，几个大家族天才注意到了王翔
- 结尾：王翔通过第一关，等待第二关实战考核

**预期字数**：约1万字
**节奏**：小高潮，主角第一次在公众面前展示天赋，震惊全场，埋下被人盯上的伏笔

## 相关设定

### 主角当前状态
- 境界：比特战士二段，接近三段
- 装备：排序剑+精铁胸甲+开山刀备用
- 位置：青岩城二进制宗分馆演武场，资质测试室
- 刚刚叫到号，进来测试

### 测试规则
- 资质测试就是测能感知多少比特能量，单位越高资质越好
- 低于100单位：不合格，直接淘汰
- 100-300：合格，能进第二关
- 300-500：良才，重点关注
- 500以上：天才，顶级资质
