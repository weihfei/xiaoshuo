# 《代码纪元》第5章 启动上下文

## 本章规划

**第5章**：宗门功法，内存泄漏

- 进入二进制宗山门，安排新人洞府，每人发一本基础宗门功法《二进制基础功》
- 主角拿到功法，一看就发现问题：祖传老功法，到处都是内存泄漏，时间复杂度太高，效率极低
- 主角按照自己的方法重构了一遍，修复了内存泄漏，优化了算法，效率提升好几倍
- 长老过来检查新人功课，发现主角改了功法，一开始很生气，后来测试了一下，发现效率确实提升太多
- 长老震惊，回去报告宗主，宗主也很惊讶，开始注意这个天才新人
- 主角在宗门里开始出名，大家都知道有个新人天才会改功法

**预期字数**：约1万字
**节奏**：展示主角金手指对顶级功法的优化能力，引发宗门高层注意，铺垫后续地位提升
