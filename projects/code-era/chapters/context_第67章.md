# 《代码纪元》第67章 章节启动包

## 一、本章规划（来自大纲）

### 位置定位
卷三「开源革命」第七章，紧接第66章大军东进，本章就是双方前锋接触战，拉开大战序幕。

### 核心内容
- 开源盟十五万大军东进，前锋抵达青牛谷山口，这是进入东部域的必经之路，万维联盟在这里布置了前锋防线，由万维联盟猛将"快剑"公孙阔跟随叔父公孙绰，三万前锋镇守
- 零音主动请战，带领五百影杀小队深夜潜入，打掉对方外围七个哨站，烧毁对方粮草大营，制造混乱
- 天亮之后巨石率领一万前锋正面强攻，公孙绰骄傲出城让公孙阔迎战，巨石vs公孙阔，正面硬刚
- 本章结尾：巨石重伤生擒公孙阔，拿下青牛谷山口外关，公孙绰带着残兵退守内关，故意败退诱敌，万相合将计就计，大军开进青云谷，西门封五万伏兵已经张网以待，留下悬念

### 关键节点
1. **大军压境**：开源盟十五万大军东进，傍晚抵达山口外扎营，帐中议事
2. **零音请缨**：零音主动请战夜袭，五百人分七队打七个哨站，计划周密
3. **深夜破袭**：零音带队夜袭，七个哨站全部端掉，火烧粮草大营，全身而退
4. **天亮强攻**：巨石正面猛攻，公孙绰激将之下派公孙阔出城单挑
5. **猛将对决**：快剑对重斧，公孙阔快剑破不开巨石重甲，巨石越战越勇，最后一斧劈断公孙阔肩膀，生擒活捉
6. **公孙绰败退**：损失先锋主将，公孙绰放弃外关退守内关，故意放走开源盟，诱敌深入青云谷
7. **万相将计就计**：万相看破诱敌之计，照样开进，就是要打西门封一个措手不及，结尾西门封伏兵等着，悬念留给下一章

### 预期效果
- 大战开幕，前锋接触战，零音秀特种作战，巨石秀正面硬刚，两个人物都得到充分展现
- 节奏张弛有度，夜袭→单挑→占领山口→诱敌→将计就计，层层递进，结尾悬念留下
- 字数控制在1.4-1.6万字，正好

## 二、相关设定回顾

### 双方兵力布置
| 方向 | 开源盟 | 万维联盟 | 指挥官 |
|------|--------|----------|--------|
| 前锋外线 | 零音五百影杀 + 巨石一万前锋 | 公孙绰三万总兵力，公孙阔五千前锋守外关 | 零音/巨石（开源）→ 公孙绰/公孙阔（万维） |
| 后手伏击 | 万相十三万主力 | 西门封五万伏兵藏在青云谷两侧山上 |  |

### 核心人物当前状态
1. **零音**：从卧底成长为特种作战统领，冷静敏锐，急于证明自己，主动承担最危险的夜袭任务
2. **巨石**：开源盟第一猛将，皮糙肉厚力量无穷，正面硬刚从来没输过，就喜欢这种面对面硬碰硬
3. **万相**：开源盟盟主，看穿公孙绰诱敌之计，但是干脆将计就计，胆子大，胸有成竹
4. **公孙绰**：西门封手下老将，老谋深算，按照计划故意败退诱敌，一切都在西门封计划之内
5. **公孙阔**：公孙绰侄子，年轻气盛骄傲轻敌，剑法快但是骄傲，看不起巨石，所以贸然出城，中计被擒
6. **西门封**：东部域主，老谋深算，整体布局就是诱敌深入，聚而歼之，已经准备好了口袋阵就等万相进来

## 四、需要记住的关键事实清单

1. **零音五百人分七队**：五百人÷七队≈每队七十人，人数正确，不能错
2. **公孙阔必须生擒不能杀死**：因为后面还要放回去继续参与诱敌，所以不能杀死，只能生擒
3. **是公孙绰不是公孙阔**：叔叔是公孙绰，侄子是公孙阔，名字别搞混
4. **故意败退诱敌**：公孙绰不是打不过，是故意放弃外关，引开源盟进青云谷，钻进西门封的口袋
5. **万相知道有埋伏，故意进去**：万相已经看穿，但是将计就计，正好把西门封主力引出来，一次性吃掉，不怕埋伏
