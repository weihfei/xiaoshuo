# 第74章 暗潜入城 - 情节上下文

## 本章核心情节回顾

零音带领十二名情报司精锐，化妆分批次潜入Git域都城提交城：
- 日夜兼程七天抵达，发现城防比情报严三倍，决定不走城门，从废弃排水口下水道潜入
- 下水道中发现有人近期出入的痕迹，并且墙上有版本控制系统的**分支标志**，暗示组织已经分裂
- 进城后遭遇巡逻队盘查，士兵从指针书箱里搜出禁书《开源心经》，零音机智用"被人栽赃"的借口化解，成功脱身
- 在醉仙楼和联络人**回滚**接头，回滚戴着分支标志面具，透露关键情报：
  - 原核心成员**分支**三个月前被暗杀
  - 组织分裂成两派：孙器为首的**主干派**（温和潜伏） vs 合并为首的**激进派**（主张立即起义）
  - 回滚属于分支派系，但愿意安排零音和孙器见面，约定三到五天后碰头
- 零音汇总情报：闭源子提供的情报已经过时，版本控制系统内部派系斗争激烈，回滚态度暧昧疑点重重
- 零音将加密情报发回万相，留在城中等待接头，悬念留给下一章

## 当前人物状态
- **零音**：在城中客栈落脚，保持高度警惕，等待和孙器接头
- **指针、数组**等十二名情报员：分散住在不同客栈，每天醉仙楼碰头
- **回滚**：身份神秘，属于分支派系，态度暧昧，疑点重重
- **孙器**：枢密院左副使，版本控制系统核心，主干派首领，等待接头
- **合并**：激进派首领，主张立即起义，认为分支是孙器害死，扬言要夺权

## 地理和势力分布
- 提交城：Git域都城，人口八十万，城防极其严密，万维皇帝所在地
- 版本控制系统：地下抵抗组织，现已分裂，主干派vs激进派矛盾公开
- 万相三路大军：约定二十天后会师提交城下，零音必须在会师前摸清城内情况

## 本章结尾悬念
- 回滚到底是谁的人？真心想合作还是设陷阱？
- 分支真的是被孙器暗杀的吗？
- 三五天后和孙器的接头，会是一场鸿门宴吗？
- 零音能在万维皇帝的眼皮底下完成任务吗？
