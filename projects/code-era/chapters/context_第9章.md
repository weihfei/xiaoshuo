# 《代码纪元》第9章 启动上下文

## 本章规划

**第9章**：妖兽矿，发现熵魔感染

- 王翔晋升内门，接到第一个内门任务：下属一座妖兽矿，最近老是出问题，矿工莫名被感染发疯，派王翔过去看看怎么回事
- 王翔下山，来到妖兽矿，下去查看
- 发现矿洞深处出现熵魔感染，代码被熵魔侵蚀，变得混乱，所以矿工被感染发疯
- 王翔第一次看到熵魔感染，仔细研究，发现熵魔就是失控的垃圾进程，侵蚀正常代码
- 王翔清理了一小块感染区域，发现清理起来挺麻烦，熵魔会传染
- 王翔回去报告宗主，宗主承认确实有熵魔感染，最近越来越严重，问王翔有没有办法解决
- 王翔说可以试试用垃圾回收算法清理，宗主很高兴，让王翔主导清理

**预期字数**：约1万字
**节奏**：引出核心反派熵魔，展示问题，推动主线剧情，主角开始面对世界层面的问题
