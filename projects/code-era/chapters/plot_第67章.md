# 《代码纪元》第67章 青牛谷山口，前锋交锋 情节设计

## 本章总览
- **本章字数**：约1.42万字
- **核心冲突**：开源盟前锋抵达山口，夜袭破防，正面单挑，拿下外关，公孙绰诱敌，万相将计就计
- **核心人物塑造**：
  - 零音：冷静敏锐，计划周密，特种作战专家风范，不贪功，见好就收
  - 巨石：勇猛沉稳，正面硬刚，力量压垮速度，符合猛将人设
  - 公孙绰：老谋深算，故意败退，执行诱敌计划
  - 万相：胸有成竹，看穿陷阱还是进去，有魄力

---

## 情节节点一：帐中议事 零音请缨（约2000字）

### 1.1 大军扎营
- 十五万开源盟大军傍晚抵达青牛谷山口外三十里，扎下连营，黑色旌旗遮天蔽日，气氛压抑
- 万相站在沙盘前，和诸将讨论进攻方案：公孙绰三万兵马依山布防，易守难攻，正面强攻伤亡太大

### 1.2 零音主动请缨
- 话音刚落，零音从帐外走进来，主动请缨："末将愿领五百影杀小队，夜袭烧掉公孙绰粮草，打掉外围哨站，为明天正面进攻开路"
- 万相沉吟："五百人是不是太少了？粮草大营有两千守军，外围还有七个哨站"
- 零音："五百足够，影杀小队擅长潜行夜战，我已经看好地形，七个哨站分七队同时动手，速战速决，天亮前就能撤回，不会贪功深入"
- 零音画出地形：西侧乱石林可以隐蔽潜入，丑时守军防备最松懈，正好动手，计划周密，头头是道
- 诸将惊讶，没想到零音一个姑娘家，竟然对地形这么熟悉，计划这么周密
- 万相点头同意："好，就依你，我让巨石带一千人在山口外接应你，万一有事，接应你撤出来"
- 零音领命，下去准备

### 情节节点二：深夜潜行 七哨齐拔（约4000字）

### 2.1 潜行摸哨
- 入夜，零音带领五百影杀小队，换上夜行衣，顺着乱石林悄悄摸上去，每个人都屏住呼吸，脚步轻得像猫
- 零音亲自带一队，摸到最北边的哨站，哨站里守军大部分都睡了，只有几个哨兵在岗楼晃悠
- 零音安排："一队上，解决哨兵，摸进去，一刀一个，不要留活口，不要发出声音"
- 影杀队员纵身跃入围墙，匕首飞出去，哨兵瞬间被解决，整个哨站一百多守军，不到一刻钟就全部解决，没有一个跑掉
- 零音点起火把，冲天而起，这就是得手信号，几乎同时，其他六个方向也都升起了火光，七个哨站全部得手
- 零音带队，悄悄的向粮草大营摸过去，守军还不知道外面哨站已经没了，根本没有防备

### 2.2 火烧粮草
- 零音带人摸到营寨栅栏外，四个岗哨，四个哨兵在打瞌睡，影杀队员摸上去，瞬间解决
- 队员们冲进去，火油泼在粮草堆上，火折子一扔，大火瞬间冲天而起，火光染红了半边天
- 粮草大营守军这才惊醒，乱作一团，根本组织不起抵抗，零音带队砍了守将，收队撤退，不贪功，不恋战，按照预定路线撤回
- 等到公孙绰得到消息，带兵出来追杀，零音已经带着队伍走远了，只剩下一片火海和满地尸体
- 巨石在山口外接应，顺利接回零音，五百人只伤亡不到五十，完美完成任务，诸将都佩服零音，果然是名将风范

## 情节节点三：天明强攻 猛将对决（约5000字）

### 3.1 公孙绰震怒
- 天亮，公孙绰站在城头上，看着后山冲天火光，粮草被烧，气得吐血，一巴掌拍在城墙上："零音小贼，竟敢如此！"
- 副将劝："敌人刚烧了粮草，现在肯定士气高涨，我们不如坚守，等待西门城主援兵"
- 公孙绰本来就骄傲，现在被人端了粮草，颜面扫地，哪里忍得住，正好他也想激对方出来强攻，于是就说："敌人连夜袭扰，现在肯定疲惫，我们正好趁乱出击，杀他们一个措手不及，公孙阔，你带五千人马出城，给我斩了巨石，提头来见！"
- 公孙阔年轻气盛，早就按捺不住，答应一声，提着宝剑出城，直奔巨石骂阵

### 3.2 巨石出战
- 巨石听到骂阵，提着开山斧就出来了，万相给了他一万前锋，猛攻山口，巨石一马当先，走在最前面
- 两个人就在山口前的空地上对仗，公孙阔快剑，巨石重斧，一个轻灵，一个厚重，棋逢对手

### 3.3 猛将单挑
- 公孙阔上来就是成名快剑"闪电十七刺"，十七剑刺出，剑剑指向巨石要害，速度快得看不清影子
- 巨石根本不躲不闪，硬扛，铠甲坚硬，剑气砍在上面，只留下白印，根本破不了防
- 一百回合打下来，公孙阔真气消耗越来越大，速度慢下来，巨石抓住机会，一斧劈下去，正好劈中公孙阔肩膀，铠甲劈开，骨头砍断，生擒活捉
- 巨石赢了，开源盟士兵齐声呐喊，声震山谷

### 3.4 公孙绰败退
- 公孙绰看到公孙阔被擒，差点气死，但是也正好按照计划，下令放弃外关，退守内关，故意打开山口，放开源盟大军进来，引诱万相进青云谷
- 万相大军开进，拿下外关，巨石派人给万相报捷

## 情节节点四：将计就计 谷口定计（约3200字）

### 4.1 零音进言
- 零音过来见万相："公孙绰退得太快了，几乎没怎么打就放弃外关，这里面肯定有问题，我猜前面青云谷肯定有埋伏，西门封肯定在那里等着我们"
- 巨石也说："对啊，这老小子败得太容易了，肯定有鬼，我们要不要停下来，搜搜山再进去？"
- 万相笑了："我当然知道有埋伏，公孙绰就是故意诱我们进去，西门封想要把我们困在谷里一网打尽"
- 零音："那我们为什么还要进去？不如停下来，团团围住，慢慢耗他们"
- 万相："我们耗不起，我们十五万人，粮草需要从后方运过来，对方以逸待劳，拖下去对我们不利。而且，西门封把五万主力都放在青云谷埋伏，我们正好一次性把他们引出来，聚而歼之，省得我们一个个城池打，反而拖时间。他想口袋阵吃我们，我们就给他来个中心开花，正好反过来吃掉他五万主力，一步打通整个东部域"

### 4.2 下令进军
- 万相下令："全军休整一天，明天一早，开进青云谷，继续东进，直取白鹿城"
- 命令下去，开源盟大营一片欢腾，大家都知道赢了前哨战，拿下山口，士气高涨
- 零音按照万相命令，提前做好准备，抢占要点，防备伏兵突袭

### 4.3 结尾悬念
- 夕阳西下，青云谷东边二十里，西门封站在山顶，看到开源盟大营灯火，笑了："万相小儿，果然上当了，明天你全军开进谷里，我就封住谷口，滚木礌石下去，叫你十五万人全都变成鬼"
- 西门封身边副将："城主神机妙算，万相这次死定了"
- 夕阳把西门封影子拉得很长，整个青云谷像一张巨大的网，已经张开，就等万相往里钻
- 本章完，留下悬念，下一章就是伏兵四起，大战爆发
