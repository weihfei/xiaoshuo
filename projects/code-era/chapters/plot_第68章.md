# 《代码纪元》第68章 青云谷惊雷 情节设计

## 本章总览
- **本章字数**：约1.45万字
- **核心冲突**：万相将计就计开进青云谷，西门封伏兵四起，关门打狗，零音巨石守住出口，万相稳住阵脚，最终主帅碰面，留下单挑悬念
- **核心人物塑造**：
  - 万相：胸有成竹，遇乱不慌，指挥若定，符合盟主身份
  - 零音：预判准确，提前布局，守住北口，冷静指挥，符合安全总长人设
  - 巨石：正面硬刚，寸步不退，扛住公孙绰反扑，符合猛将人设
  - 西门封：老谋深算，口袋阵成型，最后亲自下场，符合反派域主人设

## 情节节点一：大军入谷 将计就计（约2000字）

### 1.1 清晨出发
- 一夜休整，第二天清晨，万相升帐点兵，十五万大军开出青牛谷山口，顺着大道开进青云谷
- 道路两边山林寂静，连鸟叫声都听不到，这种寂静反而透着诡异，前锋将领有些紧张，派人请示万相要不要停下来搜山
- 万相笑了："不用搜，西门封就是要我们停下来搜山，他就能调兵围过来，我们不给他时间，继续走，加快速度，尽快走到谷中开阔地"
- 大军继续前进，万相暗中吩咐零音和巨石："你们两个提前带本部人马，抢占南北两个出口，不管对方有多少伏兵，给我牢牢把住出口，不让他们封路，就算战死，也要守住"
- 零音巨石领命，提前带着本部人马，从侧翼绕出去，抢占预定位置

### 1.2 走入口袋
- 万相带着十万主力，一步步走进青云谷深处，走到谷中最狭窄的地方，道路两边都是高山，正好容下十万大军展开
- 万相勒住马，抬头看向两侧高山，对身边参赞说："告诉各军，保持阵型，准备战斗，西门封的伏兵，马上就要出来了"
- 参赞惊讶："盟主，您知道有埋伏还进来？"
- 万相笑："我当然知道，西门封想要口袋阵吃我们，我们就给他来个中心开花，他把主力都放出来，正好一次性解决，省得我们一个个城池打"
- 话音刚落，就听到半山腰一声梆子响，紧接着，号角声冲天而起

## 情节节点二：伏兵四起 滚木封门（约3500字）

### 2.1 伏兵发动
- 号角响过，两侧高山上，突然站起来密密麻麻的万维联军，五万人马，站满了两侧山坡，喊杀声震天动地："杀！"
- 早就准备好的滚木礌石，顺着山坡轰隆隆滚下来，砸向谷底大道上的开源盟大军
- 前面不少士兵躲闪不及，被滚木礌石砸中，人仰马翻，死伤一片
- 同时，前后两个出口，也都滚下来无数巨大树干，死死堵住出口，前后都被封死，关门打狗，口袋阵彻底成型
- 万维联盟士兵呐喊着，顺着山坡往下冲，想要把开源盟大军切成几段，一口一口吃掉
- 开源盟大军并不慌乱，因为万相早就提醒过，有埋伏，所以各军很快结成圆阵，弓箭手对着山上射箭，稳住阵脚

### 2.2 北口激战
- 北侧出口，零音已经带着一千五百特种营提前抢占了有利地形，修筑了简易工事
- 一万二千伏兵从山上冲下来，想要封住北口，把出口堵死，切断开源盟出路
- 带队的是西门封心腹大将章邯，看到零音已经提前占据有利地形，不由得惊讶："万相怎么会提前知道我们在这里埋伏？"
- 来不及多想，既然已经动手，只能强攻，章邯下令，全军冲锋，一定要拿下北口
- 零音站在壁垒后面，冷静指挥："弓箭手准备，等敌人冲到五十步再放箭，不要浪费箭枝"
- 等到冲在前面的伏兵冲到五十步，零音一声令下："放箭！"
- 万箭齐发，冲在前面的伏兵纷纷倒下，第一轮攻击就倒下一片
- 章邯不服，接连组织了五次冲锋，都被零音打回去，伤亡上千人，始终没能突破零音的防线，反而被零音 counter 一波反冲，差点把冲上来的先锋队包了饺子
- 章邯没办法，只能停下来，稳住阵脚，和零音对峙，没办法前进一步，零音牢牢守住了北口

### 2.3 南口硬刚
- 南侧出口，公孙绰带着五千残兵，也想要从外面封住南口，断了万相退路
- 巨石早就带着五千重甲步兵等着了，看到公孙绰冲过来，巨石一声大吼，提着开山斧就冲了上去："公孙绰老狗，你的侄子被我们擒了，还不快快投降！"
- 公孙绰大怒，挥刀上来和巨石打，公孙绰这边将士拼命冲锋，想要突破巨石防线
- 巨石下令："盾墙！"
- 五千重甲步兵立刻结成密集盾墙，如同钢铁长城，挡住了公孙绰一轮又一轮冲锋
- 公孙绰冲了七八次，伤亡上千人，始终没能冲破盾墙，反而被巨石亲自冲出来，砍了公孙绰身边的校尉，公孙绰只能退回去，牢牢被堵在外面，进不来
- 巨石派人给万相报信："南口稳稳的，公孙绰过不去，请盟主放心"

## 情节节点三：抢占山头 稳住阵脚（约4000字）

### 3.1 万相抢山
- 万相站在中军，看到南北出口都已经被零音巨石守住，松了一口气，对身边将领说："好了，出口堵住了，他们封不住我们退路，现在我们抢占要点"
- 青云谷谷底，有三处低矮山头，正好卡在谷中要害，谁占据谁就有优势，万相早就看好了这三处
- 万相下令："三个军团，分别抢占这三处山头，不惜一切代价，拿下来！"
- 三个军团得令，立刻朝着三个山头猛攻，山上原本只有少量万维联盟守军，哪里挡得住开源盟生力军猛攻，很快，三个山头都被拿下来
- 开源盟士兵立刻构筑工事，把火炮拉上山头，调转炮口，对着两侧山坡上的万维伏兵猛轰，一下子炸死炸伤不少伏兵，伏兵进攻势头被压了下去

### 3.2 双方对峙
- 西门封在山顶看到，不由得惊讶："万相竟然提前就让零音巨石抢占了出口，还抢占了三个山头，这家伙，怎么会什么都提前想到了？"
- 原本他算好了，伏兵四起，开源盟肯定混乱，然后他就能一口气冲下去吃掉中军，现在看来，万相根本不混乱，反而稳住了阵脚，抢占了要点
- 副将问："城主，现在怎么办？我们已经封了谷口，零音巨石堵住出口我们打不开，万相抢占了山头，我们攻不上去，要不要暂缓进攻？"
- 西门封咬咬牙："暂缓什么？我们五万伏兵在这里，万相十万大军被围在谷里，现在我们兵力比他少一点，但我们居高临下，优势在我，全力进攻，我就不信，万相能挡得住我们五万伏兵"
- 于是，西门封下令，全线猛攻，两侧伏兵一齐往下冲，想要拿下三个山头
- 万维联军冲锋，山上开源盟火炮弓箭一起招呼，冲下去的伏兵死伤惨重，根本攻不上来，尸体铺满了山坡，血流成河

### 3.3 西门封决定亲自出击
- 打了一个时辰，伏兵伤亡近万人，还是没能拿下三个山头，西门封越来越生气，他原本以为万相掉进口袋，很快就能吃掉，没想到打到现在，反而自己伤亡更大
- 西门封咬咬牙："看来万相确实有两下子，我亲自下去，带队冲锋，我就不信，我亲自出手，还拿不下他"
- 副将劝："城主千金之躯，怎么能亲自上阵？还是让我们去"
- 西门封："万相就在中军，我亲自下去，斩了万相，今天这仗就赢了，你们不用多说，我去"
- 西门封收拾一下，提着万维长剑，带着两万精锐主力，顺着山坡冲下去，直奔万相中军而来

## 情节节点四：主力合围 双雄对阵（约4000字）

### 4.1 西门封冲过来
- 西门封亲自带队，两万精锐顺着山坡冲下来，直奔万相中军，气势如虹
- 万相看到西门封亲自下来，笑了："果然出来了，我还以为他会一直在山顶坐着呢"
- 万相拔剑出鞘，指针剑斜指对面，对身边众将说："西门封亲自来了，正好，我跟他算算总账，你们稳住阵脚，我去会会他"
- 众将想要一起去，万相拦住："不用，我一个人去会他，你们稳住阵脚就行"

### 4.2 阵前对骂
- 西门封冲到中军前面，勒住马，看着万相，哈哈大笑："万相小儿，你果然还是年轻，明知道是口袋阵还要进来，今天你插翅难飞，还不快快下马投降，我饶你不死"
- 万相哈哈大笑："西门封老狗，你闭源垄断东部域五十年，吸了五十年底层修士的血，今天我就是来取你狗命，你都快入土了，还在这里摆域主威风，谁给你的胆子？"
- 西门封："我万维联盟掌控东部域五十年，法度森严，层级分明，哪像你什么开源共享，乱七八糟，人人都能乱说话，早就乱成一锅粥了，你也配跟我谈天命？"
- 万相："就是因为你闭源垄断，五十年了，东部域修炼水平一点进步都没有，多少天才埋没，多少好功法烂在仓库里，你还好意思说法度森严？今天我就要打破你这个臭规矩，让东部域所有人都能自由修炼，你，给我去死！"

### 4.3 定下单挑
- 两个人骂完，都知道口舌没用，只能动手
- 西门封："别说我以大欺小，你一个后生晚辈，敢不敢跟我单挑？赢了我，你拿走东部域，输了你就去死，怎么样？"
- 万相："有什么不敢？我早就想要跟你单挑，看看你这个老牌域主，到底有多少本事，来吧，今天就在这里，分个胜负！"
- 双方将士都停下来，纷纷后退，让出一块巨大空地，给两个人单挑
- 整个青云谷，原本喊杀声震天，现在一下子安静下来，只剩下风吹过山林的声音，所有人都屏住呼吸，看着这一场决定命运的单挑

### 4.4 本章结尾 留下悬念
- 两个人都提马上前，各自握住兵器，眼神死死锁定对方，空气仿佛凝固了
- 山风卷着血雾，吹过两个人的战袍，大战一触即发
- 本章完，留下悬念，下一章就是单挑决胜负
