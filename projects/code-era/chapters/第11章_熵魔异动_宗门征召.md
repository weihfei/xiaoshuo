# 第11章 熵魔异动，宗门征召

一夜无话。

第二天清晨，天刚蒙蒙亮，王翔就被一阵急促的钟声惊醒了。

"咚...咚...咚..."

钟声沉闷，一共响了九下，节奏和平时的晨钟完全不一样，带着一股紧迫感。王翔睁开眼睛，皱了皱眉，九下钟声，那是宗门紧急召集的信号，只有遇到重大事件的时候才会敲，怎么一大早就紧急召集了？

王翔不敢耽搁，赶紧起身，收拾整齐，佩戴好排序剑，快步向着宗主大殿的方向走去。路上已经碰到了不少内门弟子，每个人脸上都带着疑惑和惊讶，显然也都是被钟声叫过来的，大家三三两两地走着，低声议论着。

"怎么回事啊，好好的怎么突然紧急召集了？"
"不知道啊，我刚打坐入定就被吵醒了，肯定出大事了。"
"会不会是万维联盟那边又要找咱们麻烦？上次他们就说咱们占了他们的矿脉，闹了好一阵。"
"别瞎猜了，去了不就知道了，看宗主怎么说。"

王翔默不作声，跟在人群后面，心里也在琢磨，能让二进制宗敲九下紧急钟，肯定不是小事，二进制宗在青云山扎根几十万年，很少有这种级别的召集。难道真的是和别的势力起冲突了？还是说...

很快，所有人都到齐了，内门弟子将近两百人，加上七大长老，宗主玄机子已经站在大殿台阶上了，脸色凝重，眉头紧锁，看得出来，情况确实不太好。

看到人都到齐了，玄机子清了清嗓子，声音带着一丝沉重，传遍了整个广场：

"各位弟子，今天紧急召集大家，是因为刚刚收到了黑风铁矿发来的急报——黑风铁矿深处，熵魔大规模扩散了！"

"嗡——"

广场上一下子炸了锅，所有人都吃了一惊，熵魔扩散？这可是大事！黑风铁矿是二进制宗最重要的铁矿，出产大量的比特铁，是宗门打造兵器的主要来源，要是被熵魔占了，那麻烦可就大了。而且熵魔这东西，扩散起来特别快，要是不及时清理，用不了半个月，整个黑风铁矿都会被感染，到时候方圆几百里都会变成感染区，后果不堪设想。

王翔心里也是一动，黑风铁矿？他昨天刚从黑风铁矿回来啊，昨天他去那边完成采集任务，一切都还正常啊，怎么一夜之间就熵魔大规模扩散了？难道是昨天他走了之后才爆发的？

玄机子抬了抬手，广场上很快安静下来，他接着说道：

"黑风铁矿的监矿长老半个时辰前用传讯符发来了急报，说三天前矿洞深处就发现了少量熵虫，当时以为只是少量泄漏，派人清理了一下，没想到昨天晚上突然爆发，一下子涌出了成百上千只熵虫，整个最深的三号矿脉全被占了，已经有十几个矿工被感染了，监矿长老拼死守住了入口，请求宗门立刻派兵清剿！"

说到这里，玄机子的眼神扫过全场，落在了内门弟子这片：

"熵魔扩散，事关重大，必须尽快清剿，不能让它们扩散出来，否则咱们整个青云山东部都要遭殃！我决定，立刻派出内门好手，由玄真长老带队，连夜赶过去，彻底清理这次熵魔爆发！"

所有人都神色严肃，熵魔这东西，比特神域所有人都恨之入骨，那玩意儿就是BUG变成的，感染了正常代码，好好一个人就会变成没有理智的熵魔，只能被清理掉，所以遇到熵魔扩散，谁都不敢大意。

玄真长老站了出来，这是宗门七大长老之一，内核级三段的修为，一辈子跟熵魔打过不知道多少次交道，经验丰富，由他带队，确实合适。

玄真长老面色刚毅，对着玄机子拱了拱手："宗主放心，我一定把黑风铁矿的熵魔清理干净，不让一只熵虫跑出来！"

"好，"玄机子点点头，又看向众人，"现在需要征召二十名内门弟子，修为在比特级巅峰以上，跟着玄真长老一起去清剿，有没有主动应征的？"

话音刚落，广场上就一片骚动，不少人往前站了一步，清剿熵魔虽然危险，但任务奖励丰厚，而且也是为宗门立功的好机会，很多内门弟子都想去搏一把。

王翔站在原地，心里快速思考着。黑风铁矿他昨天刚去过，对那里地形熟悉，而且他能直接看源代码，熵魔本质就是BUG和垃圾代码，他找起来比别人快多了，清剿效率肯定更高。更重要的是，熵魔这东西，越早清理损失越小，要是让它们扩散出来，不知道多少无辜矿工要死，作为一个有现代价值观的程序员，他对这种失控的垃圾进程本来就深恶痛绝，遇到了当然不能退缩。

想到这里，王翔往前跨了一步，朗声道："弟子万相，愿意应征！"

他这一出声，不少人都看了过来，万相，最近在二进制宗名气可不小，刚来没多久就帮好多长老和弟子改了功法，解决了不少卡住多年的瓶颈，尤其是他能一眼看出功法里的BUG，这个本事，全宗门独一份，去清剿熵魔，确实合适。

玄机子看到王翔站出来，眼睛亮了亮，点点头："好，万相你主动应征，很好，算你一个。"

又有十几个人陆续站出来，很快二十个名额就满了，都是内门中修为不错的好手，大部分都是比特级巅峰，也有两三个已经摸到了字节大师的边，实力都不弱。

"好了，二十人齐了，"玄真长老点了点头，"大家回去准备一下，半个时辰后在山门口集合，带上必备的法器和疗伤药，这次进山清剿，可能要待几天，都准备充分点。"

"是！"众人齐声应道。

王翔转身往回走，刚走没几步，就听到身后有人轻轻叫他："万相师兄，请等一下。"

声音温柔熟悉，王翔回头一看，原来是叶清语。

她站在一棵大树下面，淡绿色的衣裙被风吹得轻轻晃动，手里还拿着一个小布包，脸上带着一丝担忧，看到王翔回头，赶紧走了过来。

"叶师妹？你怎么在这里？"王翔有点奇怪，叶清语是医堂的，不用来参加这种召集啊。

"我听说了，黑风铁矿熵魔扩散，师兄你主动应征要去清剿，"叶清语走到他面前，抬头看着他，眼神里满是担忧，"熵魔很危险的，一不小心就会被感染，师兄你一定要小心啊。"

看着她担忧的样子，王翔心里一暖，笑了笑说："谢谢你担心我，没事的，我心里有数，玄真长老带队，实力强经验足，不会有大事的。"

"话是这么说，但是小心一点总是好的，"叶清语说着，把手里的小布包递给王翔，"这里面是我炼制的清心丹和消毒散，清心丹能稳住心神，防止被熵魔的混乱代码干扰心智，消毒散要是不小心沾到了熵魔的毒液，抹上就能消毒，抑制感染扩散，你带上，万一能用得上。"

王翔接过布包，入手温热，能感觉到里面丹药的灵气，心里更暖了，这都是叶清语自己亲手炼制的，她一个医女，炼丹制药都是一把好手，这些药都是很实用的东西。

"太谢谢你了师妹，"王翔郑重地把布包收进怀里，"我一定带上，也一定小心，你放心吧。"

"嗯，"叶清语点点头，咬了咬嘴唇，又说，"师兄，我听说这次是深入矿洞，里面情况复杂，熵魔最擅长在狭窄的地方埋伏，你千万不要冒进，一切听玄真长老的安排，安全最重要，知道吗？"

"我记住了，"王翔看着她温柔关切的眼神，忍不住笑了，"放心，我命大，上次在铁锈荒原被人暗算都没死，这次肯定也没事，等我清剿完熵魔回来，再给你看看突破后的功法运行有没有问题。"

叶清语听到这话，也笑了，浅浅的梨涡露出来，但是眼神里还是带着担忧："那我就在宗门等你回来，师兄...一定要平安回来。"

"一定。"王翔郑重地点点头。

时间不多了，王翔不能多聊，跟叶清语告辞，转身快步回了自己的洞府，收拾东西。他把叶清语给的丹药和药散小心收好，又检查了一下排序剑，带足了比特结晶，还有一些干粮和水，一切准备妥当，然后就往山门口走去。

半个时辰后，山门口，玄真长老已经在等着了，二十名内门弟子都到齐了，检查完人数，玄真长老点点头："好了，人都到齐了，我们走！黑风铁矿离这里一百二十里，我们尽快赶过去，争取天黑前就能进入矿洞开始清理！"

一行人出发，顺着山路往下走，快速向着黑风铁矿赶去。王翔走在队伍中间，怀里还能感觉到布包的温度，想起叶清语刚才关切的样子，心里暖暖的，他在这个世界，终于不再是孤身一人了，有人牵挂，这种感觉，真的挺好。

一路上大家都没怎么说话，气氛有点沉重，毕竟是去清剿熵魔，那是拿命在拼，谁也不敢掉以轻心。王翔一边走，一边在心里回想昨天去黑风铁矿的记忆，黑风铁矿一共分三层，一号矿脉是浅层，早就挖完了，现在主要开采二号和三号，三号矿脉最深，在山肚子里好几里地，这次熵魔就是从三号矿脉爆发的，那里地形最复杂，通道又多又窄，最适合熵魔隐藏。

走了大约三个时辰，远远就能看到黑风铁矿了，矿山笼罩在一片淡淡的灰雾之中，那是熵魔扩散后产生的感染雾气，正常人进去了，不用多久就会被感染。监矿长老带着几个弟子守在矿洞口，看到玄真长老一行人来了，赶紧迎了上来，脸色憔悴，眼里都是血丝，显然这几天都没睡好。

"玄真长老，你可来了！"监矿长老拱手行礼，语气急切，"情况怎么样？"
"不好啊长老，"监矿长老苦着脸说，"熵魔越来越多了，刚才又冲出来几次，我们拼死才挡住，再不来，我们就要顶不住了，整个洞口都要被攻破了。"

"别慌，我们来了，"玄真长老沉声道，"你给我们说说，现在里面是什么情况？"

"是这样的，"监矿长老赶紧说，"三天前，我们挖通了一个以前封闭的老溶洞，就在三号矿脉最深处，谁知道那个溶洞里不知道积压了多少万年的熵魔种子，一挖开就泄漏出来了，一开始只是几只小熵虫，我们没当回事，清理了一下，没想到第二天就多了几十只，昨天晚上直接爆发了，上千只熵虫涌出来，把整个三号矿脉都占了，我们十几个矿工来不及撤出来，被困在里面了，现在估计...估计已经凶多吉少了。"

说到这里，监矿长老声音也低沉了，那十几个都是跟着他多年的老矿工，就这么没了，谁心里也不好受。

玄真长老点点头，拍了拍他的肩膀："节哀，我们进去一定会把那些熵魔都清理干净，给兄弟们报仇。你带着你的人守住洞口，不要让任何一只熵魔跑出来，我们进去清理。"

"好，好，我这就安排！"监矿长老赶紧下去布置了。

玄真长老转过身，看着二十个内门弟子，沉声道："各位，情况你们都听到了，里面现在有上千只熵虫，大部分都是比特级，可能也有少量字节级的熵魔领主，我们进去之后，保持队形，不要分散，跟着我走，遇到熵魔不要慌，一起出手清理，记住了吗？"

"记住了！"众人齐声应道。

"好，出发！"

玄真长老一马当先，提着玄铁重剑，率先走进了矿洞，王翔跟在队伍中间，走了进去。矿洞里一片昏暗，空气中弥漫着一股刺鼻的腐臭味，那是熵魔感染后产生的味道，就像代码腐烂发臭一样，让人闻了很不舒服。王翔运功封住口鼻，拿出早就准备好的夜明珠，淡淡的光芒照亮了前面的路。

矿洞里的通道还算宽敞，一开始是一号矿脉，已经废弃了，一切正常，没有熵魔的踪迹，走到二号矿脉和三号矿脉的交界处，就能明显感觉到不对劲了，墙壁上开始出现一块块灰黑色的斑点，那就是熵魔感染的痕迹，原本稳定的矿石代码，已经被侵蚀得不稳定了，摸上去刺手，就像摸到了烂掉的木头一样。

"大家小心，前面就是三号矿脉了，熵魔就在里面。"玄真长老停下脚步，压低声音说道，"我走前面，你们分成两队，左右呼应，不要落单。"

众人调整队形，继续往前走，刚进去没多远，前面黑暗中就传来了"嘶嘶"的声音，一团团灰黑色的东西爬了过来，个头不大，也就半米多长，形状不规则，像是一堆会动的烂肉，上面长满了细小的口器，这就是熵虫，最低级的熵魔小兵，没有智力，只会本能地吞噬一切遇到的代码。

"杀！"

玄真长老一声低喝，玄铁重剑劈了出去，一道凌厉的二进制剑气斩出去，正面的十几只熵虫一下子被劈成了碎片，这些碎片掉在地上，还在蠕动，想要重新融合，玄真长老早有准备，反手打出一道火球符，火焰窜起来，把这些碎片烧得干干净净，彻底碳化，再也不能复原了。

看到掌门动手了，弟子们也纷纷出手，各种剑气法术打出去，熵虫虽然多，但是都是低级的，在一众好手面前，根本不堪一击，一只只熵虫被杀死，清理得很快。

王翔也拔出了排序剑， sorts the entropy bugs just like sorting an array——他这套排序剑法，本来就是用来梳理混乱的能量，对付熵魔这种混乱代码，正好相克，一剑出去，剑气按照二进制顺序排列，碰到混乱的熵魔代码，直接就给梳理拆解了，比普通剑气效果好太多，一剑下去就能清理掉三四只熵虫，效率比别人高了一倍都不止。

旁边一个弟子看到了，忍不住惊叹："万相师兄，你这剑法对付熵魔也太好用了吧？"

王翔笑了笑："我这剑法本来就是排序用的，正好克它们这些混乱玩意儿，赶紧清理，往前推进。"

一行人一路推进，一边杀一边清理，遇到被感染的墙壁，就用净化符烧掉感染斑点，前进速度不算慢，大约走了一个多时辰，就清理了大半个三号矿脉，杀死了好几百只熵虫，一路推进到了矿洞最深处，那个新挖通的老溶洞入口前。

到了这里，熵虫越来越多，也越来越强，已经出现了一两米大的熵虫，实力接近比特级巅峰，皮肤也更硬，普通剑气砍上去只能砍破一层皮，要好几下才能杀死。

玄真长老停下脚步，皱着眉看向前面漆黑的溶洞入口，沉声道："不对啊，按照监矿长老说的，应该有上千只熵虫，我们才杀了几百只，剩下的去哪里了？怎么都聚集在这里？"

王翔也皱了皱眉，他能感觉到，前面溶洞里面，传来一股强烈的混乱能量波动，比这些小熵虫强太多了，那股波动，浑厚而邪恶，绝不是普通熵虫能有的。他凝神细看，能看到源代码层面，前面整个溶洞都被混乱的灰色代码填满了，中心那团代码，特别巨大，还在不断蠕动，吸收着周围的能量。

"玄真长老，"王翔往前站了一步，低声说道，"我感觉到里面核心有一股很强的能量波动，比我们杀过的所有熵虫都强，恐怕...不止是普通熵虫，可能有熵魔领主，甚至...熵魔统领在里面。"

玄真长老眼神一凝，点点头："我也感觉到了，看来这次我们运气不好，碰到成气候的了。不过也好，一次性挖出来，省得它以后长大了再祸害一方。大家准备一下，进去之后，保持队形，我先上，你们两翼接应，千万不要靠近那东西的核心，它会感染你的代码！"

"明白！"众人都握紧了兵器，做好了战斗准备。

玄真长老深吸一口气，提着玄铁重剑，一步一步走进了溶洞。王翔跟在他身后不远，排序剑握在手里，全神戒备。

溶洞很大，比外面的矿洞宽敞多了，穹顶很高，钟乳石从上到下垂下来，地面凹凸不平，整个溶洞里都弥漫着厚厚的灰雾，能见度很低，空气中的腐臭味更浓了，浓得让人作呕。

所有人都打开了夜明珠，十几颗夜明珠的光芒汇聚在一起，照亮了整个溶洞，当大家看清楚溶洞中心的东西时，所有人都倒吸了一口凉气。

溶洞中心，一大团灰黑色的庞然大物，黏糊糊地趴在地上，不断蠕动，体积差不多有一间房子那么大，表面不断有小熵虫从里面爬出来，又爬回去，显然，外面那些熵虫，都是从这个大家伙身上分离出来的。

这玩意儿，就是熵魔的核心，看样子，已经孕育出了简单的智力，是名副其实的熵魔统领，实力至少是字节级巅峰，比很多内门长老都强！

熵魔统领感觉到了有人进来，庞大的身躯微微一动，抬起了一个模糊的脑袋，没有眼睛鼻子，只有一张巨大的嘴，发出低沉的嘶吼声，声音震得整个溶洞都微微发抖，岩壁上的碎石纷纷往下掉。

"果然是熵魔统领！"玄真长老脸色凝重，握紧了玄铁重剑，"大家小心，它已经是字节级巅峰了，我来缠住它，你们清理它分出来的小熵虫，不要让小熵虫绕到我们后面！"

"是！"众人应道，立刻分成两队，站在两翼，戒备着。

玄真长老大喝一声，提着剑就冲了上去，玄铁重剑带着强劲的剑气，狠狠劈在熵魔统领身上，"砰！"一声巨响，剑气炸开，砍进去足足有半尺深，灰色的汁液流了出来，流到地上，把石头都腐蚀出一个个小洞。

熵魔统领吃痛，发出一声怒吼，巨大的身躯一挥，无数触手打了过来，玄真长老纵身一跃，躲开了，落在旁边的一块大石头上，顺势又是一剑，又砍下来一大块熵魔组织。

但是这玩意儿生命力太顽强了，被砍掉的地方，很快就有新的组织长出来，伤口飞速愈合，而且它还能把被砍掉的碎块重新变成新的小熵虫，扑过来咬人，杀不完一样。

"不好，这样下去不行，"王翔看了一会儿，就看出问题来了，这熵魔统领本身就是一堆混乱代码的集合体，你砍它它反而分裂出更多小熵虫，这么耗下去，我们算力迟早会被耗光。

他集中精神，看向熵魔统领的源代码，很快就找到了问题的核心，这玩意儿的核心代码在它身体中心，大约心脏位置，那里有一块黑色的核心结晶，所有的熵魔组织都是从那个核心结晶生长出来的，只要不毁掉那个核心结晶，你砍多少它都能长出来。

找到了核心位置，王翔心里就有底了，他对旁边一个弟子使了个眼色："你去告诉玄真长老，核心在中心心脏位置，不毁核心杀不死它，我们想办法牵制它，让长老找机会刺中心脏！"

那个弟子赶紧跑去，把话传给玄真长老，玄真长老一听，立刻明白了，他也猜到大概是这样，就是找不到核心具体位置，现在王翔找到了，那就好办了。

玄真长老卖了个破绽，故意往后退，熵魔统领以为他力竭了，咆哮着追上来，整个身子都扑了过来，正好把核心位置露了出来。

就是现在！

玄真长老眼中精光一闪，早就准备好了的必杀一击狠狠打了出去，玄铁重剑带着全部算力，像一道黑色闪电，精准地刺向熵魔统领身体中心的核心位置！

"噗嗤！"

利剑入肉，整把剑几乎都刺进去了，正好扎在核心结晶上！

熵魔统领发出一声凄厉到极点的嘶吼，整个庞大的身躯剧烈地抽搐起来，大量灰色的汁液喷溅出来，溅得到处都是，它拼命扭动，想要把玄真长老甩开，玄真长老双手握住剑柄，死死按住，大喝一声："爆！"

内力从剑柄输进去，狠狠炸开，核心结晶里面传出一声闷响，然后就看到整个熵魔统领的身体都开始变软，慢慢融化，抽搐越来越弱，最后终于不动了，瘫在了地上，变成了一滩灰色的黏液，再也没有动静了。

所有的小熵虫失去了核心控制，一下子变得混乱，四处乱爬，弟子们赶紧出手，很快就把所有小熵虫都清理干净了。

溶洞里终于安静下来，只剩下众人粗重的呼吸声，刚才这一战，打得也挺激烈，虽然赢了，但是也有几个弟子受了伤，还好不重，都是被小熵虫擦伤的。

玄真长老拔出血淋淋的重剑，脸上露出一丝笑意，看向王翔："万相，这次真是多亏你了，一下子就找到了核心位置，不然我们还要纠缠好久，搞不好还会有人受伤。"

王翔笑了笑："长老过奖了，我就是能看得清楚一点而已，主要还是长老功力深厚，一剑就解决了它。"

就在大家都松了一口气，准备清理战场的时候，王翔心里突然一动，他感觉到，从溶洞最深处，黑暗的角落里，传来一股更强、更邪恶的能量波动，那股波动比刚才这个熵魔统领还要强得多！

王翔脸色一下子变了，猛地看向溶洞最深处的黑暗，沉声道："不对！还有一个！在那边！"

所有人都吓了一跳，赶紧看过去，玄真长老立刻提起剑，警惕地盯着那个方向："怎么回事？还有一个？"

就在这时，黑暗中传来一阵低沉的笑声，声音沙哑，像是金属摩擦石头一样："呵呵呵呵...没想到啊，二进制宗的小家伙们，来得还挺快，正好，我刚醒过来，肚子饿了，就送你们来给我当点心..."

话音未落，一个更加庞大的身影，慢慢从黑暗中走了出来，这一个，比刚才那个熵魔统领还要大两倍，身材已经隐约有了人形，但是全身都是灰黑色的粘稠组织，脸上只有一双发红的眼睛，散发着嗜血的光芒，它身上散发出的能量波动，竟然达到了内核级！

是熵魔统领！不对，是真正的熵魔领主，内核级的熵魔！

所有人脸色都变了，玄真长老也是一脸凝重，握紧了重剑，沉声道："原来这里不止一个，这个才是真的核心，没想到啊，这个老溶洞里，竟然孕育出了一个内核级的熵魔领主，再给它几十年，整个青云山都要被它祸害了！"

熵魔领主咧开巨大的嘴，露出无数细碎的牙齿，笑声更加刺耳："没错，我在这里睡了三百年了，就等着今天，你们不来，我过几天也要出去，既然来了，那就全都留下来吧，成为我身体的一部分，哈哈哈！"

整个溶洞的空气，都瞬间变得冰冷压抑，一场苦战，在所难免。
