# 第1章 猝死！穿越到代码世界

凌晨两点十七分，写字楼只剩下安全通道的应急灯还亮着淡绿色的光，整层开发部都陷入了沉睡，只有靠窗那个角落还亮着刺眼的蓝光。

显示器的光映着王翔疲惫的脸，胡茬已经两天没刮了，黑色的镜框上沾着一点咖啡渍，头发因为几天没洗油乎乎贴在额头上。键盘敲击声已经持续了六个小时，从晚上八点吃完外卖到现在，他只起身去过一次卫生间，接了三次水。

屏幕右下角的企业微信一直弹着红点，老板的头像一直在右上角闪烁，最后一条消息是一小时前发的："小王，那个空指针的BUG务必今天修好，明天上线不能再拖了，客户那边已经催了三次。"

"最后一个BUG，改完就上线。"

王翔揉了揉发胀的太阳穴，太阳穴突突直跳，心脏也有点发闷，最近半个月天天加班到凌晨，这种情况已经出现好几次了。他没当回事，年轻人，熬熬就过去了，房贷还剩二十年没还，信用卡账单每个月八千多，父母在老家身体也不太好，都指着他这份工资呢。

他端起桌边那杯早已凉透的美式咖啡，猛灌了一大口，苦涩的液体顺着喉咙滑下去，像是一块冰掉进胃里，稍微驱散了一些如山的睡意。味蕾早就麻木了，尝不出咖啡到底是什么味道，只知道要提神，要保持清醒，把这个BUG改完才能下班。

屏幕上，一行行JavaScript代码静静流淌，王翔的光标停在最后一行，这就是那个阴魂不散的偶现空指针。测试环境复现了三次，每次都出现在这个位置，生产环境已经炸过一次，导致用户订单提交失败，客服邮箱被投诉塞满了。

作为一个有着六年经验的全栈程序员，王翔什么场面没见过。从刚毕业进公司做前端切图，到后来转写后端接口，再到数据库优化、服务器运维，公司从小到大，他一步一步跟着走，早就练出来了，什么疑难杂BUG没碰过？现在这个偶现空指针，对他来说不算什么大问题，就是烦人，耗时间。

他盯着那几行异步回调代码，皱起了眉头，指尖在触摸板上轻轻滑动，一行一行往上翻。

"问题应该出在这里，异步回调执行的时候，Promise还没resolve，对象还没初始化就被调用了..."

王翔自言自语，伸出手指，在键盘上敲下了一行防御性判断：

`if (obj != null) {`

很标准的防御式编程，就算对象为空，最多就是不执行，不会直接炸掉。他心里松了口气，准备敲下右花括号，然后测试一下，没问题就提交，就能回家睡觉了。

就在他的指尖落到回车键上的瞬间，胸口突然炸开一阵剧烈的绞痛。

那疼痛来得太突然，太猛烈了，像是有一只无形的手，从肋骨缝隙里伸进去，狠狠攥住了他的心脏，然后猛地一拧。

王翔瞬间疼得喘不过气来，喉咙里发出一声压抑的闷哼，汗一下子就出来了，顺着额头往下淌，瞬间打湿了额前的头发。

"靠...不会吧..."

他脑子里第一个念头不是疼痛，而是离谱，他才二十八岁，怎么可能心脏出问题？最近就是加班多点，熬了几个夜，不至于吧？

他想伸手去够桌上的手机，打120，可是胳膊重得像是灌了铅，指尖离屏幕还有整整三厘米，他用尽全身力气，也没法再往前挪动一分。

眼前的代码开始扭曲模糊，红蓝绿的像素在视野里旋转起来，变成了一片闪烁的光斑，那些熟悉的变量名和函数名变得认不出来，只剩下一片光影变幻。

王翔的意识一点点模糊，他能感觉到自己的呼吸越来越微弱，心脏跳动越来越慢。他盯着那行刚敲出来的`if (obj != null) {`，心里闪过最后一个离谱的念头：

**我这算工伤吧？这BUG真不赖我...是产品经理改了三回需求，改来改去改出问题的...不知道HR会不会给我家属赔抚恤金...房贷还没还完呢...**

然后，无边无际的黑暗，瞬间吞噬了一切。

---

"咳...咳咳咳咳！"

剧烈的咳嗽像是要把肺都咳出来，撕裂般的疼痛撕扯着胸腔，王翔猛地睁开眼睛，呛出两口带着铁锈味的淤血，热乎乎的血沫沾在嘴角，又腥又咸。

他躺在一张坚硬而潮湿的木板床上，映入眼帘的不是医院洁白的天花板，更不是阴曹地府的奈何桥，而是一扇裂着好几道大口子的木制屋顶，屋顶的木板缝里透进灰蒙蒙的天光，空气中弥漫着一股潮湿木头和劣质泥土混合的腥气，还有一股子淡淡的霉味。

"这是哪儿？"

王翔脑子一片混乱，还没从那致命的绞痛里反应过来。他想撑着坐起来，结果稍微一动，全身骨头都像是散了架一样，每一块肌肉都在疼，尤其是胸口，像是被大象狠狠踩过，每一次呼吸都带着针扎一样的刺痛。

他下意识地低下头，看向自己伸出来的那只手。

这不是他的手。

他的手，因为常年敲键盘，指腹磨出了厚厚的茧子，手腕因为腱鞘炎，还有一块小小的增生硬块，手掌宽大，带着成年人的厚实。可这只手，瘦小，苍白，指节纤细，分明是一个十几岁青少年的手，而且手背上布满了新新旧旧的伤口，一道长长的刀疤从手腕延伸到小臂，新鲜的伤口还在慢慢渗血，血珠顺着指尖一滴滴落在木板床上。

"不对...这身体不是我的..."

一个荒谬但又唯一合理的猜测浮现在王翔心头。紧接着，一股庞大的陌生记忆碎片像是潮水一样轰然涌入他的脑海，带着原身浓烈得化不开的情绪——愤怒，仇恨，不甘，还有深入骨髓的绝望。

王翔赶紧闭上眼，咬着牙屏住呼吸，任由那些记忆碎片在脑海里冲撞，花了整整五分钟，才慢慢把这些碎片拼接成一幅完整的轮廓。

原身名叫万相。

这里叫黑铁聚落，是蛮荒代码平原上一个最底层的小村子。而这个世界，叫做比特神域。

原身是个孤儿，两年前，父母被村里的恶霸势力"零号帮"逼死，抢走了家里唯一一块能产出低纯度比特结晶的矿地。那块矿地虽然贫瘠，但产出的比特结晶足够一家三口活下去，零号帮看上了，一句话就要收走，父母不肯，就被活活打死了。

原身忍气吞声，苟活了两年，昨天，零号帮的人又来收"保护费"，原身实在交不出来，忍不住说了几句硬话，就被零号帮的头目"疤脸"亲自出手，一拳打在胸口，打成了重伤，扔在这间破木屋里，放话今天天亮了再来收尸，顺便把这破房子拆了，那块矿地就能正式开工了。

"也就是说...我一个在地球996猝死的程序员，这就穿越了？还穿到了一个马上就要死的孤儿身上？"

王翔理清了记忆，忍不住在心里吐槽，"这开局也太地狱难度了吧？猝死就算了，穿越过来还要再死一次？有没有这么玩的？"

他定了定神，强迫自己冷静下来。现在吐槽没用，得先搞清楚状况，这个比特神域到底是个什么世界？

根据原身的记忆，这个世界，**万物皆由源代码构成**。

山川河流是代码，飞禽走兽是代码，人也是代码，每个人都是一个运行在比特神域上的独立进程。修炼，就是打磨自己的核心代码，提升算力，让自己的进程运行得更快更稳定，就能更强大。

原身是这个世界最底层的比特战士，刚刚能感知到比特能量，还没入门，就被打成了重伤，核心代码被打坏了，进程都快要崩溃了。

王翔按照原身记忆里的方法，闭上眼睛，试着调动身体里那点微弱的比特能量——也就是这个世界修炼者口中的"斗气"。

一股细得几乎感觉不到的微弱数据流，慢悠悠地在经脉里流动起来，所过之处，一阵阵刺痛传来，就像是一台老旧不堪的机器，零件都快磨坏了还在强行运转。

王翔敏锐地发现，这股数据流的运行路径，简直歪得离谱。

走走停停，绕了无数不必要的远路，还有好几个节点严重堵塞，能量流到那里就卡住半天才能过去，简直就是一坨屎一样的代码。

"这路径...怎么这么眼熟..."王翔皱着眉，"跟我昨天维护的那个十年老项目一样，循环嵌套了五层，还到处都是goto乱跳，能运行就是奇迹了。"

他心里这么想着，眼前突然一花，一行行半透明的浅蓝色代码，就那么毫无征兆地浮现在了空气中。

就在他胸口受伤的位置，二进制代码清晰地悬浮着，一行行红色的报错信息不停地闪烁，刺得人眼睛疼：

```
// 万相 - 核心进程
PID: 184756
Status: RUNNING (严重内存损坏)
Memory: [=====......] 42% 已损坏
Errors: 
  - NullPointerException at line 1872
  - StackOverflowError at line 2451
  - MemoryLeak detected: 128KB uncollected garbage
Process uptime: 18 years 3 months 12 days
Remaining lifetime: 0 days 03 hours 12 minutes 47 seconds
```

王翔整个人都看傻了。

他使劲眨了眨眼睛，再看，那些代码还在那里，清清楚楚，每一行报错都看得明明白白。

他下意识地转动眼珠，把视线移到旁边那张歪歪扭扭的木桌上。

下一秒，木桌的源代码也应声浮现在他眼前：

```
// 普通木桌 - 数据对象
Object ID: 89745
Material: pine_wood
Quality: broken
Durability: 12/100
Functions: 
  - support()
  - putThings()
```

王翔的心跳一下子漏了一拍，他又赶紧看向自己身上穿的那件破烂皮衣。

果然，皮衣的源代码也弹了出来：

```
// 粗皮衣 - 装备
Slot: body
Level: common
Defense: +1
Durability: 3/20
Special: none
```

王翔的心脏开始狂跳起来，扑通扑通，撞得胸腔都在响。

他能看见...源代码？

这个世界真的是由代码构成的，而他，一个地球来的程序员，能直接看见这些代码？

原身的记忆里确实说过，万物皆源代码，修炼就是感悟源代码。可是...所有原住民都只能"感悟"，只能"调用"，从来没有人能直接看见源代码啊！

这个世界的修炼者，就像是一个个只会调用第三方库的API调用工程师，只知道接口参数和返回值，从来看不见源码，更别说修改源码了。

而他...直接拿到了root权限？

王翔深吸一口气，强迫自己冷静下来。他是程序员，遇到问题就是解决问题，慌没用。现在最大的问题是什么？看屏幕上写的，剩余寿命还有三个小时。

三个小时内不修好，他就要再死一次了。

那还有什么好说的，DEBUG吧！

他集中精神，仔细看着胸口核心代码的那些报错，一条一条捋过去。

问题其实很清晰，一点都不复杂。

第一，经脉里的能量节点排序混乱，运行路径不对。原身修炼的那本基础功法《黑铁基础诀》，本身就写得烂，能量路径是无序的，相当于给一个无序数组排序用了冒泡排序，时间复杂度O(n²)，效率低得令人发指，而且还有很多重复计算，完全是浪费算力。

第二，被疤脸一拳打在核心区域，打坏了42%的核心内存，很多块内存损坏了，没有被垃圾回收，一直占着空间，导致内存不足，进程随时都会OutOfMemoryError崩溃。

就这两个问题，对一个程序员来说，那都不叫事。

"这都不叫事。"

王翔嘴角勾起一抹冷笑，作为一个天天跟BUG和内存泄漏打交道的程序员，这种场面他太熟悉了。改就是了。

首先解决第一个问题，优化能量运行路径。

原路径是冒泡排序，效率太低，那换成快速排序不就行了？查找能量穴位用线性查找太慢，换成二分查找，时间复杂度直接从O(n)降到O(logn)。

王翔在脑子里重新规划了一遍路径，把无序的能量节点按照访问频率重新排序，把最常用的节点放在最前面，用快速排序排列，把二分查找替换掉原来的线性查找，又删掉了那些重复计算的冗余节点。

这一番优化下来，路径瞬间清晰了，原来O(n²)的复杂度，直接降到了O(n logn)。

仅仅这一步优化，身体里比特能量的流动速度，瞬间提升了整整十倍！

原来需要一个周天才能吸收完的外界比特能量，现在不到一分钟就完成了一个循环。那股原本细弱的暖流，一下子变得有力起来，顺着优化后的路径快速游走，所过之处，受损的代码正在一点点被修复，那些刺痛感也减轻了不少。

"漂亮！"

王翔心中一喜，这方法管用。

搞定了路径，接下来就是处理损坏的内存。

原身核心代码里，那些损坏的内存块都被标记成了NULL，一直占着空间没人清理，正好做一次垃圾回收。王翔想起Go语言的标记-清除算法，思路很清晰：先把所有还在正常使用的内存块都标记一遍，然后把所有没有标记的损坏块统一清除，回收空间，重新放回空闲链表。

说干就干。

王翔集中精神，按照算法步骤，一个一个标记有效内存块，这个过程很繁琐，需要注意力高度集中，不能错漏一个，不然可能会把有用的内存给清掉，那就直接凉凉了。

这一步，花了他将近半个小时。

当最后一个损坏内存块被清理完成，重新放进空闲链表的时候，王翔再看那行剩余寿命，已经从三个小时，变成了三天。

虽然三天还是不多，但至少...他暂时活下来了。

王翔长长地松了一口气，后背已经被冷汗湿透了，刚才那半个小时，精神高度集中，简直比连续改三个小时BUG还累。

他撑着墙壁，慢慢坐起身，靠在墙上，稍微缓了口气，这才有机会环顾一下这间破木屋。

果然是家徒四壁，除了他躺着的这张晃悠悠的破木板床，就只有一张歪歪扭扭的木桌，一把断了一条腿用木棍撑着的破椅子，墙角靠着一把原身留下的新手铁剑，除此之外，什么都没有了。

真是比他刚毕业的时候住的地下室还穷。

王翔笑了笑，其实这样也不错，至少他活下来了，而且还拿到了这么一个金手指，能直接看源代码，这要是放在任何一个穿越小说里，那都是主角模板啊。

他的目光落在墙角那把新手铁剑上，下意识地看了过去。

果然，源代码自动浮现在眼前：

```
// 新手铁剑 - 武器
Item ID: 123
Type: sword
Level: common
Damage: 2-5  // 硬编码
Durability: 12/30
Quality: poor
Special: none
Code smell: 硬编码伤害值，耦合度极高，可维护性为零，典型的面条代码
```

王翔忍不住笑了。连代码坏味道都能看出来，看来他这程序员本能，真是刻进灵魂里了，走到哪儿都改不了。

说实话，这把剑真是烂得可以，伤害低，耐久差，代码写得一塌糊涂，全是魔法数字硬编码，耦合度高得离谱，一点可扩展性都没有。反正现在他也没事，刚修好自己，闲着也是闲着，重构一下吧，就当练练手，熟悉一下这个世界的代码规则。

王翔撑着墙，慢慢挪下床，走到墙角，弯腰拿起那把铁剑。入手很沉，铁锈味扑面而来，剑刃上满是缺口，确实烂得可以。

他把铁剑放在那张歪歪扭扭的木桌上，集中精神，凝视着它的源代码。

原代码结构真是混乱不堪，一百行不到的代码，到处都是硬编码，逻辑耦合得一塌糊涂，看得人头疼。王翔一点点重构，把硬编码的伤害值抽出来，做成动态函数，根据使用者的力量属性动态计算伤害，又把硬编码的材质依赖改掉，换成依赖注入，增加了可扩展性，以后升级材质只要替换依赖就行，不用改核心逻辑。

一顿操作下来，原本几十行的烂面条代码，被他重构得干干净净，模块化拆分，单一职责，可读性和可扩展性提升了不止一个档次。

重构完成的那一瞬间，铁剑突然轻轻震颤起来，表面泛起一道淡淡的绿色微光，那微光从剑柄慢慢蔓延到剑尖，原本锈迹斑斑坑坑洼洼的剑身上，渐渐浮现出一行行清晰的银色纹路，那些纹路像是二进制代码，又像是天地符文，看起来说不出的玄妙。

王翔伸手握住剑柄，入手温润，比之前称手多了，重量也变得恰到好处，原本的锈迹消失了不少，剑刃也变得锋利起来。

他再看向源代码，已经焕然一新：

```
// 排序剑 - 武器（重构后）
Item ID: 123 (refactored by 万相)
Type: sword
Level: green
Damage: (user.strength * 0.8) - (user.strength * 1.2)  // 动态计算
Durability: 25/25
Quality: green（优秀）
Special: 
  - 排序斩：对多名敌人自动按剩余血量排序，优先击杀残血目标，暴击率提升10%
Creator: 万相
```

"绿名装备？"

王翔眼睛一下子亮了。他就是随便重构了一下，竟然把一件白色垃圾修成了绿名装备？这波血赚啊！

而且这个特效"排序斩"还挺实用，以后一打多的时候，自动排序优先杀残血，提升暴击率，简直就是新手神技。

不愧是程序员重构，立竿见影。

王翔握着排序剑，挥了两下，风声呼啸，比原来好用太多了，心里正高兴着，门外突然传来了粗鲁的喝骂声，伴随着沉重的脚步声，一步步朝着这间破木屋这边过来。

"那小杂种应该还没断气吧？疤脸哥说了，今天来收尸，拆了这破房子，那块矿地过几天就能开工了。"一个粗嗓门说道，声音里满是嚣张。

"嘿嘿，一个穷逼也敢跟咱们零号帮作对，真是活腻歪了。我估计啊，熬不到天亮就咽气了，咱们今天过来直接抬走扔铁锈荒原喂野怪就行，省事。"另一个更尖的声音笑着附和。

脚步声越来越近，停在了木屋门口，两人说着话，已经到了门口。

然后，一只粗糙的大巴掌狠狠拍在破门板上，砰的一声闷响，震得屋顶都掉下来不少木屑。

"万相！出来！别他妈躲里面装死！出来受死！"

粗鲁的喊叫声推门而入，震得王翔耳朵嗡嗡响。

王翔握着手里的排序剑，嘴角慢慢勾起一抹冰冷的笑容。

来了？

正好。

原身的父母被零号帮逼死，原身又被他们打成这样，扔在这里等收尸。这个仇，今天就得报。

反正他现在是万相，原身的仇，就是他的仇。

这群不开窍的原住民，一辈子只会调用现成API，连源代码都看不见，也敢在我一个专业程序员面前撒野？

今天就让你们好好尝尝，什么叫做**代码重构降维打击**。

门被一脚踹开，两个穿着粗布短打的汉子走了进来，脸上带着凶狠的笑，当他们看到坐靠着墙，握着剑的王翔时，都愣了一下，显然没想到这个被打成重伤的小子居然还能坐起来。

"咦？你居然还没死？"带头那个粗汉子惊讶道，上下打量着王翔，眼里满是不可思议。

王翔握着排序剑，慢慢站起身，身体还有些虚，但眼神冷得像冰：

"我死了，谁去给疤脸收尸？"

那两个零号帮的喽啰愣了一下，随即反应过来，脸上露出狰狞的笑。

"好小子，命还挺硬，挨了疤脸哥一拳居然还能说话。"粗汉子朝前走了一步，手按在了腰间的刀柄上，"看来今天不亲自动手，你是不肯乖乖走了。"

王翔握着排序剑，慢慢站直了，胸口还有些疼，但是经过刚才的修复和半小时的能量吸收，已经能正常战斗了。他看着眼前这两个喽啰，脑子里下意识地就调出了他们的源代码。

没错，现在他已经试出来了，只要他想看，任何人任何东西，源代码都能直接看到，一目了然。

他看向那个粗汉子，源代码浮现在眼前：

```
// 张三 - 零号帮喽啰
Level: 比特战士 5段
Health Points: 100/100
Damage: 4-8
Equipment: 粗铁刀
Skills: 劈砍，野蛮冲撞
Code smell: 气血运行路径混乱，有旧伤未修复，内存泄漏约5%
Weakpoint: 左腰旧伤，这里的代码有一个越界读写漏洞
```

王翔眼睛一下子亮了。

还能这么玩？弱点直接标出来了，这不是开全图挂吗？

他再看另一个瘦个子：

```
// 李四 - 零号帮喽啰
Level: 比特战士 3段
Health Points: 70/70
Damage: 3-6
Equipment: 匕首
Skills: 偷袭，捅刀子
Code smell: 刚偷吃过过期比特肉，肠道内存损坏，运行不稳定
Weakpoint: 腹部，受到重击容易引发拉肚子直接失去战斗力
```

王翔差点笑出声来。这也太方便了吧？两个人什么水平，哪里弱，一眼就看明白了，这还打什么？直接怼弱点不就完了。

"小子，挺狂啊。"张三脸上横肉抖动，"疤脸哥说了，今天要你的命，你就得死，别以为拖到天亮就能有人救你，这一片谁他妈敢管我们零号帮的事？"

王翔握着排序剑，微微侧身，冷静地看着对方："少废话，要动手就快点，我还得去疤脸那里收账呢。"

"嘿，还挺硬气！"张三怒极反笑，"我看你是伤成这样，脑子都打糊涂了，老子今天就成全你，让你跟你爹妈团聚去！"

话音未落，张三猛地拔出腰间的粗铁刀，一个箭步冲了上来，抬手就是一刀，劈头盖脸朝着王翔脑袋砍下来，刀风呼呼，力气还真不小。

这一刀，要是原身，挨上肯定就没了。但现在，王翔早就看清楚了他的发力路径，源代码看得一清二楚，哪里有空档，哪里是弱点，一目了然。

王翔脚步轻轻一晃，侧身躲开这一刀，同时手里排序剑轻轻刺出，直奔张三左腰那个旧伤弱点而去。

这一剑又快又准，正好刺中那个位置！

"呃啊！"

张三发出一声惨叫，疼得浑身一颤，力气一下子就泄了。王翔说得没错，这里就是他上次跟人火拼留下的旧伤，一直没好彻底，没想到这小子第一剑就扎在这里！

被刺中旧伤的瞬间，张三就感觉到那里的气血一下子乱了，原本顺畅的能量运行直接卡住，就像是程序跑飞了一样，半个身子都不听使唤了。

"你...你怎么知道我这里有伤？"张三不敢置信地看着王翔，疼得额头直冒冷汗。

王翔没说话，趁着他剧痛浑身僵硬的时候，抽剑再刺，正中胸口。

张三吭都没吭一声，直接倒了下去，源代码上的Status变成了TERMINATED，进程结束。

整个过程，从张三冲过来，到张三倒下，不到十秒钟。

旁边的李四都看傻了，手里攥着匕首，愣在原地，眼睛瞪得大大的，不敢相信自己看到的。

张三可是比特战士五段，比他还强两级，居然被一个已经挨了疤脸一拳濒死的小子，两剑就干掉了？

这不对啊！这剧本不对！

"该你了。"

王翔转过目光，看向李四，眼神冰冷。

李四吓得往后退了一步，咽了口唾沫，声音都发颤了："你...你你...你别过来啊！我告诉你，疤脸哥就在后面，马上就来了，你敢动我一下，疤脸哥肯定杀了你！"

王翔根本不理他，一步步往前走。他知道李四的弱点在腹部，就是吃坏了东西，现在只要给一下重击，直接就失去战斗力了。

李四看着王翔走近，牙一咬，心一横，握着匕首就扑了上来，想要捅王Modal肚子："你别过来！我跟你拼了！"

匕首直刺而来，王翔轻轻侧身躲开，同时抬脚，照着李四肚子就是一脚。

这一脚，用了十分力气，正好踹中弱点位置！

"呃！"

李四脸色瞬间就变了，捂着肚子弯下腰，刚才还嚣张跋扈，现在疼得站都站不住，肚子里翻江倒海，一股说不出的绞痛袭来，瞬间就控制不住了。

别说战斗了，他现在连站都站不住，就想找地方拉肚子。

王翔上前一步，一剑放在他脖子上："说，疤脸是不是马上就来？"

李四疼得满头大汗，话都说不利索了，一个劲点头："是是是...疤脸哥...疤脸哥马上就到...我就是先来探路的...饶命啊...我也是被逼的..."

王翔点点头："行，我知道了。你走吧，回去告诉疤脸，我在这儿等他。"

李四简直不敢相信自己的耳朵，愣了一下，然后连滚带爬爬起来，也顾不上肚子痛了，扭头就跑，跑出去不远还掉了裤子，别提多狼狈了。

王翔看着他跑远，没有追。放他回去报信，正好叫疤脸过来，一次性解决了，省得他还要找上门去。

原身的仇，今天就一次性了了。

他低头看了看张三的尸体，从张三身上搜出了几块低纯度比特结晶，还有几个铜板，都是今天收保护费收上来的。王翔毫不客气收进自己怀里，现在他穷得叮当响，有资源不用白不用。

然后，他靠在门边，看着外面灰蒙蒙的天空，稍微调息了一下，刚才战斗用了不少能量，现在趁着疤脸没来，赶紧再吸收一点，恢复一下状态。

他按照优化后的路径运行比特能量，吸收速度比原来快了十倍，仅仅几分钟，消耗的能量就恢复得差不多了。这就是算法优化的好处，同样时间，吸收的能量多了十倍，修炼速度就是这么一点点拉开的。

王翔能感觉到，自己现在虽然还是比特战士一段，但能量运行效率比原来高了十倍，真打起仗来，打原来的五个自己都没问题。

这就是降维打击。

原住民修炼，讲究的是慢慢感悟，几十年如一日，一点点磨，磨出一点感悟，进步一点点。主角上来直接改源码，优化算法，一步到位，效率碾压。

没等多久，就听到远处传来沉重的脚步声，一个粗犷的声音怒气冲冲地吼道：

"臭小子！居然敢杀我零号帮的人！我看你是活腻歪了！今天我不把你碎尸万段，我疤脸名字倒过来写！"

王翔抬起头，看向声音传来的方向，一个满脸横肉的壮汉大步走了过来，脸上一道刀疤从额头一直划到下巴，显得格外凶狠，手里提着一把沉重的开山刀，杀气腾腾。

来了。

正主终于来了。

王翔握紧了手里的排序剑，绿色的微光在剑身上一闪而逝。

疤脸，你打死原身，逼死原身父母，这笔账，今天该算了。

（第一章完）
