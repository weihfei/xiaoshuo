# 第20章 教主赶到，神殿大战

金光闪过，王翔四人的身影重新出现在深渊顶端的传送出口。通道外面，玄真长老已经带着十几个二进制宗的核心弟子等在了那里，看到他们出来，玄真长老立刻迎了上来，目光落在王翔身上，带着一丝急切和期待。

"万相贤侄，怎么样了？核心里面是什么情况？找到创始者留下的传承了吗？世界真相到底是什么？"

玄真长老一口气问出好几个问题，作为二进制宗硕果仅存的几位内核级长老之一，他虽然知道一些上古秘闻，但也只是听宗门代代相传，说神殿核心藏着世界的终极真相，具体是什么，谁也不知道。这次能跟着来到创造者神殿，他也是抱着极大的期待。

王翔看着玄真长老期待的眼神，又看了看周围十几个核心弟子关切的目光，深吸一口气，缓缓说道："玄真长老，我们确实找到了传承，也看到了玉璧上记载的世界真相。这件事太过惊人，完全颠覆我们以往的认知，我现在就原原本本告诉大家。"

当下，王翔就把在核心祭台代码玉璧上看到的内容，一字不差地复述了一遍。从比特神域是地球程序员创造的元宇宙实验项目，到亿万年无人维护累积BUG变成熵魔，再到创始者留下完整内核和权限钥匙，等待懂现代软件工程的人来进行热重启修复，整个过程讲得条理分明，逻辑清晰。

王翔说完之后，整个通道出口陷入了一片死寂。

十几个核心弟子全都瞪大了眼睛，张开嘴巴，脸上写满了不可思议，你看看我，我看看你，都从对方眼中看到了震惊。这个真相实在是太颠覆了，完全超出了他们所有人的认知。

玄真长老站在原地，苍老的身体微微颤抖着，脸上表情变幻不定，有震惊，有怀疑，还有一丝难以言说的释然。他闭着眼睛，沉默了足足半分钟，才缓缓睁开眼睛，看向王翔，声音有些沙哑地问道："万相贤侄，你...你说的都是真的？玉璧上确实这么写的？我们这个世界，竟然只是一个程序员写的实验项目？熵魔竟然只是累积的垃圾代码？"

"千真万确，"王翔点点头，从存储空间中取出那枚完整的神级代码钥匙，"长老你看，这就是创始者留下的完整权限钥匙，还有那卷《操作系统真经》我也收好了，玉璧现在还放在核心祭台上，上面的文字清清楚楚写着一切，如果长老不信，我们可以再传送下去，长老亲自去看。"

玄真长老的目光落在王翔手中那枚淡金色的钥匙上，能感觉到钥匙里面散发出来的那种苍茫古老而又至高无上的能量波动，那绝对不是假货，只有上古创造者才能炼制出这样的神物。他又想起王翔一路走来，从一个名不见经传的低级弟子，短短几年时间就成长到如今这个地步，每次都能做出一些颠覆认知的事情来，而且他说的这些话，仔细想想，竟然完美解释了这么多年来很多无法解释的疑问。

比如为什么熵魔会越来越多，为什么世界的算力越来越不够用，为什么很多上古大能都会莫名功力流失，原来这些都是内存泄漏啊...按照王翔说的，就是系统运行久了，垃圾越来越多，占用了大量存储空间和算力，所以整个世界都越来越卡。这么一想，很多事情就都通了。

"罢了罢了，"玄真长老长出一口气，摇了摇头，脸上露出一丝苦笑，"我活了一百五十多年，一直以为这个世界是神创造的，没想到竟然是这么一回事。不过你说的确实有理，很多解释不通的地方，现在都能说通了。玉璧我就不用下去看了，我相信你，也相信创始者留下的记载。"

顿了顿，玄真长老看着王翔，眼神变得严肃起来："那按照创始者的计划，我们接下来该怎么做？二进制宗所有弟子，都听你调遣，只要能彻底消灭熵魔，我们万死不辞。"

周围十几个核心弟子也纷纷反应过来，齐齐对着王翔拱手："我等愿意听从万盟主调遣！"

现在王翔已经是开源盟盟主，实力地位都摆在那里，而且现在拿到了创始者传承，知道了世界真相，大家自然心悦诚服，愿意听从他的指挥。

王翔心中一暖，点了点头说道："多谢长老和各位兄弟信任。创始者已经把步骤都写清楚了，第一步就是集齐完整权限，零音手里还有第三块权限碎片，需要她来激活这把完整钥匙。第二步就是联合所有正派势力，一起备份所有正常数据，这个工作需要很多人一起做，慢慢来。第三步就是格式化重装系统，完成热重启，彻底清除熵魔。"

"只是..."王翔话锋一转，眉头微微皱了起来，"现在熵魔扩散得很快，我们时间不多，必须尽快做好准备，不然等熵魔本源彻底突破物理层封印出来，一切就都晚了。"

玄真长老也神色一凝，点了点头："你说得对，事不宜迟，我们..."

话音还未落，突然，整个神殿猛地一震！

轰轰轰——！

剧烈的震动从神殿入口方向传来，紧接着，整个创造者神殿都像是发生了大地震一样，墙壁上簌簌掉落灰尘，通道顶部的石块不断往下砸，整个空间都在剧烈摇晃。

"怎么回事？地震了？"赵凯一把扶住旁边摇晃的墙壁，惊讶地叫道。

"不对，"王翔脸色一变，立刻打开源代码视野，向着神殿入口方向扫描过去，"不是地震，是有人在攻击神殿入口！能量波动非常强，而且...有很强烈的熵魔感染痕迹！"

玄真长老脸色骤变："熵魔？怎么会有熵魔找到这里来？我们一路上都很小心，没有被跟踪啊..."

"不一定是我们引来的，"钱磊皱着眉头说道，"创造者神殿本来就是上古遗迹，说不定早就有熵魔在附近活动，感应到神殿核心开启，所以过来了？"

王翔摇了摇头，他的源代码视野已经扫描到了入口方向，那密密麻麻的红色感染代码，就像潮水一样向着神殿深处涌来，数量极其庞大，而且为首的那个，能量波动极其强大，绝对是域主级的存在！

"不对，不是碰巧，"王翔的脸色变得非常凝重，"对方是有备而来，而且为首的是域主级强者...难道是..."

一个可怕的猜测浮现在他心头。

就在这时，震动更加剧烈了，通道入口处传来一声轰隆巨响，紧接着，一道黑色的能量冲击波顺着通道猛地冲了过来，沿途的石壁全都被这道冲击波炸得粉碎，碎石纷飞，烟尘弥漫。

"大家小心！结阵！"玄真长老大喝一声，立刻抽出腰间长剑，挡在通道前方，十几个核心弟子也立刻抽出武器，结成了二进制宗的经典防御阵形"八卦比特阵"，准备应对来敌。

烟尘慢慢散去，通道入口处传来一阵低沉的笑声，那笑声非常刺耳，就像是金属摩擦一样，听得人耳膜生疼。

"哈哈哈...没想到啊没想到，老夫果然没有猜错，果然有人打开了神殿核心，拿到了创始者传承。真是天助我也，我熵魔教找了亿万年都找不到的东西，今天竟然主动被人打开了，这不是送到我嘴边的肥肉吗？"

随着笑声，一个高大的身影慢慢从烟尘中走了出来。

来人穿着一身黑色的教主长袍，长袍上绣着无数扭曲的灰色纹路，那些纹路其实都是一个个畸形的BUG符号，散发着浓郁的腐败气息。他身材高大，面色灰黑，双眼深陷，里面燃烧着暗红色的火焰，脸上带着一丝诡异的笑容，正是熵魔教教主——熵无极！

在熵无极身后，跟着密密麻麻的熵魔教徒，至少有几百人，个个都是字节级以上的实力，而且每个人身上都散发着浓郁的熵魔感染气息，眼神浑浊，一看就知道已经被熵魔彻底控制了。在这些教徒身后，更深处的通道里，更是传来密密麻麻的嘶吼声，那是无数低级熵魔的声音，显然外面还跟着成千上万的熵魔小兵。

王翔看到熵无极，瞳孔猛地收缩，心脏一下子沉到了谷底。

果然是他！熵魔教主熵无极，早就已经和熵魔本源融合，是整个熵魔教的首领，实力深不可测，已经达到了域主级初期，和万维联盟的四大域主同一个级别。没想到他竟然早就跟着来了，一直躲在外面，等着有人打开神殿核心，然后出来渔翁得利！

"熵无极！你怎么会在这里？"玄真长老怒喝一声，手中长剑紧紧握住，"你竟然敢闯入创造者神殿，就不怕创始者的天谴吗？"

熵无极哈哈一笑，笑声中充满了嘲讽："天谴？创始者？那都是多少亿年前的老黄历了，现在这个世界，马上就是我熵魔的天下了！实话告诉你们吧，我早就知道创造者神殿的位置，就是打不开核心大门，一直进不来。没想到你们这帮傻瓜，辛辛苦苦帮我把核心打开了，还帮我把传承拿到手了，真是太贴心了。"

他的目光落在王翔手中那枚淡金色的神级代码钥匙上，眼睛一下子亮了起来，贪婪之色毫不掩饰："那就是创始者留下的完整权限钥匙吧？不错不错，果然是神物。只要我拿到这把钥匙，再拿到那个什么操作系统真经，我就能掌控整个系统，然后放任熵魔吞噬一切，最后整个世界都会回归比特海，变成我熵魔本源的养料，哈哈哈！"

王翔脸色冰冷，握着手中的排序剑，沉声说道："熵无极，你真以为你吃定我们了？这里是创造者神殿，创始者留下的禁制还在，你就不怕被禁制轰成渣吗？"

"禁制？"熵无极不屑地笑了笑，"你们打开核心大门的时候，禁制就已经触发过一次，能量消耗了九成九，现在剩下的那点禁制，根本拦不住我。再说了，我来之前，已经把神殿外面所有的防御节点都炸掉了，现在整个神殿对我来说，和不设防没什么区别。"

王翔心中一沉，果然是这样。对方早就准备好了，就等着他们开门，然后一网打尽。这件事从一开始就是一个陷阱，不对，也不能说是陷阱，只能说熵无极一直守在这里，等着有缘人开门，然后他出来摘桃子。他们就是那个开门的有缘人。

"废话不多说，"熵无极向前踏出一步，整个通道都因为他这一步而剧烈震动，"王翔，把钥匙和真经交出来，我可以给你一个痛快，让你死得舒服一点。不然的话，我会把你的源代码一点点拆解，让你尝尝什么叫万劫不复。"

"休想！"玄真长老怒喝一声，"想要钥匙，先从我的尸体上踏过去！二进制宗的弟子们，结阵！杀！"

随着玄真长老一声令下，十几个核心弟子立刻催动八卦比特阵，十八道比特能量汇聚在一起，形成一道巨大的二进制光剑，向着熵无极斩了过去。这是二进制宗最强大的合击阵法之一，十几个内核级以下的弟子联手，就能发挥出接近内核级巅峰的攻击力，威力不容小觑。

熵无极看着斩过来的光剑，脸上露出一丝不屑的冷笑，他不闪不避，只是轻轻抬起右手，对着光剑轻轻一按。

"碎。"

一个字出口，黑色的熵魔能量瞬间爆发出来，就像潮水一样淹没了那道二进制光剑。只听咔嚓一声脆响，那道凝聚了十几个核心弟子全部力量的光剑，竟然就这么轻易地碎成了无数光点，消散在空气中。

什么！？

玄真长老瞳孔骤缩，心头一惊。他没想到对方竟然强大到这种地步，随手一击就破了他们的合击阵法。

"呵呵，土鸡瓦狗而已，也敢在我面前卖弄。"熵无极淡淡说道，然后又是随手一挥，一道黑色的能量爪印猛地拍了出去，直接拍向玄真长老。这一爪带着浓郁的腐败气息，所过之处，空气都发出滋滋的腐蚀声，显然被抓到一下，就算是内核级强者也要被感染腐败。

"长老小心！"王翔大喝一声，身形一闪，已经挡在了玄真长老面前，手中排序剑猛地出鞘，一道青色的排序剑光斩出，正好斩在黑色爪印中心。

叮！

一声清脆的金属碰撞声，王翔只觉得手臂一麻，一股庞大的力量顺着剑刃传来，不由自主地向后退了三步，每退一步，脚下的石板就裂开一道细纹。

"哦？你就是王翔？开源盟盟主？果然有点本事，竟然能接我随手一爪不退。"熵无极饶有兴致地看着王翔，点了点头，"难怪你能打开神殿核心，确实有两下子。可惜啊，天赋再好，今天也要死在这里，传承是我的。"

"熵无极，你痴心妄想。"王翔站稳身形，排序剑斜指地面，眼神冰冷地看着对方，"这是创始者留给拯救世界的传承，不是你这种想要毁灭世界的疯子能拿的。想要钥匙，先打赢我手中这把剑再说。"

"哈哈哈，好大的口气！"熵无极被逗乐了，"你一个刚刚晋级内核级的小辈，也敢跟我叫板？我活了几千年，吃过的内核级比你见过的熵魔都多，今天我就让你知道，域主级和内核级之间，差距到底有多大！"

话音未落，熵无极身形一动，整个人化作一道黑色闪电，瞬间就冲到了王翔面前，一掌对着王翔胸口拍了下来。这一掌快如闪电，带着毁天灭地的力量，整个通道都因为这一掌而气压骤降，周围的石壁纷纷龟裂。

王翔早有准备，对方是域主级，硬拼肯定吃亏，他立刻施展分布式分身术，身体一瞬间分出八个分身，分别从八个不同方向躲开，同时本体躲在侧面，排序剑凝聚出一道二分查找剑光，直指熵无极腰间的破绽。

他能直接看到熵无极的源代码，自然一眼就看出了对方腰间那块代码有轻微的内存泄漏，是个弱点。

"哦？分布式分身？有点意思，"熵无极不惊反笑，"看来你确实得到了一些上古传承，可惜，在绝对的实力差距面前，这些小花招没用！"

熵无极不闪不避，身体突然猛地膨胀了一圈，无数黑色的熵魔代码从他体内爆发出来，化作一张大网，瞬间就把八个分身全都笼罩在内。只听滋滋几声，那些分身接触到黑色大网，立刻就被熵魔感染，很快就腐败消融了。

同时，熵无极一掌变拍为抓，对着王翔本体抓了过来，速度比刚才更快！

"不好！"王翔心中一惊，没想到对方竟然这么快就破了自己的分身术，他连忙横剑格挡，排序剑挡在胸前。

砰！

一声巨响，王翔只觉得胸口像被大锤砸了一下，闷哼一声，一口鲜血喷了出来，身体像断线的风筝一样向后飞出去，重重撞在通道的石壁上，石壁当场裂开一个巨大的蜘蛛网纹路。

"王盟主！"

"万师兄！"

赵凯、钱磊、孙博三个人立刻冲了过来，扶住王翔，一脸焦急。玄真长老也立刻冲上来，挡在王翔前面，再次挥剑对着熵无极攻了过去。

"老东西，你也找死！"熵无极冷哼一声，迎上玄真长老，两人瞬间战在一起。玄真长老是内核级中期，在整个二进制宗也是数一数二的高手，但面对域主级的熵无极，完全不是对手，仅仅十几招，玄真长老就落入下风，身上已经挨了一下，嘴角流出鲜血，节节败退。

"玄真长老，退下！"王翔擦了擦嘴角的血迹，重新站起身，深吸一口气，调整了一下体内紊乱的算力。刚才那一掌确实让他受了点轻伤，但问题不大，熵无极的熵魔能量想要感染他的代码，结果被他直接用垃圾回收机制清理掉了，并没有造成持续性伤害。

他能看出来，熵无极虽然和熵魔融合了，获得了庞大的算力，但他的源代码也被熵魔感染得一塌糊涂，到处都是BUG和内存泄漏，整体结构臃肿不堪，运行效率极低。这就是他的弱点！

"怎么，还能站起来？不错不错，生命力挺顽强的。"熵无极停下手，看着王翔，脸上露出一丝戏谑，"我倒要看看，你还能接我几招。"

王翔没有说话，他知道，硬拼肯定打不过域主级的熵无极，必须用技巧，用他作为程序员的优势，找到对方的BUG，一击制胜。他打开源代码视野，仔细扫描着熵无极的代码结构，寻找着可用的漏洞。

就是现在！

王翔眼睛一亮，熵无极催动全身算力震碎死循环，他的核心域暂时打开了一个缺口，那个原本就有漏洞的地方，现在暴露得更加明显了！

王翔抓住这个机会，排序剑全力催动，一道带着SQL注入代码的剑光猛地射出，直指熵无极丹田核心域的那个漏洞！

剑光瞬息即至，直接命中了熵无极丹田处的那个漏洞点。

滋啦——！

就像强酸滴在棉花上一样，剑光瞬间侵入了熵无极的核心域，SQL注入代码开始疯狂破坏对方的数据库索引，拖库开始了！

熵无极脸色猛地一变，他能感觉到，自己丹田核心域的数据库正在被人攻击，大量的算力正在不断流失，这种感觉非常难受，就好像有人在悄悄掏空他的根基一样。
