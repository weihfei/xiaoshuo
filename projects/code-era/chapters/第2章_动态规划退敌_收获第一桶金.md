# 第2章 动态规划退敌，收获第一桶金

沉重的脚步声从村道那头传来，每一步踩在泥土上，都让地面微微一颤，隔着几十步就能感觉到那股凶戾的气势。

疤脸走了过来，满脸横肉因为愤怒而扭曲变形，那道从额头左上角一直划到下巴右下角的刀疤，在晨光下显得格外狰狞，配合他那双铜铃大的充血眼睛，确实能把小孩子吓得半夜哭醒。他身高接近两米，比王翔现在这个身体高出整整一个头，肩膀宽得像是门板，全身肌肉虬结，充满了力量感，手里提着那柄足足有四十斤重的精铁开山刀，刀身反光晃得人睁不开眼睛。

他走到离木屋还有十几步的地方停下，脚步稳稳站住，目光先扫过门口倒在地上的张三尸体，瞳孔微微一缩，然后又移到王翔手里那把泛着淡绿色微光的铁剑上，眉头一下子皱了起来。他认出这把剑就是原身万相那把破得不能再破的新手铁剑，怎么现在看起来完全不一样了？还泛着绿光？

"你居然杀了张三？"疤脸声音低沉，像是闷雷从胸腔里滚出来，震得人耳朵嗡嗡响，"看不出你这小杂种命还挺硬，挨了我一拳不仅没死，还能杀了我的人？"

王翔握着排序剑，站在木屋门口，身体站得笔直，后背靠着门框，眼神平静地看着对方，没有丝毫慌乱。反正伸头一刀缩头也是一刀，慌也没用，何况他已经看过源码，知道对方弱点在哪里，心里有底得很。

"你杀了我父母，抢了我家矿地，又把我打成这样，"王翔声音不高，但每个字都很清楚，"今天，该算算了。"

"算账？"

疤脸像是听到了天底下最好笑的笑话，突然哈哈大笑起来，笑声粗哑，像是破锣在敲，震得旁边树梢上的叶子都哗哗往下掉，"就凭你？一个刚入门的小比特战士，一段的修为，也敢跟我疤脸算帐？我在这黑铁聚落横行这么多年，碾死你就跟碾死一只蚂蚁一样简单！"

笑声骤停，疤脸脸上的笑容瞬间消失得无影无踪，取而代之的是几乎要溢出来的滔天杀意，他朝前走了一步，开山刀微微提起，刀身上的寒气让人皮肤都泛起凉意。

"我承认你有点邪门，挨了我一拳还能站起来杀人，但是那又怎么样？等级摆在那里，你比特战士一段，我比特战士九段，差了整整八级，你能翻得了天？"疤脸语气里充满了不屑，"今天我就把你碎尸万段，给张三报仇，然后这块矿地，还是我的！"

王翔内心平静得像是一潭深水，没有因为对方的话有丝毫波动。他抬起眼睛，目光缓缓落在疤脸身上，下意识地开启了源码视觉——这已经成了他的本能，遇到任何东西，先看源码再说。

下一秒，疤脸的完整源代码就清晰地浮现在了他的眼前，每一行每一句都看得明明白白：

```
// 疤脸 - 零号帮头目
ID: 84759
Level: 比特战士 9段
Health Points: 280/280
Mana: 0 (不会技能，纯物理)
Damage: 12-20
Defense: +8
Equipment: 精铁开山刀 (blue quality), 精铁胸甲 (blue quality)
Skills: 狂斩，野蛮冲撞，大力劈砍
Code smell: 
  - 五年前被前任村长砍中胸口，旧伤未愈，核心代码内存泄漏约15%
  - 长期暴饮暴食，肠道代码有BUG，运行不稳定
  - 发力习惯不好，每次重击后有0.5秒僵直
Weakpoint: 
  - 胸口旧伤位置，击中后会引发内存溢出，导致整个进程卡顿三秒
Special: 
  - 携带一块未知代码碎片，从碎片区挖出来的，来源不明
  - 碎片有微弱能量波动，目前无法解析
```

王翔眼睛一下子亮了。

果然不出所料，弱点明明白白标在那里，还有意外收获，真的带了一块未知代码碎片，正好对应大纲里要埋的伏笔。这源码视觉真是太好用了，相当于开了全图挂，对方什么都给你看得清清楚楚，怎么打怎么赢。

"怎么？吓傻了？不说话？"疤脸看到王翔只是盯着他看，不说话，以为王翔是怕了，更加嚣张，气焰更加嚣张，"现在跪下来磕三个响头，自己抹脖子，我可以给你个痛快，让你少受点活罪。"

王翔从源码视觉里退出来，轻轻摇了摇头，手指微微用力，握紧了手里的排序剑，剑柄贴着手心，温润舒适，那是他亲自重构出来的武器，感觉就是不一样。

"废话少说，动手吧。"

一句话，不卑不亢，直接亮剑。

疤脸愣了一下，似乎没想到这个濒死的小鬼居然这么硬气，顿时怒极反笑："好！好得很！真有种！既然你找死，那老子就成全你！"

话音未落，疤脸猛地一声大喝，脚下发力，整个人像是一辆重型战车，直接冲了上来，沉重的开山刀被他高高举起，借着冲刺的力道，居高临下，一刀朝着王翔的脑袋狠狠劈了下来！

刀风呼啸，空气都被这一刀劈得开裂，呼呼作响，力道之大，可想而知。这一刀要是劈实了，以王翔现在的防御，绝对是一刀两半，连抢救的机会都没有。

换了原身那个吓得手都抖的弱小子，肯定躲不开，必死无疑。但现在里面换了个来自地球的程序员灵魂，那就完全不一样了。

疤脸的整个发力过程，在王翔眼里就是一行行清晰的代码，从腿部发力，到腰部扭转，到手臂挥刀，每一个节点，每一步路径，哪里快哪里慢，哪里会有空档，看得一清二楚。

就像是调试代码的时候，断点下在那里，每一步怎么走都看得明明白白。

王翔脚步轻轻一晃，重心下沉，提前半秒钟侧身，恰到好处地躲开了这开山一刀，刀锋从他肩膀旁边几厘米的地方擦过去，带起一阵劲风，吹得他头发都飘起来了，但就是没碰到衣服。

紧接着，开山刀劈在旁边的木门上，咔嚓一声巨响，那扇本来就破破烂烂的木门，直接被劈成了两半，木屑飞溅，不少木屑都飞到了王翔脸上，他巍然不动，连眼睛都没眨一下。

"嗯？躲开了？"

疤脸有点惊讶，拿刀的手都顿了一下。他真没想到，王翔一个挨了他重拳，濒死快一天的小鬼，居然还能有这样的反应速度，恰到好处躲开了他这一刀。这不可能啊，在他预想里，这一刀下去，直接就把小王劈成两半了。

"有点本事，难怪能杀了张三。"疤脸冷哼一声，胸腔里吐出一口浊气，"不过躲开一刀没用，今天你插翅难逃！"

疤脸确实凶捍，一击不中，立刻第二刀就上来了，招招狠辣，刀刀直奔要害，开山刀挥舞得密不透风，一刀接着一刀，丝毫不停歇，整个木屋门口方圆十几步，都被他的刀气笼罩，肉眼可见的劲风吹得周围杂草都伏倒在地。

打斗声这么大，早就惊动了黑铁聚落的村民，大家都远远地跟过来，躲在树后面墙后面，探头探脑地看，小声议论着。

"我的天，疤脸亲自出手了，这下万相那孩子肯定没了。"一个老头叹了口气，声音压得很低，满是惋惜。
"唉，谁说不是呢，好好一家三口，就这么没了，真是造孽啊。"旁边一个老妇人接话，语气里也充满了同情。
"可是刚才我明明看到万相杀了张三啊，张三可是比特五段，怎么说没就没了。"一个年轻小伙子小声说道，他刚才看得比较清楚。
"那肯定是偷袭，张三没想到万相居然还能站起来，不防备才中招的。"一个中年人摇了摇头，"疤脸是什么人？比特九段，在咱们这一片都是数得着的高手，万相一个一段小鬼，怎么可能打得过？我看啊，撑不过三刀。"
"嘘...小声点，别让疤脸听到了，不然咱们都没好果子吃，他那个脾气你又不是不知道。"

议论声断断续续飘进王翔耳朵里，但他根本不在意，注意力完全放在了疤脸的刀法上，冷静地不停地走位躲避，从来不跟疤脸硬拼，就是绕着圈子跑，让疤脸的每一刀都劈在空处，白白浪费力气。

你别说，这种战术看起来有点怂，但非常有效。

疤脸砍了十几刀，一刀都没碰到王翔衣角，每次都被王翔提前躲开，像是王翔长了透视眼，早就知道他要砍哪里一样，这让疤脸越来越愤怒，火气越来越大，喘气也越来越粗，额头上豆大的汗珠不停地往下掉，顺着脸颊流进脖子里，浸湿了粗布衣襟。

"你特么还是不是男人！只会躲吗？有种跟我正面硬拼！"疤脸砍不着人，气得怒吼，声音都变调了。

王翔只是微微一笑，也不答话，就是继续躲。

正面硬拼？我傻啊？你等级比我高八级，攻击比我高两倍，血量是我的五倍，我跟你正面硬拼，那不是找死吗？

打架又不是比武招亲，非要讲规矩讲公平，能赢就行。他现在就是以逸待劳，你砍我躲，你消耗力气，我保存体力，反正你的弱点我都知道，等你力气耗得差不多了，内存泄漏顶不住了，我再给你致命一击，这不香吗？

这其实就是程序员解决问题的经典思路——动态规划。把一个大问题分解成多个最优子结构，分步解决，先消耗，再寻找最优解，性价比最高，风险最小。

要是换了这个世界的原住民，哪里会懂这个？他们从小接受的教育就是，修炼就是感悟源代码，打架就是比拼等级蛮力，谁等级高谁力气大谁赢，根本不懂什么战术优化，什么分步消耗，寻找最优解。

这就是**方法论的降维打击**。不是等级碾压，是思维方式碾压。

就这么，疤脸一口气砍了五十多刀，整个人都快喘成狗了，速度明显慢了下来，挥刀的幅度也变小了，准确度更是下降了很多。本来他就是靠爆发力吃饭，这么连续猛烈输出，消耗极大，何况他胸口还有旧伤，内存泄漏，越是发力，泄漏越严重，现在整个能量运行都不顺畅了。

王翔能清晰地看到，每次疤脸挥刀到极致，胸口旧伤位置都会微微一顿，出现一个非常短暂的卡顿，这个卡顿一开始只有零点几秒，现在已经变成了快一秒，越来越明显。

机会，一点点成熟了。

王翔眼睛微微眯起来，紧紧盯着那个卡顿点，等待着最佳时机。他不着急，现在时间站在他这边，疤脸越打越虚，他越打越稳，没必要急着出手，一击不中，反而被动，等机会最好的时候再出手，一剑就能定乾坤。

又砍了十几刀，疤脸已经是满身大汗，胸口剧烈起伏，每一次呼吸都带着粗重的风箱声，像是要把肺都咳出来一样。他确实没想到，这个看起来弱不禁风的小鬼这么能躲，躲到现在都不累，而他自己已经快没劲了。

"你...你特么..."疤脸喘着粗气，指着王翔，话都说不连贯了。

王翔看到疤脸又一次蓄力，挥出一记大力劈砍，这一刀几乎用了他剩下的所有力气，幅度很大，收势也会很慢，旧伤卡顿一定会非常明显。

就是现在！

王翔心中一声低喝，抓住了这个千载难逢的机会。

疤脸的大刀高高举起，狠狠劈下，和之前几十刀一样，但这一次，王翔没有躲。

他反而迎着刀风，突然朝前踏出一步，脚步稳稳扎住，整个人重心压低，排序剑贴着开山刀的刀背，轻轻一滑，借着疤脸劈砍的力道，顺势就抹向了疤脸的胸口。

那个旧伤弱点的位置！

这一剑，蓄谋已久，这一剑，快准狠，完全抓住了那个瞬间的卡顿，疤脸根本来不及变招，来不及防守！

"噗嗤！"

剑锋入肉的声音清晰可闻，不深不浅，正好命中弱点。

"呃啊——！"

疤脸发出一声惨绝人寰的惨叫，声音像是被人捏住脖子的鸭子，完全变调了，疼得他浑身都颤抖起来，那把沉重的开山刀哐当一声掉在地上，他自己也捂着胸口，连连后退。

他真的不敢相信，王翔居然一剑就刺中了他的旧伤！

这个伤口是他五年前跟前任村长拼命留下的，这么多年了，除了他自己，根本没人知道这个秘密，这个小鬼怎么可能一下子就找到？他开了天眼吗？

被一剑刺中旧伤，瞬间就引发了内存溢出，原本就泄漏的内存彻底崩了，整个核心进程直接卡住，就像是程序跑飞了一样，他全身都僵住了，半天动弹不得，连话都说不出来。

这三秒的卡顿，足够王翔做很多事情了。

王翔眼神一冷，没有丝毫犹豫，趁你病要你命，这个道理他懂。他手腕发力，握住排序剑的剑柄，狠狠往里一送，剑锋又深入了几分，然后猛地绞动了一下。

这一下，彻底断了疤脸的生机。

"噗——！"

疤脸张大嘴巴，一大口鲜血喷了出来， mixed着破碎的内脏碎片，直接喷了王翔一身，血腥味瞬间弥漫开来。他眼睛瞪得大大的，像是要从眼眶里凸出来，充满了不敢置信，死死盯着王翔，嘴唇哆嗦着，半天才能挤出一句话：

"你...你怎么知道...我这里有伤..."

王翔站在他面前，手里握着剑，静静地看着他，语气平淡，听不出喜怒：

"你杀我父母的时候，怎么不问问他们，为什么打不过你？"

一句话，说完，王翔猛地抽出排序剑。

鲜血喷涌而出，洒了一地，疤脸庞大的身躯晃了晃，然后轰隆一声，直直倒了下去，砸得地面都微微一颤，扬起一阵尘土。

王翔低头看去，疤脸的源代码状态已经变成了TERMINATED，进程彻底结束。

比特战士九段，黑铁聚落的一霸，祸祸了这片地方十几年的疤脸，就这么死了。死在一个刚刚入门的比特战士一段手里，死在一剑穿心，正中原点。

围在远处看热闹的村民们，这下彻底看傻了。

一个个张大了嘴巴，能塞进一个拳头，半天说不出话来，现场一片寂静，只剩下风吹树叶的沙沙声，还有王翔轻微的呼吸声。

刚才还凶神恶煞不可一世的疤脸，居然就这么死了？被一个十八岁的小鬼，一剑就解决了？

这太离谱了！这根本不符合大家的认知啊！等级差了八级，怎么可能一剑秒杀？

过了好半天，才有人小声嘀咕了一句："我...我没看错吧？疤脸死了？"
"死了...真死了，你看那地上那么多血..."
"我的天...万相这小子...藏得太深了...居然这么厉害..."
"以后...黑铁聚落，没有零号帮了..."

议论声渐渐起来，从小声嘀咕变成了大声讨论，震惊之后，就是压抑不住的兴奋。这些年，零号帮疤脸作威作福，大家都受够了，敢怒不敢言，现在疤脸死了，相当于压在大家头上的一座山倒了，怎么能不高兴？

王翔对此浑然不在意，他拔出剑，在疤脸衣服上擦掉剑上的血，然后想起源码里说的那块未知代码碎片，于是蹲下身，开始在疤脸怀里摸索。

果然，没摸两下，就摸到一块冰凉坚硬的东西，掏出来一看，是一块指甲盖大小的黑色碎片，质地像是黑曜石，表面刻着奇怪的纹路，歪歪扭扭，确实像是二进制代码，但排列方式又跟正常的代码不太一样，散发着非常微弱的能量波动，现在王翔实力太低，根本解析不了。

"果然有货。"王翔笑了笑，把这块碎片收进了怀里的布袋子里。反正现在看不懂，先收着，按照小说里的惯例，这种莫名其妙捡来的碎片，肯定是后面大剧情的钥匙，说不定就是上古遗迹的开门钥匙，先留着准没错。

解决了疤脸，接下来就是收获战利品了。王翔也不客气，疤脸抢了他家的矿地，杀了他父母，现在拿他点东西，天经地义，利息都没算他的。

搜身，很快就有收获。一个牛皮钱袋，沉甸甸的，打开一看，里面有六十八个铜板，还有十五块低纯度比特结晶，这就是今天的收获，第一桶金到手了。除此之外，还有疤脸那柄精铁开山刀，蓝色品质，伤害12-20，比王翔的排序剑伤害高多了，但王翔用惯了自己亲手重构的排序剑，顺手，而且排序剑有排序斩特效，打群架特别好用，所以他把开山刀收起来，以后缺钱了卖掉，换点比特结晶也好。

还有疤脸身上那件精铁胸甲，也是蓝色品质，加八点防御，王翔现在穿的还是原身那件破烂粗皮衣，正好换上。穿上去大小还算合适，就是有点重，习惯了就好，防御一下子提升了不少，以后挨打也能多扛几下。

搞定这些，零号帮群龙无首，张三死了，疤脸死了，李四跑了，自然也就树倒猢狲散了，对王翔再也没有威胁。

王翔站起身，拍了拍手上的灰，看向远处围在那里看热闹的村民，声音清晰地传了过去：

"各位乡亲，疤脸已死，零号帮从此不存在了。我万家原来那块矿地，本来就是我家祖上传下来的，从此还是我万家的， whoever有意见，现在可以站出来说。"

声音不高，但足够每个人都听到，现场瞬间安静了一下，然后没人说话，过了几秒钟，不知道是谁先鼓的掌，掌声渐渐响起来，越来越大，最后变成了一片欢呼。

这些年，零号帮欺压大家，苛捐杂税保护费，不知道害了多少人家，现在疤脸死了，大家心里都乐开了花，怎么可能有意见？

王翔笑了笑，悬着的一颗心放了下来。

至此，原身的仇报了，矿地拿回来了，他也正式在黑铁聚落站稳了脚跟，第一步，成功迈出去了。

他叫了两个跟原身父母关系还不错的村民帮忙，把张三和疤脸的尸体拖到村外，扔到铁锈荒原边缘，那里有不少野怪，晚上就会被拖走，不用自己挖坑埋，省事。

处理完尸体，王翔回到破木屋里，关好门，终于可以安静下来调息了。

刚才那一场战斗，虽然赢了很轻松，但也消耗了不少能量，正好趁着这个机会，吸收几块比特结晶，巩固一下刚刚优化好的经脉，顺便提升一下修为。

王翔从钱袋里拿出三块低纯度比特结晶，放在手心，摆好姿势，按照自己优化好的运行路径，开始吸收能量。

低纯度比特结晶，里面大部分都是杂质，只有不到五分之一是纯净能量，原住民吸收的时候，因为看不到源码，只能慢慢感悟，一点点过滤杂质，一块结晶就要吸收大半天，效率低得令人发指。

但王翔不一样，他能直接看到哪些是纯净能量，哪些是杂质，直接用快速过滤算法，二分查找区分，一秒钟就能区分开，杂质直接扔掉不要，只吸收纯净能量，吸收效率一下子提升了不知道多少倍。

一块比特结晶，不到十分钟就吸收完了，所有纯净能量都融入经脉，杂质排出体外，干干净净。

王翔连续吸收了三块，感觉到能量在经脉里不断沉淀，他能清晰地感觉到，自己的修为稳稳地从比特战士一段，升到了比特战士二段，而且根基非常稳，没有任何虚浮。

从一段到二段，就用了不到一个小时，三块结晶，这速度要是让原住民看到，估计能直接吓死。他们从一段升到二段，最少也要三五个月，运气不好三五年都升不上去，你这一小时就升了？开外挂了吧？

但对王翔来说，这不过是算法优化的基本操作而已。同样的时间，你O(n²)，我O(logn)，效率就是差这么多，不服不行。

吸收完结晶，修为巩固好了，王翔站起身，伸了个懒腰，全身骨头噼噼啪啪响了一阵，说不出的舒服。

他走到窗边，推开那扇已经破了的木窗，外面的晨光一下子涌了进来，照亮了这间破破烂烂的木屋，也照亮了他脚下的路。

王翔看着窗外，黑铁聚落错落的土房子屋顶冒着袅袅炊烟，村民们开始了新一天的生活，远处就是铁锈荒原，灰蒙蒙的天际线一直延伸到视野尽头，那里有低级野怪，也有未开垦的矿地，更远处就是碎片区，传说中有上古遗迹，也有无数危险。

这就是比特神域，他的新世界。

昨天这个时候，他还是地球互联网公司那个天天996的程序员王翔，改BUG改到心脏骤停，猝死在电脑前。今天，他就是黑铁聚落的万相，父母双亡，仇已经报了，矿地拿回来了，手里有剑，怀里有结晶，还有一个能看源代码的金手指。

命运这东西，真的太奇妙了，谁能想到呢？

不过既然来了，那就既来之，则安之。在这里，好好活下去，活出个人样来。

这个世界，万物皆代码，原住民只会调用API，感悟源码，而他能直接看，直接改，直接重构，那还有什么好说的？

他就要用手里这把剑，用他脑子里这些软件工程知识，在这个由代码构成的世界里，打出一片属于自己的天地，开创一个新的纪元。

说不定哪天，还能找到回去的路，就算回不去，那也比在地球天天996挤地铁还房贷强多了，在这里，至少能活得像个人。

王翔握紧了手里的排序剑，剑身轻轻震颤，泛着淡淡的绿色微光，那是他在这个世界亲手重构的第一件作品，也是他梦想开始的地方。

接下来，好好修炼，攒够路费和报名费，下个月就是二进制宗在附近城邦招学徒的日子，去考比特学徒，考上了就能进宗门，见识更广阔的世界，学到更高级的修炼方法，比在这个小聚落待着强多了。

他已经走出了第一步，接下来，一步步往前走，总会到达目的地。

阳光洒在他身上，温暖而明亮，新的一天，新的人生，刚刚开始。

（第二章完）
