# 第35章 混沌初定 第一城

踏出白桦林的那一刻，我就感觉到了不同。

空气里弥漫着一股淡淡的焦糊味，像是服务器过载烧了主板的气味，周围的空气都在微微颤抖——就像是屏幕信号不稳，时不时闪一下，连影子都跟着歪歪扭扭，一会长一会短，看着说不出的不舒服。脚下的土地，有些地方已经变成了灰黑色，摸上去冰凉刺骨，那是熵魔长年侵蚀留下的痕迹。

"大家小心，这里数据流不稳，容易被熵魔偷袭。"零音走在最前面，右手一直按在刀柄上，银色真气隐隐流转，她的感知比我们所有人都敏锐，"我能感觉到，四周不少低阶熵魔盯着我们。"

叶清语轻轻往我身边靠了靠，微凉的手指插进我的指缝，和我紧紧握在一起。她抬头看我，眼神里有担忧，但更多是坚定："这里能量波动太乱，我带了足够的银针和修复符文，真有人被熵魔感染，我能救。"

我拍拍她手背，笑了笑："放心，我们有产权玉牌，域名权限在我手里，熵魔不敢放肆。"

我举起左手，那枚青色的混沌域产权玉牌在夜色中微微发光，一丝丝暖流从玉牌里涌出来，顺着手臂流进丹田。我把丹田真气注入玉牌，激活了域名权限。

一瞬间，玉牌爆发出一层淡青色光罩，从我们脚下向四面八方扩散开去。光罩过处，原本颤抖扭曲的空气瞬间稳定，那些隐约可见的黑色熵魔残影，一碰到光罩就像雪碰到开水，瞬间蒸发得干干净净。

"真有用！"赵石眼睛一下子亮了，伸手摸了摸身边空气，"刚才还觉得经脉发痒，现在舒服多了！"

"产权玉牌就是这点好，"我收起玉牌，光罩收敛了些，只维持着基本防护圈，"拿到域名产权，整个混沌域的根目录就是我们说了算，熵魔属于未授权进程，自然进不来核心防护区。走，往中心去，按地图说，上古域名核心在中心盆地，地基完好，适合建城。"

一行人排成纵队，跟着我往混沌域中心走。大锤扛着他那柄精铁大锤走在队尾殿后，他每走一步，大锤都在地上顿一下，咚一声闷响，躲在草丛里的小熵魔被震得现行，接着就被光罩蒸发，省事得很。偶尔有胆子大的还想跟着，他就一声大吼，吼声震得树叶哗哗往下掉，那些低阶熵魔吓得跑得比兔子还快。

走了不到十里，前面乱石林里突然哗啦一声响，三只成年熵魔冲了出来。这些家伙长得就像一团黑粘液，没有固定形状，流着朝我们扑过来，所过之处，石头都被它们腐蚀得滋滋冒烟。

"小心！它们能吞噬算力！"零音话音未落，已经拔刀冲上去，刀光一闪，银色刀气劈出去，直接把最前面那只熵魔劈成两半。

但预想中的解决并没有来——被劈开的两半熵魔，每一半都很快收缩成型，变成两只更小的熵魔，继续扑过来。

"不好，它们会分裂！"大锤冲上去，一锤砸在一只熵魔身上，砸得它四分五裂，结果碎片又分裂出更多小熵魔，满地都是，爬得飞快。

眨眼功夫，三只熵魔分裂成十几只，越来越多，再这么打下去，用不了多久我们就要被淹没了。众人都紧张起来，纷纷拔刀戒备。

我站在原地没动，眼睛快速扫过那些熵魔。在我眼里，它们每个都带着相同的特征码，就像是正则表达式能匹配的字符串。我笑了笑，气运丹田，对着那些熵魔轻喝一声：

"正则匹配，批量清除。"

我抬手一挥，一道青色数据流从丹田发出，瞬间扫过整个战场。就像电脑执行了一遍正则替换，所有带着熵魔特征码的进程，一瞬间全部被终止。

十几只熵魔，不管大小，全部定格在原地，然后瞬间化作飞灰，干干净净，一只不剩。

整个过程——不到三秒钟。

零音收刀站着，愣住了。大锤举着大锤，也愣住了。赵石等人瞪大了眼睛，嘴巴张得能塞进一个拳头。

"这...这就完了？"大锤挠挠头，"我锤子还没热呢。"

"熵魔分裂繁殖，本质就是同一个特征码不断复制，用正则一次性全部匹配出来直接清除，比一个个砍省事多了。"我收了真气，笑着说道。

叶清语走上前，掏出绢布，轻轻帮我擦去额头上的汗水。她眼睛亮晶晶的，带着笑意："你刚才用正则清除的时候真好看，行云流水，一点不拖泥带水。"

我笑着握住她的手："有你在旁边看着，我当然要表现得好看点。"

零音在旁边轻轻哼了一声，转过头看向前面树林，不过我瞥见她嘴角微微翘了一下，也没戳破。

继续往前走，又走了半个时辰，我们路过一片废墟。这里原本应该是个小型聚落，现在只剩下断壁残垣，墙上沾满了黑色的熵魔腐蚀痕迹。废墟入口，一块半截石碑倒在地上，上面还能辨认出字迹。

赵石蹲下来，擦掉石碑上的灰尘，辨认着上面的字，脸色越看越阴沉。他抬头对我们说："是黑石城残碑，十五年前，这里有一百多户人家，三百多口人，全是受不了万维联盟的算力税，逃到混沌域开荒的。"

"然后呢？"大锤问。

"然后？"赵石苦笑一声，"万维联盟说他们私开域名，犯法，派了一队人过来，能带走的全抢走，放了一把火，还故意引了一堆熵魔过来，把剩下的人全吃了。你看这儿——"

他指着石碑底部，那里刻着一行小字：**万维联盟，不得好死**。字迹已经被雨水冲刷得模糊了，但那股怨气，隔着一百多年还能摸到。

所有人都沉默了，气氛一下子沉得像铅。这些事，大家或多或少都听过，但亲眼看到残碑，感觉还是不一样——这些都是活生生的人，就因为不想被吸血，想找个地方安身，结果落得这么个下场。

"万维联盟就是这么干的，"我蹲在石碑前，用手抚摸着那行模糊的字迹，声音低沉，"他们不允许任何人在他们垄断之外开域，谁开，他们就灭谁。今天这石碑，就是他们的罪证。等我们在这儿站稳了，就把这块碑立在开源城门口，让所有人都记住——我们为什么要来这里，我们要反抗的是什么。"

"对！立在城门口！让所有人都记住！"赵石咬着牙，恶狠狠地说道。

众人默默绕过废墟，继续往前走。每个人脸色都不好看，但脚步却比之前更坚定了。

又走了一个时辰，天边泛起鱼肚白，天快亮了。我们转过一片乱石岗，前面豁然开朗，一个巨大的盆地出现在眼前。盆地中心，一块巨大的水晶基岩露出地面，隔着几里地，都能感觉到里面蕴含的庞大算力波动。

"就是那儿了！"赵石激动得声音都抖了——他这辈子都在中小势力夹缝里求生，从来没想过能有一块属于自己的域名，"上古域名核心！真的还在！"

我们加快脚步，走到水晶基岩跟前。这块基岩真大，足有一个篮球场那么大，高出地面两三丈，表面刻满了上古铭文，经过亿万年岁月，很多铭文已经模糊，但那股厚重的历史感依旧扑面而来。我伸手摸上去，冰凉的触感顺着指尖传来，基岩内部，隐隐有数据流在缓缓流动——说明核心其实一直都在转，只是缺了一块，所以才不稳定。

绕着基岩走了一圈，我们找到了那个缺口。基岩朝北一面，确实缺了老大一块，一人多高，断面黑黢黢的，全是熵魔侵蚀的痕迹，黑色晶屑一碰就往下掉。

"就是这里坏了。"大锤伸手敲了敲缺口周围的晶壁，发出沉闷的响声，"问题不小啊，万相盟主，咱们这点人能修好吗？我听说之前有个大势力来了几百人，修了半年都没成，最后被熵魔攻破核心，全灭在这儿了。"

赵石也皱起眉："是啊盟主，修复域名核心得花海量算力，我们二十多个人，就算把所有算力都投进去，不知道要修到猴年马月。万维联盟那时候打过来，我们根本挡不住。"

其他人也沉默了，脸上都带着担忧。他们能跟着来混沌域，已经是把命赌上了，但真看到这么大烂摊子，心里还是没底。

我笑了笑，没急着解释，转头看向叶清语。她正伸手轻轻摸着缺口边缘的黑色晶屑，感觉到我看她，抬起头对上我的目光，眼睛弯成月牙："我相信你，你肯定有办法。"

就这么一句话，我心里一暖。不管什么时候，她都是这样，永远站在我这边，永远相信我。我走过去，轻轻帮她理了理被风吹乱的鬓角——她头发上沾了一点草屑，我拿掉草屑，她微微仰头对着我笑，阳光从盆地边缘照过来，落在她脸上，好看得让我心跳漏了一拍。

"放心，我有办法，而且这个办法，正好需要大家一起出力。"我转过身，对着所有人高声说，"万维联盟修复核心，都是顶层高手出大部分算力，底层小角色只能打打下手，所以效率低，一旦顶层算力不够，直接就修不下去。"

"但我们不一样，我们是开源盟，讲究的就是众人拾柴火焰高，每个人都贡献一点算力，大家一起参与修复，用分布式共识算法，每个人都来验证每一段代码，这样修起来，比集中式修复快好几倍！"

我掏出早就画好的修复分块图，展开给大家看："我把缺口分成了二十四块，正好我们二十四个人，每人一块，各自修复自己那块，然后通过共识算法确认，只要超过半数认没问题，这块就算修好，最后拼起来就是完整的了。"

所有人都愣住了——他们从来没听过这种修复方法。赵石挠挠头："盟主，这...这样真能行吗？每个人修自己的，拼起来不会出错吗？"

"不会，"我肯定点头，"万相框架我已经写好了分布式共识模块，每个人修好一块，框架会自动同步验证，出错了马上提示，重来就是了。比一个人修一整块效率高多了，而且就算哪个人出点错，也影响不了整个核心。"

零音第一个点头："我相信盟主的办法，我先来，哪块？"

"我也来！"大锤往前一站，瓮声瓮气，"俺力气大算力足，给俺一块大的！"

"算我一个！"
"我们都跟着盟主干！"

看到零音和大锤表态，其他人也纷纷站出来，没人退缩。我攥了攥手里的分块图，鼻子有点发酸——这么多人愿意信我，这份信任，我不能辜负。

我按照每个人算力大小分配了修复块，算力大的负责块头大一点，算力小的负责小块。然后把万相框架客户端通过传讯玉符传到每个人手里，教他们怎么用。这个客户端很小，只占一点点丹田空间，就能连接核心参与共识。

"好了，大家各就各位，注入真气，开始修复！"

一声令下，二十四个人分别站到自己负责的修复块前，同时运转丹田，把真气注入水晶基岩。万相框架自动启动，分布式网络建立起来，每个人的修复进度，都实时同步到所有人那里。

神奇的事情发生了。

一开始，修复进度走得很慢，每个人都在摸索，但随着第一块修复完成，超过半数共识通过，那块水晶瞬间恢复光泽，淡青色光芒从里面透出来。所有人都看到了希望，修复速度一下子快了。

一块接一块，修复完成的提示不断在框架里跳出来。每修好一块，核心的算力波动就稳定一分，整个盆地的空气都跟着清新一分。那些原本躲在远处窥视的熵魔，感受到核心重新稳定，吓得纷纷退走，不敢靠近。

一天...

两天...

三天...

我们轮流休息，吃饭就是简单啃点干粮，修复工作一刻没停。叶清语不光要修自己那块，还要帮真气耗竭的人恢复，忙前忙后，一句怨言都没有。每天晚上我替她按摩肩膀，她都靠在我怀里说不累，看着核心一点点变好，她比谁都开心。

第三天傍晚，最后一块修复完成，共识通过。

轰隆隆——

整个水晶基岩突然震动起来，爆发出一团耀眼的青色光芒，光芒直冲云霄，把半个混沌域的天空都照亮了。我能感觉到，整个混沌域的数据流都在欢呼，都在重新对齐根域名服务器，那些原本扭曲错乱的区域，都慢慢稳定下来。

"成了！"赵石大吼一声，激动得眼泪都出来了，一把抱住身边的人，"真成了！我们修好核心了！"

所有人都欢呼起来，有的跳起来大喊，有的互相拥抱，还有的喜极而泣。他们当中很多人，一辈子被万维联盟压迫，交了一辈子算力税，从来不敢想自己能有一块属于自己的域名——今天，他们做到了。

我站在水晶基岩最高处，从怀里掏出早就准备好的域名铭刻刀，深吸一口气，真气灌注到刀上，在基岩最上方，郑重刻下四个大字：

**开源城**

刻完最后一笔，我收起刻刀，真气催动，四个大字瞬间亮起，青色光芒凝聚不散，永远留在了水晶基岩上。

从今天起，混沌域有了第一座城池，开源盟有了第一个根据地，比特神域有了第一个开源共享的域名。

欢呼声更大了，二十多个人的吼声，在盆地里久久回荡，惊飞了盆地边缘树林里所有飞鸟。

等欢呼声渐渐平息，我正准备说几句定军心的话，盆地入口突然传来一阵轻微的能量波动——很隐蔽，但零音瞬间就察觉到了。她手按刀柄，就要冲过去："有人！"

"别冲动，"我拦住她，"应该不是敌人，让他自己出来。"

话音刚落，一个穿灰色长袍的老人从树后走出来。他头发胡子都白了，脸上布满皱纹，但眼神很亮。我认出来了，就是之前给我送地形图，又在流云城递纸条的那个神秘人。

他走到基岩底下，抬头看着我刻的"开源城"三个字，点了点头，赞道："好名字，开源...说得好，这个世界闭源太久了，确实该开源了。"

他从怀里掏出一块令牌扔给我，我伸手接住——令牌是狐尾形状，上面刻着一个九字，正是九尾狐族的令牌。

"我就是你猜的，一个老开源人。当年跟着九尾狐族一起对抗熵魔，全族都快死光了，我侥幸活下来，躲在混沌域这么多年，就是等着一个能拿到产权、敢来开基的人。"老人看着我，语气郑重，"九尾狐族遗迹在混沌域深处的地下溶洞，等你站稳脚跟，可以去看看，狐族留下了一件好东西，对付熵魔有用。"

我攥紧狐尾令牌："前辈到底是谁？为什么要帮我？"

老人笑了笑，往后退了几步，身影渐渐模糊："我是谁不重要，重要的是，当年创造者离开的时候说过，总有一天，会有一个懂源代码的人回来，重新开启这个世界。我不过是遵守诺言，等着你而已。对了，我在核心深处看到一块奇怪的代码碎片，像是一把钥匙，你以后实力强了，可以挖出来看看，可能和创造者留下的遗产有关。"

说完，老人身影彻底消失在空气中，只留下一阵风吹过，什么都没剩下。

我握着狐尾令牌，若有所思。钥匙...创造者遗产...看来这混沌域里，藏着不少秘密。那块代码碎片，应该就是传说中三枚代码钥匙之一，没想到这么快就有了线索。

"盟主，你看那边！"大锤突然指着盆地边缘的小山头，那里有个鬼鬼祟祟的身影，看到我们发现他，转身就跑。

零音眼神一冷，就要追："万维联盟的探子！我去杀了他！"

"不用追，"我拉住她，摇摇头，"让他回去报信吧，告诉欧阳震，我们已经在混沌域站稳了，有本事就来抢，我们接着就是。"

零音收刀站好，点了点头，不再多说什么。她就是这样，话不多，但我说什么她都听。

天色完全黑下来，我们在开源城核心广场点起篝火，每个人都拿出带的干粮和酒，简单办了个开城宴。没有山珍海味，都是普通干粮和劣酒，但每个人都吃得开心，脸上带着对未来的憧憬。

我吃完晚饭，走上水晶基岩顶端，眺望整个盆地。夜色里，盆地已经开始有虫鸣，空气清新，再也没有刚进来时那股焦糊味。

一只温暖的手从后面轻轻搂住我的腰，叶清语靠上来，把脸贴在我后背上，轻声说："终于有家了，真好。"

我转过身，把她搂进怀里。她头发上有篝火的烟味，还有淡淡的草药香，闻着特别安心。零音站在不远处的暗影里，手扶刀柄，警惕地望着盆地入口——她不凑热闹，但她会帮我们守着。

"是啊，有家了。"我看着怀里的叶清语，又看了看黑暗中那个挺拔的身影，心中感慨，"从今天起，这里就是我们的家，万维联盟想要拿回去，就得从我的尸体上踏过去。"

我低头，轻轻吻了吻叶清语的额头，她闭上眼睛，脸上泛起红晕，乖乖靠在我怀里。远处，篝火噼啪作响，人们的说笑声传过来，温暖而安稳。

而在盆地外面十里地，那个逃出来的万维联盟探子，气喘吁吁跑到一片树林里，掏出传讯玉符，对着玉符低声说："主子，不好了...万相他们真把核心修好了，还建了城，叫...叫开源城..."

玉符那头，欧阳彻听完，沉默了很久，最后发出一声阴冷的笑："好...好得很...没想到他们真能修成。没关系，让他们先高兴几天，我已经请了万剑宗的三位内核宗师，再过几天，就去混沌域，替我...把万相的人头带回来！"

探子应了一声，收起玉符。他不知道，就在他身后不远的大树后面，一个开源盟哨兵已经握紧短刀，悄无声息摸了上来...

风暴，正在酝酿。但我们已经不再是当年那支无家可归的流浪队伍了。

开源城，立。
