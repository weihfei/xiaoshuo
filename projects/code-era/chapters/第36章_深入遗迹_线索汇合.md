# 第36章 深入遗迹 线索汇合

开源城立，已经七天。

这七天里，整个盆地都变了模样。

赵石带着十几个汉子，硬生生把黑石城那半截残碑从十几里外抬了回来，稳稳立在城门入口。我亲自找了块整块的青石板，刻了一行字嵌在旁边：

**凡算力被剥削者，都可来此安家。**

字刻好的第二天，就有第一批流民找到了这里——三个一家人，男的被熵魔感染了半只胳膊，女的抱着个几岁大的孩子，饿得面黄肌瘦，站在城门口哆哆嗦嗦不敢进。守门的弟子按照我定的规矩，直接放了进来，叶清语亲自给那男人治疗，免费。

三天后，那男人就能下地干活了，出去一传十十传百，周边混沌域的流民像是找到了方向，源源不断往开源城来。不到七天，城里的人口就从二十几个变成了两百多，而且还在不断增加。

大锤带着人在盆地入口用当地石材砌了一道简易城墙，虽然不高，但挡住低阶熵魔和小股散兵足够了。他天天光着膀子干活，皮肤晒得黝黑，每次我去看，他都把大锤往地上一墩，咧着嘴笑："盟主你放心，城墙修得结实，万维联盟的人不来则已，来了我一锤子砸扁一个！"

零音带着五个感知敏锐的弟子，在城外五十里范围内拉了一道警戒线，每天巡逻，清理漏网的熵魔，第一天就清理了三只熵魔领主，带回不少熵魔结晶。她话不多，每天早出晚归，警戒线越拉越远，现在只要有敌人进入百里范围，我们提前一天就能得到警报。

我每天白天就在中心广场给新人讲万相框架的基本用法，教大家怎么用分布式节点同步修炼，晚上和叶清语一起整理医疗记录，和零音一起调整警戒部署。日子过得忙碌但踏实，看着从一片废墟一天天变成真正的城池，那种感觉，真的很奇妙——这是我们自己的家，每一块石头，每一间木屋，都是我们自己亲手建起来的。

第七天一大早，出去追踪万维探子的哨兵回来汇报了。

"盟主，那探子确实跑回流云城了，他给欧阳彻传了信，欧阳彻已经请了万剑宗的三位内核宗师，三天前就出发了，照着路程算，最多还有八九天就能到这儿。"

哨兵话音落下，帐篷里一下子安静下来。正在喝水的大锤放下水囊，手不自觉握住了锤柄："三个内核宗师？欧阳彻那小子下血本啊，居然请动这么大阵仗。"

赵石皱着眉，手指敲击着桌案："万剑宗的人都是疯子，个个都是死士，出手从来不留活口，三个内核宗师，不好对付啊。我们现在城里虽然有两百多人，但真正能打的内核级一个都没有，连字节级也只有我们几个......"

我笑了笑，没急着说话，端起茶碗抿了一口。茶是叶清语刚泡的，野生的清心茶，清香解渴。

"大家慌什么？"我放下茶碗，看着众人，"我们有产权玉牌，整个混沌域的域名权限都在我手里，他们是客场，我们是主场，真打起来，谁吃亏还不一定呢。正好，我也想试试，分布式防御到底好不好使。"

顿了顿，我继续说："正好神秘老人说的那块代码钥匙碎片还在核心深处没挖，我打算明天一早就带队伍出发，先去把碎片挖了，再按图索骥去找九尾狐族遗迹，把第三把钥匙拿到手，回来正好迎接万剑宗的客人。"

"我跟你去！"零音第一个开口，手按刀柄，"我去探路，熵魔我来清。"

"我也去！"大锤瓮声瓮气，"我力气大，挖东西砸石头我在行，遇到硬茬子我来扛！"

叶清语看着我，笑着说："我自然也跟你去，万一有人受伤，我好治疗。"

我点点头："好，就我们四个，轻装简行。赵石你留下来主持大局，城墙加固，流民安置，都靠你了，记住，重心先稳，不要主动出击，只要守住城门，等我们回来就好。"

赵石站起身，抱拳："盟主放心，我在城在，绝对不会出问题！"

事情就这么定了。四个人分头准备，一天之内收拾停当，带足干粮、水、疗伤草药，第二天一早，出发。

水晶基岩就在城中心，入口在基岩底部朝北一面的暗缝里，老人早就给了提示。零音围着基岩底部转了一圈，很快就找到了那道暗缝——也就一人多高，半丈宽，里面黑黢黢的，一股古老冰冷的气息从里面飘出来。

"就是这儿了。"零音退回来，对我点点头。

我运起真气，丹田一股暖流涌上来，掌心亮起淡淡的青色光芒，光芒照亮了前面的道路："我走前面，你们跟着，小心点，里面可能有当年留下的机关。"

侧着身子，一个接一个钻进暗缝。通道一开始很窄，越往里走越宽，石壁都是天然水晶，能反射我掌心的光芒，走了大概几十步，眼前豁然开朗，我们来到了一个天然的晶洞。

这晶洞真漂亮。

四壁都是纯净无瑕的透明水晶，我掌心的青光照上去，折射出满天星光，整个洞都亮晶晶的，像是把整个星空都装进来了。空气中的算力浓度浓得像是要滴下来，吸一口气，都感觉丹田真气轻快了几分——这里真不愧是混沌域域名的心脏，亿万年积累的算力，都凝在这块基岩里了。

洞中央，一块半人高的天然石台上，果然有一个巴掌大的凹槽，形状规则，边缘还有清晰的拼接痕迹，一看就是当年放钥匙的地方。

我们四个人围上去，我蹲下身，仔细看着那个凹槽。能感觉到凹槽里残留着淡淡的代码波动，和我灵魂里那块钥匙碎片波动频率一模一样，就是这儿了。

"我来挖，你们站远点，小心有机关。"我抽出排序剑，真气灌注剑身，开始小心翼翼清理凹槽周围松动的晶屑。几千上万年了，水晶都有些风化，一碰就往下掉渣，我不敢太用力，怕把里面的碎片弄坏了。

挖了大约一个时辰，剑刃终于碰到了硬物，发出清脆的叮当声。

"找到了。"我心里一喜，放慢动作，一点点把周边的晶屑清理干净，然后用剑刃轻轻把碎片撬出来。

一块巴掌大的青色水晶碎片，落在我掌心。

碎片边缘整齐，正好有一块锯齿状的缺口，和我怀里那枚狐尾令牌缺的那块一模一样。碎片表面，刻着半只九尾狐的纹路，线条流畅，透着古老的气息。

"真的是钥匙碎片！"大锤凑过来，眼睛发亮，"老爷子说的是真的！"

我从怀里掏出那枚狐尾令牌，比对着放到缺口上——咔哒一声轻响，严丝合缝，正好拼合在一起。

完整的九尾狐尾令牌，终于成型了。

就在拼合完成的瞬间，令牌突然爆发出耀眼的金色光芒，一道巨大的投影从令牌里投射出来，正好映在对面的晶洞石壁上。

是一幅地图！

地图上清晰标注着从我们现在的位置，往西北方向走三百里，有一座九尾山，山腹里就是九尾溶洞，一个红色的叉标记在溶洞最深处，旁边还有一行上古狐族文字。

"零音，你认得这字吗？"我转过头问。

零音走上前，仔细看了一会儿，点头道："认得，杀毒者传承里有上古文字字典。写的是——"她一字一顿念出来，"第三枚代码钥匙碎片，在此遗迹最深处，等待能读懂源代码的人前来开启。"

我心里一动，扭头看向零音："这么说来，还真对上了。我灵魂里带着一块，你手里有一块，这里还有一块......三块钥匙，正好对应创造者神殿的三锁，集齐了就能开门。"

零音眼中也闪过一丝惊讶："我拿到我那块的时候，只知道是上古权限钥匙，不知道具体用途，原来是这样。"

叶清语走到我身边，伸出微凉的手指，轻轻帮我擦去手上沾的水晶碎末，她指尖带着淡淡的草药香，动作轻柔："我就说肯定能找到，你看，这不就对上了吗？所有线索都指向那儿，说明我们走的路没错。"

我握住她的手，她的手微凉，很软："嗯，等我们从遗迹回来，拿到第三块钥匙，万剑宗那帮人也差不多该到了，正好给他们送一份大礼。"

零音已经检查完了整个晶洞，走回来摇摇头："没有其他机关，也没有别的东西了，地图和文字都出来了，我们可以出发了。"

四个人收好令牌，按照原路退出晶洞，重新把暗缝入口掩盖好，背上行李，朝着地图标注的西北方向出发。

从开源城到九尾溶洞，要穿过一片黑森林，叫噬魂黑森林，地图上标注这里是熵魔活动频繁区，变异野怪也多。零音走在最前面，她感知敏锐，能提前十几里感觉到熵魔的气息，大锤走在最后压阵，我和叶清语走在中间，一行人保持着战斗队形，稳步前进。

第一天走了大约八十里，路上遇到三波低阶熵魔，都是零零散散的，零音走在前面，远远发现，直接上去一刀一个，干净利落，根本不用我们出手。傍晚时分，走到森林边缘，遇到了第一只像样的对手——一只字节级巅峰的熵魔蜘蛛，躲在大树上，等着偷袭我们。

那蜘蛛个头很大，身子比磨盘还大，八条腿长满了倒刺，毒液顺着倒刺往下滴，落在地上，草瞬间就变黑枯萎了。它从树上扑下来的时候，带起一阵腥风，直奔叶清语而去。

大锤反应比我还快，怒吼一声，直接迎上去，一锤砸出去，正好砸在蜘蛛脑袋上。

砰！

一声巨响，蜘蛛脑袋被砸得稀烂，黑色的粘液溅了一地。但预想中的结束并没有来——碎裂开的蜘蛛身体，很快分裂出八只小蜘蛛，每一只都有脸盆大，张着毒牙扑过来。

"又是分裂，老套路了。"我笑了笑，也不慌，气运丹田，一道青色数据流扫出去，"正则匹配，批量清除。"

就像是扫了一遍病毒库，所有带着蜘蛛熵魔特征码的小蜘蛛，一瞬间全部定格，然后瞬间化作飞灰，干干净净。

前后不到五秒钟。

大锤收起锤子，挠挠头："还是盟主你这招省事，我砸完还得收拾残局，你这一下就干净了。"

叶清语已经找了块开阔地，捡了干树枝，架起篝火。她从包袱里掏出白天打到的野兔，处理干净，穿在树枝上烤，很快，滋滋的油声响起来，肉香飘得老远。

大锤一闻到肉香，肚子就咕咕叫了，也不客气，蹲在篝火旁边，盯着那只野兔咽口水。我靠着大树坐下，零音站在我对面，靠着另一棵树，手按刀柄，目光扫向四周黑暗，始终保持着警戒。

叶清语把烤得金黄油亮的野兔腿撕下来，先递给我，油汪汪的，撒了她带的香料，香气扑鼻："走了一天路，饿了吧，快吃。"

我接过来，咬了一口，外焦里嫩，咸香适中，味道真不错。她又撕了一块大的给大锤，再给零音递过去一块。零音接过，低声说了声谢谢，小口吃着，目光依旧没放松警戒。

"盟主，你说那九尾遗迹里头，真有狐族活着吗？"大锤啃着肉，含含糊糊问，"都这么多年了，当年全族都被灭了，还能有活口？"

我啃着肉，想了想："不好说，壁画上说是残余族人躲进去关闭了入口，说不定真有剩下的，就算人没了，东西肯定还在，那块钥匙碎片跑不了。"

零音突然开口："我能感觉到，越往里面走，熵魔的气息越少，反而有种很干净的能量波动，像是有人一直在维护着什么，不像没人的样子。"

大锤咽下嘴里的肉，眼睛一亮："那更好，真有活的狐族，我们就能问问当年到底发生了什么，说不定还有更多对付熵魔的好办法。"

"吃完早点休息，明天一早继续走，照这个速度，后天下午就能到了。"我拍了拍手上的油灰，说道。

夜里我和零音轮流守夜，我守上半夜，她守下半夜。篝火噼啪作响，林子里只有虫鸣，风吹过树叶沙沙响，很安静。叶清语靠在我肩膀上睡着了，呼吸均匀，头发上带着篝火的烟味和她特有的草药香，闻着很安心。

我轻轻搂着她，看着黑暗中的森林，心里想着钥匙的事。

灵魂里一块，零音一块，这里一块，三块集齐，就能打开创造者神殿，就能看到那个README文档，就能知道这个世界真正的真相，还有解决熵魔危机的办法。这条路，走得比我预想的顺利，没想到刚建城就找到了第三块钥匙的线索。

第二天一早，吃过早饭继续出发。越往森林深处走，树木越粗壮，阳光很难透下来，林子里阴沉沉的，空气也更潮湿，但确实像零音说的那样，熵魔越来越少，偶尔遇到一两只，也是低级的，随手就能解决。

路上我们还发现了一些狐族留下的路标，都是用狐爪在树干上刻的小九尾记号，隔几里地就有一个，说明我们走的路没错，这确实是当年狐族进出遗迹的老路。

下午的时候，我们路过一片开阔地，地上有很多碎骨头，都发黑了，一看就是很多年前留下的。零音蹲下来看了看，捡起一块碎骨头，皱着眉说："都是万剑宗弟子的骨头，看断口，都是兵刃砍的，应该是当年攻破狐族外城的时候留下的。"

我捡起一块骨头，骨头已经风化了，但上面还能感觉到淡淡的熵魔侵蚀痕迹。万维联盟联合万剑宗，灭了狐族，抢了他们的地盘，现在又来打我们的主意，真是走到哪里都改不了强盗本性。

"等我们拿到钥匙，打完这仗，回头把这里收拾一下，给狐族前辈立个碑。"我把骨头放下，沉声说。

第三天下午，太阳快落山的时候，我们终于走出了黑森林。抬头一看，前面一座巨大的石山拔地而起，山腹正中，一个巨大的溶洞入口敞开着，洞口上方，九条天然形成的钟乳石垂下来，远远看去，真像是九条尾巴垂在那里。

"九尾溶洞，到了。"大锤眯着眼睛看，兴奋得声音都抖了，"真找到了！"

我掏出狐尾令牌，令牌在我手里微微发热，像是在回应什么。"就是这儿了，走吧，进去看看。"

我们走到洞口，果然有一层看不见的能量屏障，拦在洞口。我把狐尾令牌举起来，注入真气，令牌亮起金光，屏障像是被钥匙打开的门，无声无息分开一道口子，正好容我们四个人过去。

进去之后，屏障又自动合上，外面的风声一下子被隔绝了，整个溶洞安静得只剩下我们四个人的脚步声和呼吸声。我掌心亮起青光，照亮前面的路，溶洞很高很大，钟乳石石笋随处可见，有些石笋上刻着古老的文字和图案，一看就是人为刻上去的。

走了大约一里地，前面洞壁上出现了一幅巨大的壁画，颜料虽然褪色了，但轮廓还很清晰。我们四个人停下来，仰头看这幅壁画。

壁画分三部分。

第一部分画的是狐族全盛时期，漫山遍野都是狐族，每个人都坐在那里码代码，天上飘着各种代码片段，很多其他种族的人都来狐族请教问题，狐族都敞开大门，毫无保留分享自己的代码。壁画角落写着一行字：**狐族之道，开源共享，天下代码，人人可读写。**

我心里微微一动，这不就是我们现在在做的事吗？

第二部分画的是战争，一大群穿着统一服饰的武士，打着万维联盟前身的旗号，攻进了狐族领地，所过之处，烧杀抢掠，狐族战士拼死抵抗，尸横遍野。壁画上画着对方首领说的一句话："代码岂能让人人都懂？顶级代码只有少数人才能掌控，狐族妖人，惑乱人心，必须全族诛灭！"

看到这句话，大锤怒了，一拳砸在旁边石壁上，震得石屑往下掉："放屁！纯属放屁！他们就是想垄断，怕别人都看懂了代码，没人给他们交税了！"

我没说话，继续看第三部分。

第三部分画的是最后，狐王带着最后残余的族人，躲进了九尾溶洞最深处，把第三块钥匙碎片藏在九尾寝宫，下了诅咒："只有能读懂源代码的人，才能拿到钥匙，继承狐族遗志，凡是抢钥匙的闭源强盗，代码必被熵魔感染，全身溃烂而死。"最后，狐王亲手关上了入口，把自己锁在了里面。

看完整个壁画，山洞里一片沉默。

原来如此。

狐族不是因为什么邪恶被灭的，他们就是因为坚持开源共享，把代码分享给所有人，动了万维联盟这些垄断者的蛋糕，所以才被联合灭了全族。

这和我们现在开源盟，处境简直一模一样。

我伸出手，轻轻抚摸着壁画上最后那行字，指尖能感觉到当年刻字的狐族工匠，心里怀着多大的愤怒和不甘。

"狐族没能做成的事，我们来做成。"我声音不高，但每个人都听得清清楚楚，"他们坚持的理念，我们会继续坚持。他们没完成的事，我们会完成。"

零音点点头，手按刀柄："没错，今天我们来了，就是继承他们遗志的时候。"

继续往前走，前面一道巨大的石门挡住了去路，石门上刻着一个九宫格，九个格子里都是空白的，旁边刻着规则："把九个数按从小到大排列，门自开。"

大锤挠头："这玩意儿怎么排？谁知道该填什么数啊？"

我笑了，走上前去，这对我来说简直就是送分题。九宫格排序，不就是冒泡排序的入门题吗？

石门上给出了九个乱序的数字，我按照冒泡排序的思路，一次次比较交换，没用一刻钟，就把九个数字按顺序排好了。

最后一个数字放对位置的瞬间，石门轰隆一声巨响，慢慢向两边打开，露出里面更深的通道。

"盟主你太神了！这都能解开！"大锤一脸佩服，跟在我后面走进去。

再往里走，又是一道机关，这次是迷宫，入口石碑上写着："错一步，万劫不复。"

我看了一眼迷宫入口，分叉路口都是二叉分支，这不就是二叉树结构吗？找到正确出口，前序遍历就行了，最简单的方法就是右手定则，一直靠着右边墙走，肯定能走出去。

"跟着我走，别离开我右边。"我带头走进迷宫，靠着右侧墙壁一直走，遇到岔口就右转，走出去再右转，零音他们跟在我后面，谁也没走错。

走了大约半个时辰，眼前豁然开朗，我们走出了迷宫。

面前是一个巨大的前厅，足足有一个足球场那么大，前厅尽头，一道巨大的玉石门紧闭着，门面上，雕刻着一只完整的九尾狐，九条尾巴舒展着，栩栩如生。门额上，刻着四个上古大字：

**九尾寝宫**

我停下脚步，屏住呼吸，能感觉到玉石门后面，传来一股微弱但清晰的能量波动，还有......轻微的呼吸声？

零音也皱起眉，握紧了刀柄，声音压低："门后面有活的生命，能量不弱，应该至少是内核级。"

叶清语轻轻走到我身边，手按在了腰间的银针囊上，眼神警惕但镇定："不管里面是谁，我们都来了，总得进去看看。"

我点点头，握紧了手里的狐尾令牌，明天一早再进去，今天先在前厅休息，养足精神。我们找了个角落，点起篝火，简单吃了点干粮，各自休整。

而在几百里外，开源城方向的森林里，三个穿着灰色道袍的老者，正坐在篝火旁边，为首的长老喝了一口酒，抹了抹嘴，阴冷地笑了："没想到那小子还真把城建起来了，可惜啊，明天一早我们就过去，三个内核宗师杀他一个刚建城的小鬼，万维盟主交代的任务，明天就能完成了。"

另外两个长老都笑了，笑声在黑夜里显得格外刺耳。"听说那小子懂什么源代码，依我看，就是异端邪说，代码哪能人人都看？灭了他，也是替天行道。"

篝火跳动，映着三张阴狠的脸。

千里之外，大战将至。

而九尾寝宫门前，所有线索都已经汇合，只等明天推开这扇门，揭开最后的谜底。

油灯（我的真气光芒）摇曳，映着玉石门上古老的九尾纹路，九条尾巴在光芒下像是活了一样，轻轻摆动。

所有的等待，都快结束了。

所有的线索，都指向这里。

大战，也已经在门外了。

**本章完**
