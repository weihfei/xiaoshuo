# 第3章 重构基础诀，出发青岩城

疤脸死了，零号帮树倒猢狲散，那些跟着疤脸混吃混喝的小喽啰，要么逃去了别的聚落，要么老实了，不敢再作威作福。黑铁聚落笼罩了十几年的阴霾，一下子就散了，连天空看起来都比以前晴朗了许多。

王翔关好破木屋的门，坐在那歪歪扭扭的木桌前，终于能安安静静坐下，好好看看那本原身父亲留下来的《黑铁基础诀》。

这是一本线装的小册子，纸张发黄，有些页边角都磨破了，是原身父亲当年不知道从哪里换来的，传给了原身，也是万家唯一的传承。原身靠着这本基础诀，辛辛苦苦修炼了三年，才刚刚摸到比特战士一段的门槛，然后就出了事，被疤脸打成重伤，没等到王翔来，就撑不住了。

王翔把小册子摊开在桌上，就着从窗户透进来的晨光，一页一页慢慢看，结合原身的记忆，一点点理解这个世界基础功法的运行逻辑。

说实话，看完一遍，王翔只有一个感想：这代码写得真烂，比他刚上大学时候写的课程设计还烂。

《黑铁基础诀》总共也就千余字，讲的就是最基础的吐纳方法，引导比特能量在经脉里运行，吸收能量，提升实力。但是整个能量路径设计得乱七八糟，到处都是冗余节点，到处都是不必要的绕路，时间复杂度高得吓人，而且整个功法是把所有功能写在一起，完全没有模块化拆分，逻辑耦合得一塌糊涂，难怪修炼效率这么低，普通人从一段升到二段都要好几个月，运气不好三五年都卡着。

这要是放在GitHub上发PR，王翔直接就给打回去了，留言："重构了再来，这代码根本没法维护。"

不过现在，这本烂代码功法落在了他这个有六年经验的全栈程序员手里，那自然就是要重构一遍的。反正现在也没什么事，刚杀了疤脸，稳住了阵脚，正好把基础功法重构好，给以后的修炼打下一个好基础。

王翔拿起那个豁口的陶碗，倒了一碗粗茶，喝了一口，茶叶质量很差，苦涩，但能提神。他静下心来，拿出程序员重构老代码的劲头，开始从头梳理。

第一步，模块化拆分。原来的写法是所有代码都揉在一起，吐纳、过滤、排序、存储全混在一起，改一个地方牵一发而动全身，太烂了。王翔直接拆分成四个独立模块：

1. **吐纳模块**：负责从外界吸入比特能量，独立出来，方便以后优化吸入算法。
2. **能量过滤模块**：负责区分纯净能量和杂质，把杂质排出去，这一块独立出来，以后可以换更好的过滤算法。
3. **路径排序模块**：负责给能量节点排序，规划最优运行路径，这是核心性能点，单独拿出来好优化。
4. **能量存储模块**：负责把提纯好的能量存储到丹田，独立出来，方便以后扩容。

拆分完，整个逻辑一下子就清晰了，哪里出问题直接改对应模块，不影响其他地方，舒服多了。

第二步，算法优化。这是提升效率最关键的一步。

原来的路径排序用的是类似冒泡排序的思路，一遍一遍交换，效率极低，时间复杂度O(n²)。王翔直接换成快速排序，时间复杂度降到O(n logn)，性能提升好几倍。

原来查找能量穴位用的是线性查找，从第一个摸到最后一个，太慢了。王翔换成二分查找，时间复杂度从O(n)降到O(logn)，找穴位快了不知道多少。

第三步，解决内存泄漏问题。原来的功法设计有缺陷，吸收进来的杂质用完了就堆在那里，从来不清理，日积月累，就会造成内存泄漏，不仅影响修炼速度，还会留下暗伤，年纪大了就会爆发。王翔直接加上了自动垃圾回收机制，每次吐纳完成，自动扫描，把没用的杂质空间清理出来，放回空闲链表，完美解决内存泄漏问题。

第四步，适配动态调整。原来的功法路径是硬编码写死的，不管你经脉宽窄，天生什么格局，都必须走固定路径，这不科学，每个人经脉都不一样，强行走一样的路径，当然效率低。王翔改成动态适配，根据每个人实际的经脉情况，自动调整出最优路径，适配性大大提高。

这么一顿操作下来，原本千余字乱七八糟的《黑铁基础诀》，被王翔重构得干干净净，条理清晰，模块分明，算法最优，没有内存泄漏，运行效率比原来提升了整整十倍还多。

王翔把重构好的《黑铁基础诀·万相优化版》牢牢记住，放进自己的核心记忆里，然后闭上眼睛，按照新的路径运行了一个小周天，试试效果。

这不试不知道，一试真的吓一跳。

原来按照原版功法，运行一个小周天，需要将近半个时辰，吸收的能量不多，还留下一堆杂质。现在呢？不到五分钟就完成了一个小周天，吸收的纯净能量是原来的三倍，杂质几乎全部过滤掉了，全身说不出的舒畅，效率提升了何止十倍。

"完美。"王翔收功，睁开眼睛，满意地点点头。

这就是重构的力量。同样的功能，不同的算法和架构，效率就是差这么多。原住民几百上千年就是靠着感悟慢慢磨，一点点修改，一辈子也改不了几处，我上来直接用现代软件工程方法论重构一遍，一步到位，直接把性能拉满，这就是方法论的降维打击。

基础打好了，以后修炼速度就是别人的十倍，这一开始就赢在起跑线上了。

就在这时，门外传来了轻轻的敲门声，一个苍老的声音响起：

"万相小哥，你在吗？我是老周啊。"

王翔站起身，走过去开门，门外站着一个白发苍苍的老头，弓着背，脸上都是皱纹，就是黑铁聚落的村长周老头。原身记忆里，周村长算是个好人，当年原身父母被疤脸逼死，他也想帮忙，但是斗不过疤脸的势力，只能偷偷给原身送点吃的，帮着料理后事，这份人情，王翔记下了。

"村长您来了，快进来坐。"王翔侧身让村长进来，给他倒了一碗粗茶。

村长走进来，目光扫了一圈屋里，看到地上已经打扫干净，血迹都处理掉了，点点头，叹了口气："唉，真是想不到啊，疤脸那个畜生作威作福十几年，居然就死在你手里，真是老天有眼，你爹娘在天有灵，也能瞑目了。"

坐下喝了一口茶，村长放下茶碗，说明了来意："我今天过来，主要是两件事。第一件，就是那块西山矿地，本来就是你家祖传的，当年被疤脸强抢了去，现在疤脸死了，村里开了会，一致同意，那块矿地还给你万家，没人有二话，你放心。"

王翔点点头，这个在意料之中。这些年零号帮在村里作恶多端，大家早就受够了，现在疤脸死了，把矿地还给原主，谁也不会说什么，就算有几个以前跟疤脸混的，也不敢放屁。

"谢谢村长，谢谢乡亲们。"王翔道谢，这块矿地虽然贫瘠，每天也就能出几块低纯度比特结晶，但那是祖产，拿回来理所当然。

"第二件事，"村长看着王翔，眼神里带着一点期待，"我听说你现在已经是比特战士二段了？真是年轻有为，你爹要是活着，肯定高兴坏了。正好下个月，二进制宗要去青岩城招收学徒，周边几个聚落的年轻人都可以去报名考核，考上了就能进二进制宗当学徒，那可是一步登天的好事，你年纪轻轻，天赋又好，不去试试？"

王翔心里一动，哦，二进制宗招生？他原身记忆里确实有这么一回事，每年一次，三月份报名，四月份考核，在青岩城，周边上百个聚落的年轻人都会去抢那几十个名额，考上了就是宗门弟子，从此鲤鱼跃龙门，考不上就只能回来继续种地挖矿。

这对他来说，确实是最好的机会。他不可能一辈子待在黑铁聚落这个小地方，想要变强，想要了解这个世界更多秘密，想要去更大的舞台，二进制宗就是最好的起点，必须去。

"报名费需要多少钱？我怕我这点钱不够。"王翔问了个实际问题，他现在兜里也就几十个铜板，得先问清楚。

"报名费不贵，就要五个铜板，主要是路费，从咱们这儿走到青岩城，要三天路程，得准备干粮住宿，怎么也要几十个铜板。"村长笑着说，"你要是想去，钱的事不用担心，村里给你凑，乡亲们都愿意出这点钱，你给咱们黑铁聚落争口气，考上了，咱们脸上都有光。"

王翔心里一暖，在他刚穿越过来，无依无靠的时候，乡亲们能这么帮他，这份情，他记住了。

"那我肯定去，谢谢村长，谢谢乡亲们。"王翔立刻就答应了，没有丝毫犹豫。

"好！好！年轻人就是要有这份志气！"村长高兴得站起来，"那我回去就跟大家说，你安心准备，钱明天就能给你凑齐。"

村长走了之后，果然，不到一个时辰，就有村民陆陆续续过来了，你三个铜板我两个铜板，你一块干粮我一双布鞋，都是一点心意，王翔一一接过来，记在心里，没多少时间，就凑了五十二个铜板，加上王翔自己从疤脸那里搜来的六十八个，一共就是一百二十个铜板，足够报名费和路费了，还有富余。

王翔送走最后一个乡亲，关好门，坐下来，开始整理自己的东西，算一算自己现在的身家：

- **铜板**：一百二十个，足够路费报名费。
- **比特结晶**：十五块用了三块，还剩十二块低纯度，留着路上修炼用。
- **武器**：排序剑（绿名，自己重构，顺手带特效），精铁开山刀（蓝名，从疤脸那里缴获得，备用）。
- **防具**：精铁胸甲（蓝名，疤脸的，穿上加八点防御，比原来的破皮衣强多了）。
- **特殊物品**：未知黑色代码碎片一块，从疤脸怀里掏出来的，看不懂，先收着，以后再说。

东西不多，都塞进一个半旧的麻布袋子里，轻轻松松，不重，走路方便。

接下来就是矿地的安排，王翔自己要去青岩城考试，考上了就要去二进制宗，好几年都回不来，不可能自己打理矿地，所以他找了村长，说这块矿地我暂时委托村里打理，每年挖出来的比特结晶，村里拿七成，我拿三成，等我回来再说，亏了算我的，赚了大家分。

村长一听，这条件太合理了，村里占便宜，当然立刻就答应了，拍着胸脯保证，肯定给你打理得好好的，一分一厘都不会差你的，你放心去考试。

一切都安排得妥妥当当，没有后顾之忧了。

晚上，王翔早早就睡了，这一天从早上杀人到重构功法，忙了一整天，确实累了，需要好好休息，养足精神，明天好上路。

第二天，天还没亮，王翔就醒了，这是他在地球当程序员时候养成的习惯，早起习惯了。他起床收拾好，背着麻布袋子，挎着排序剑，推开木门，外面晨雾还没散，空气清新，带着泥土和野草的香味，沁人心脾。

村长已经带着几个村里的老人在门口等着了，要送他一程，王翔心里有点感动，在这个陌生的世界，还有人来送他，这就是人情冷暖。

"万相，此去青岩城，一路小心，好好考试，争取考上，给咱们黑铁聚落争口气！"村长握着他的手，语重心长地说。

"放心吧村长，我一定尽力。"王翔点点头，"我走了之后，矿地麻烦大家了。"

"不麻烦不麻烦，你放心去。"

又说了几句客气话，王翔拱手告别，转过身，朝着聚落外走去。

走到聚落门口那个土坡上，王翔停下脚步，回头看了一眼。

晨雾渐渐散去，黑铁聚落的土房子错落分布，炊烟已经升起来了，袅袅娜娜，飘在浅蓝色的天空里，安静祥和。这里是他穿越到这个世界的起点，他在这里醒来，在这里报了血海深仇，在这里收获了第一桶金，在这里站稳了脚跟，现在，他要走了。

"再见了，黑铁聚落，"王翔在心里默默说了一句，"等我回来的时候，我肯定会更强。"

他转过身，看着那条延伸向远方的土路，深深吸了一口气，迈开步子，大步走了下去。

从黑铁聚落到青岩城，要三天路程，第一天都是沿着修好的官道走，比较安全，大部分区域都是已经开垦过的，没有什么厉害的野怪，最多就是出来几只兔子野鸡，不伤人。

王翔一路走着，一边走，一边按照优化后的《黑铁基础诀》慢慢吐纳，吸收路上的比特能量，反正走路不耽误吸收，能多攒一点是一点，反正效率是别人的十倍，日积月累，进步飞快。

中午，找了个树荫，坐下休息，吃了两块干粮，喝了几口凉水，休息了半个时辰，继续上路。

傍晚，走到了一个叫落风坡的地方，这里有一个专供行人休息的驿站，不用钱，只要自己找地方住就行，王翔找了个靠墙角的位置，生了一小堆火，烤了一下干粮，吃了晚饭，然后靠着墙打坐，运行了几个周天，吸收了一点能量，就躺下睡觉了。

驿站里还有其他几个去青岩城考试的年轻人，都是周边其他聚落的，大家聊了几句，都是去报名的，算是同路，气氛还算融洽，没人上来找事。

第二天一早，大家一起出发，结伴走，路上也能有个照应，遇到野怪也能一起打。王翔跟着大家一起走，听他们聊天，也了解到了不少关于青岩城和二进制宗招生的信息，知道这次考试大概有多少人，录取多少个，考试内容大概是什么，收获不小。

第二天晚上，到了第二个驿站，一切顺利。

第三天，吃过早饭，大家早早出发，因为下午就能到青岩城了，所以都走得更快了，每个人心里都有点激动，毕竟这是改变命运的机会，考上了就是宗门弟子，一辈子衣食无忧，考不上就只能回来继续挖矿种地，谁不激动呢。

王翔走在队伍中间，不急不慢，保持着均匀的步伐，一边走一边暗中吐纳，吸收能量，他的基础诀优化过，走路的时候都能吸收，比别人效率高了十倍，这种日积月累的进步，最后积累起来就是惊人的差距。

路上，同路的几个年轻人聊得热火朝天，讨论着这次二进制宗招生会有多少人，录取多少个，考试会考什么，谁哪个聚落的天才去年没考上，今年又来了，说得津津有味。

王翔大多时候只是听着，很少说话，他刚来这个世界没多久，多听少说总是没错的，能听到不少有用的信息。

从他们聊天里，王翔知道了，这次二进制宗招生，整个青岩城片区，差不多有上千个年轻人报名，最后只录取五十个，淘汰率二十比一，相当残酷，大部分人都只能空手回去。

考试内容分两关，第一关是测资质，测你能感知多少比特能量，不够标准直接淘汰，第二关是实战交手，分出胜负，排名靠前的录取。

也就是说，不仅要有天赋，还要能打，才能考上。

王翔心里有数了，测资质他不怕，他的基础诀优化过，能量感知比别人强太多，实战他更不怕，他都杀过比特九段的疤脸，还怕跟这些同级别年轻人交手？

走着走着，前面的人突然停下了，有人叫了起来："看！前面就是青岩城了！"

所有人都停下脚步，抬起头，朝着前面看过去。

只见远处地平线上，一座高大的城池出现在视野里，城墙是用青色石头砌成的，所以叫青岩城，城墙高达十几丈，上面有士兵巡逻，城门宽阔，进进出出的行人络绎不绝，远远就能感觉到那股繁华热闹的气息，跟黑铁聚落那种小地方完全不是一个量级。

"哇，这就是青岩城，真高啊！"一个年轻小伙子激动地叫了起来，他还是第一次来青岩城。
"废话，青岩城是这一片最大的城池，管辖着上百个聚落，能不大吗？"一个年纪大点的笑着说道。

王翔也抬起头，看着那高大的青色城墙，心里也有点感慨。

三天前，他还是地球那个996加班猝死的程序员，现在，他已经站在了比特神域青岩城的脚下，马上就要参加宗门学徒考核，人生际遇，真是太奇妙了，谁能想到呢？

他收拾了一下心情，背上包袱，握紧了排序剑的剑柄，跟着人流，朝着城门走去。

越靠近城门，人越多，各种各样的行人，挑着担子的货郎，赶着马车的商人，穿着劲装的修炼者，来来往往，热闹非凡，叫卖声、吆喝声、车马声，混杂在一起，充满了烟火气，这就是活生生的比特神域，比黑铁聚落那个小地方，精彩太多了。

走到城门门口，有士兵收进城税，一个铜板一个人，王翔掏出一个铜板交了，跟着人流走进了城门。

一进城门，一股热气扑面而来，街道宽阔，两边都是店铺，饭馆、客栈、兵器铺、比特结晶商店，什么都有，琳琅满目，看得人眼花缭乱。

王翔深深吸了一口气，空气中都带着热闹繁华的味道，跟黑铁聚落的宁静小村庄完全不同。

他找到了一家便宜的客栈，一天五个铜板，包吃住，进去开了一间房，放下包袱，稍微休息了一下，然后出门，去找报名的地方。

报名地点就在青岩城中心的二进制宗分馆，很好找，随便问个人就知道了。王翔沿着大街走过去，很快就找到了，门口已经排起了长队，都是来报名的年轻人。

王翔排到队尾，慢慢等着，很快就轮到他了。

"名字，哪个聚落的，报名费五个铜板。"负责报名的管事头也不抬地说道。

"万相，黑铁聚落。"王翔拿出五个铜板递过去。

管事登记好了，给了他一块木牌："拿着这个，三天后清晨，到演武场集合参加考核，别迟到，迟到了取消资格。"

"谢谢管事。"王翔接过木牌，收好，退出了报名处。

报好名了，接下来就是三天等待时间，正好可以好好熟悉一下青岩城，顺便调整一下状态，准备考核。

王翔沿着大街慢慢逛，看到了兵器铺，进去看了看，里面各种兵器都有，他那把排序剑是自己重构的，用着顺手，不需要换，开山刀也够用，所以什么都没买。

又走到比特结晶商店，进去看了看价格，低纯度比特结晶一个铜板一块，中纯度的十个铜板一块，价格不算贵，王翔身上还有十二块低纯度，够用了，等考上了再买也不迟。

逛了一圈，回到客栈，吃过晚饭，王翔回到房间，关好门，坐下打坐，运行基础诀，吸收能量，一天赶路积累的疲劳很快就消失了，能量循环几个周天，整个人又精神饱满。

吐纳完，王翔拿出从疤脸那里得到的那块黑色代码碎片，放在手里，仔细研究。

碎片不大，指甲盖大小，黑色，质地温润，上面刻着奇怪的纹路，像是二进制代码，但排列方式又跟正常代码不一样，散发着非常微弱的能量波动，王翔现在实力太低，看不懂这里面到底是什么。

研究了半天，也看不出个所以然，王翔只好收起来，反正反正也不急，以后实力强了，自然就能看懂了，现在急也没用。

一切收拾妥当，王翔躺在床上，看着天花板，心里想着接下来的考核。

第一关测资质，他应该没问题，第二关实战，他更没问题，杀过疤脸，对付这些年轻人，应该没问题。考上了，就能进二进制宗，学到更高级的功法，见识更广阔的世界，考不上，大不了回黑铁聚落自己修炼，慢慢攒钱，下次再考，反正他修炼速度比别人快，早晚能上去。

心态放平，没什么好紧张的。

这一天赶路也累了，王翔很快就睡着了，睡得很香。

三天时间很快就过去了，王翔每天就是吐纳修炼，逛街熟悉环境，心态平稳，实力反而悄悄涨了一点，距离比特三段只差一点了，估计考核完就能突破。

考核这天，终于来了。

一大早，天刚亮，王翔就起床了，收拾好东西，拿上木牌，吃了早饭，朝着演武场走去。

演武场就在二进制宗分馆后面，很大，能容纳上千人，王翔赶到的时候，这里已经挤满了人，上千个年轻人，都在那里等着，议论纷纷，气氛有点紧张。

王翔找了个角落站着，等待叫号，心里平静得很。

很快，考核开始了。

第一关，测资质。

一个个年轻人进去，测试完，要么高兴地出来，要么垂头丧气地出来，淘汰率很高，差不多一半的人第一关就被淘汰了。

很快，叫到了王翔的号："下一个，万相，黑铁聚落！"

王翔答应一声，走了进去。

（第三章完）
