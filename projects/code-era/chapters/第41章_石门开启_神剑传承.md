# 第41章 石门开启 神剑传承

黑风潭底的水压比想象中大得多，但在钥匙发出的柔和光晕笼罩下，水压被自然而然地推开，在我们四个人周围形成了一片干燥的空气屏障，可以正常呼吸说话。

我站在石门前，能感觉到丹田里面那三枚碎片正在发热，温度越来越高，像是要自己飞出来一样。

"三门凹槽，正好对应我们三个人的钥匙。"零音伸出手指，指着石门上那个菱形凹槽分成的三个小孔，左中右排列，大小正好匹配，"我那枚是杀毒权限，应该在右边。"

"我这枚狐族碎片在左边。"九尾晃了晃手里凝聚出来的金色碎片，碎片在空中微微震颤，像是急于投入凹槽，"天生就有种吸引力，应该没错。"

我点点头，看向叶清语："清语，帮我检查一下凹槽周围有没有隐藏的机关或陷阱，我们开门之前小心点。"

"好。"叶清语应了一声，走到我身边，伸出纤细的手指，轻轻按在石门表面，闭上眼睛，运起医疗算力细细扫描。

她的指尖发出淡淡的绿色柔光，顺着石门表面慢慢滑过，所过之处，任何细微的代码异常都逃不过她的眼睛。我站在她身后，能闻到她发梢淡淡的清心草香气，混合着潭水的湿气，闻起来很舒服。

她扫描完了，转过身对我笑了笑，伸手帮我拂了拂被水流浸湿贴在额头上的头发，又轻轻抚平了我衣襟上的褶皱："没问题，凹槽周围没有陷阱代码，就是正常的钥匙验证结构，很干净，没有被熵魔感染。"

手指不经意间碰到了我的胸口，温热的触感隔着浸湿的衣料传过来，我心里微微一动，对她笑了笑："谢了。"

叶清语脸颊微微一红，轻轻点头，退回了我身后站好，不再说话，但我能感觉到她的目光一直落在我背上，带着温柔的担心。

"准备好了吗？"我回头看向九尾和零音，"我们一起，插入钥匙。"

"准备好了。"两人同时点头。

九尾上前一步，掌心托着那枚金色的狐族碎片，走到左边凹槽前，轻轻一推，碎片就像磁石遇铁，"咔"的一声精准落了进去。

嗡——

金色光芒瞬间从凹槽里扩散开来，顺着石门表面的纹路飞快游走，整扇石门都开始微微震动。九尾的狐耳不自觉地露了出来，这是她运功时的习惯，金色的血脉之力顺着钥匙注入石门，石门上的纹路越来越亮。

"该我了。"零音走上前，右手一翻，那枚银色的杀毒权限钥匙出现在掌心，她对准右边凹槽，轻轻一送，钥匙也稳稳落了进去。

又是一阵嗡鸣，银色光芒跟着金色光芒亮起，两种光芒在石门表面的纹路里交错流动，像是两条光蛇，飞快地向中间凹槽汇合。

"最后就看你了，万相。"零音退开一步，对我点点头。

我深吸一口气，凝神内视丹田，引导着那片藏在灵魂深处的根域名碎片，那碎片本来就是透明的，只有在感应到石门凹槽时才会显形。我用意念包裹着它，慢慢引导着它从丹田出来，穿过经脉，飞到掌心，然后轻轻推送进中间的凹槽。

咔哒！

三声轻微的卡锁声几乎同时响起，三枚钥匙正好卡在三个凹槽里，严丝合缝，就像天生就长在这里一样。

三道光芒同时爆发，金色、银色、无色透明，三种光芒在石门中间汇合，然后猛地交融在一起，三枚钥匙缓缓转动，最后合成了一把完整的菱形钥匙，镶嵌在石门正中央，散发出温润的白光。

整个石门都开始震动起来，从底部传来沉闷的隆隆声，像是有古老的机关正在缓缓启动。石门开始向两边分开，一道更亮的光从门缝里透出来，潭水顺着门缝往神殿里面涌，发出哗哗的水声。

石门越开越大，最后完全向两边打开，露出了里面一条向上延伸的石阶。神奇的是，潭水涌进去一段后，里面像是有什么控水机关，水位开始慢慢下降，很快石阶就完全露了出来，空气也流通了，我们能感觉到风从里面吹出来，带着一股古老纸张的干燥气息，完全不是潭底那种潮湿腐败的味道。

"没想到创造者时代的机关还能运行得这么好，都过去亿万年了。"九尾吸了吸鼻子，一脸惊讶，"里面空气很干净，没有熵魔的腐败味。"

零音拔出长剑，走在最前面探路："创造者留下的东西，质量肯定没问题，我们小心点，走，进去。"

我跟在零音后面，九尾在我左边，叶清语在我右边，四个人沿着石阶一步步往上走。石阶很长，大概有一百多级，越往上走空气越干燥，等我们走完最后一级台阶，回头看，入口已经在我们脚下很深的地方了，石门早就自动关上了，但神殿里面灯火通明，墙壁上镶嵌着许多发光的晶块，把整个大厅照得清清楚楚。

我们走出石阶入口，眼前豁然开朗。

这是一个巨大的圆形大厅，直径差不多有一百米，正好一个足球场那么大。地面是用整块的白色大理石铺成，上面刻着整整齐齐的二进制网格，每个格子里都刻着一个0或者一个1，整个地面就是一张巨大的真值表。

正对着我们的墙上，刻着几个很大的古字，笔迹苍劲有力：

**创造者神殿，第一层：编译试炼**

下面还有一行小字，是试炼规则：

> 欲入神殿，先证能力。此殿为能读写源代码者开，不为只会调用API者开。
> 
> 中央石台上有空白石碑，刻一段可运行之排序算法，成功者，守护者自出，过此厅，可入下一层。
> 
> 失败者，留此为神殿能源，永镇山门。

我读完这些字，忍不住笑了。

排序算法？

这考题出得，简直就是往我枪口上撞啊。我当了六年程序员，排序算法写没有一万遍也有八千遍了，从冒泡插到快排归并，闭着眼都能写出来。这创造者怕不是知道我要来，特意给我开绿灯？

九尾凑到我身边，捂着嘴笑："万相大哥，你看，这不是特意给你出的送分题吗？整个比特神域，还有谁比你更会写排序算法？那些原住民连源代码都看不到，让他们写排序算法，不是为难他们吗？"

零音也微微点头，嘴角勾起一丝不易察觉的笑意："确实像是为你量身定做的。我们三个都凑过来了，三把钥匙，又考排序，看来创造者早就等着你来呢。"

叶清语站在我身边，轻轻拉了拉我的袖子："小心点，虽然是你擅长的，但也可能有陷阱，别大意。"

"放心，我心里有数。"我拍了拍她的手，让她安心，"我去去就回，分分钟搞定。"

我向着大厅中央的石台走过去，果然，石台正中央放着一块一人多高的黑色石碑，表面光滑平整，确实是空白的，正好用来写代码。石碑旁边放着一把石凿，应该就是给挑战者刻字用的。

我走上前，伸手握住石凿，能感觉到石凿里面蕴含着淡淡的算力，正好可以用来在石碑上刻字，激发代码运行。

我闭上眼睛，想了想，写什么排序好呢？冒泡排序太简单了，时间复杂度O(n²)，显得我没水平。归并排序空间复杂度O(n)，有点麻烦。希尔排序不稳定，堆排序写起来麻烦……那就写快速排序吧，最常用，平均时间复杂度O(nlogn)，代码也简洁，正好展示一下什么叫现代软件工程。

我握紧石凿，深吸一口气，指尖运起算力，开始在石碑上刻字。

```c
// 快速排序 - 分区函数
int partition(int arr[], int low, int high) {
    int pivot = arr[high];
    int i = (low - 1);
    for (int j = low; j <= high - 1; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(&arr[i], &arr[j]);
        }
    }
    swap(&arr[i + 1], &arr[high]);
    return (i + 1);
}

// 快速排序主函数
void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
```

我刻得很慢，每一行都工整清晰，注释清楚，分区和递归分开，模块化设计，结构一目了然。这是我刚开始学编程的时候就刻在脑子里的代码，闭着眼都不会写错。

刻完最后一个大括号，我放下石凿，后退一步，静静地等着。

不到三秒钟，石碑就亮了。

一道清亮的青光从石碑内部发出来，越来越亮，很快整个石碑都变成了一块发光的青玉，青光顺着地面的二进制网格飞快扩散，整个大厅都开始轻轻震动。

"成了！"九尾在那边兴奋地小声喊。

震动越来越强烈，大厅对面那堵平整的墙壁突然从中间裂开一条缝，然后缓缓向两边分开，露出了里面一个石制棺椁，棺椁慢慢从地下升起来，升到和地面平齐的位置。

咔擦——

棺盖自己裂成两半，向两边倒下去，灰尘扬起，一个身影从石棺里慢慢站了起来。

那不是一个活人，是一个傀儡，由石头和一把剑组成。身体就是石甲，手中握着一把长剑，剑身散发着淡淡的二进制铭文光泽，即使过了几百年，依然锋利不减。

傀儡慢慢抬起头，空洞的石眼望向我们，然后发出了低沉的声音，像是从很远的地方传过来，带着沙沙的电流声：

"通过初试者，报上名来。"

我往前走了两步，握住排序剑的剑柄，淡淡道："二进制宗，万相。"

"万相……"傀儡重复了一遍我的名字，又继续说，"我乃二进制宗宗主凌云天，三百年来，吾留此剑于此，为神殿第一层守护者。能写出如此干净利落的快速排序，说明你确实通晓源代码底层，不是那些只会悟道的凡人，可以进行下一步试炼了。"

凌云天？

我愣住了，没想到这个守护者竟然就是凌云天本人，或者说，是他留在剑里的残魂凝聚成的傀儡。之前我们在外面得到的就是他的内核结晶，没想到他人已经在这里了。

"凌云宗主？"我拱了拱手，"晚辈万相，见过宗主。我们得到了宗主留在外面的内核结晶，知道了神殿位置，特来闯一闯，看看能不能拿到创造者留下的README。"

"好，好，好。"傀儡连说三个好字，"三百年来，你是第一个能写出完整可运行排序算法的人。之前来的那些，要么就是根本不会写，只能瞎画一些符文，要么就是写出来死循环，运行不起来，都给我当燃料了。你不错，比他们强太多了。"

他顿了顿，举起手中的长剑，剑尖指向我："规则还是那个规则，打赢我，我就把我一生的感悟和这里的传承都给你，放你过去下一层。打输了，你就留下来，和我一起守着这扇门，给神殿当燃料。准备好了吗？"

"准备好了。"我抽出排序剑，剑身在晶块灯光下泛着清冷的光，"请宗主出手。"

话音刚落，凌云天傀儡就动了。

没有多余的废话，一剑直刺我胸口，速度快得惊人，剑光一闪就到了我面前，剑风扑面而来，压得我呼吸都一滞。

好快！

不愧是三百年前二进制宗宗主，内核三级巅峰的实力，这一剑比刚才那个熵魔领主快了不止一倍。我赶紧侧身躲开，剑光擦着我肩膀过去，斩在空处，砸在地面上，二进制网格直接被劈碎了一块，石屑飞溅。

"好剑！"我赞了一声，身形不退反进，排序剑挽了个剑花，直接刺向傀儡手腕。

一场大战就此展开。

凌云天的剑法走的是刚猛路子，大开大合，每一剑都势大力沉，正是二进制宗的成名绝学"二分查找剑"。一剑出去，总能精准找到你防御的漏洞，就像二分查找找元素一样，命中率极高。

我一边打，一边打开源代码视野，分析他剑法的源代码。

确实是好剑法，思路完全正确，每次出剑都用二分查找快速定位我的破绽，比那些只会乱劈的剑法不知道高到哪里去。凌云天能成为二进制宗宗主，本事确实不是吹的。

但是……

我看着看着，就发现问题了。

他这个二分查找的实现，有BUG。

边界条件处理错了。

二分查找最容易出问题的就是边界，left <= right还是left < right，这里凌云天写的是left < right，但是在某些情况下会漏掉最后一个元素，导致查找失败，反映在剑法上就是，最后临门一脚的时候总是差那么一点点，威力发挥不出来。

我躲开他又一剑劈砍，跳到一旁拉开距离，忍不住叹了口气。

可惜了，凌云天悟了一辈子，就差这么一点点，边界条件错了，结果就是差之毫厘谬以千里，他到死都没能闯过第二层，只能把残魂留在这里当守护者。

"你叹什么气？"凌云天傀儡停下攻击，沉声问我，"是不是觉得我剑法有问题？"

不愧是前辈，感知还是敏锐。我点点头："回宗主，晚辈确实看出来一点小问题，您这二分查找剑法，边界条件处理错了，应该是left <= right，不是left < right，错了这一个符号，威力至少少了三成。"

大厅里瞬间安静下来。

零音和九尾都愣住了，看着我，又看着傀儡。他们知道我能找BUG，但是没想到我打了十几个回合就直接找出了凌云天悟了三百年都没发现的问题。

傀儡站在那里，一动不动，过了好几秒钟，才发出一声长长的叹息："唉——！"

这一声叹息，充满了遗憾和不甘，在大厅里回荡了好久。

"你说得对……"傀儡的声音低沉下来，"我悟了三百年，就觉得哪里不对，威力总是差那么一点，可我就是找不到问题出在哪里。原来……原来就是一个符号错了！"

他突然抬起剑，指向我："你既然能看出来，那你能改好吗？"

"能。"我点点头，很肯定地说，"就改一个判断条件，很简单。"

"好！那你就用改好的剑法来打赢我！"傀儡再次摆出起手式，身上的气势一下子提了起来，"如果你改得对，打赢了我，我就认你，传承给你！如果改不对，你还是输，就得留下来！"

"没问题。"我收起排序剑，活动了一下手腕，"那宗主，晚辈就得罪了，试试看改得对不对。"

我深吸一口气，把凌云天的二分查找剑法在脑子里过了一遍，把那一个边界条件改过来，然后运起内力，适应了一下改好后的节奏。

不得不说，改了这一个条件之后，整个剑法流畅多了，原来那种总是差一点点的滞涩感一下子就没了，剑气运行通畅，威力直接提升了三成不止。

怪不得凌云天一直卡在这里，就错了一个符号，这就是原住民悟道的悲哀，全靠瞎蒙，找到问题不知道要多少年，换了我，直接debug，分分钟就找到了。

"来吧！"我大喝一声，主动出击。

改好的二分查找刺剑式，一剑直刺傀儡左肩，这一剑速度比原来快了一倍，定位精准，一下子就找到了傀儡防御的一个缝隙。

傀儡大惊，急忙回剑格挡，但是已经晚了，我的剑尖轻轻一点，点在他剑脊上，一股内力顺着剑身传进去，直接震得他握剑的石手裂开一道缝。

"好快！准确！"傀儡忍不住喝彩，"真的对了！难怪你这么年轻就能内核一级，就这份找BUG的本事，我不如你！"

嘴上喝彩，手上却不慢，他挥舞长剑，又是一套组合剑法攻过来。但是他现在用的还是原来那个错的版本，在我这个改对了版本面前，所有破绽都一目了然，我就像拆穿了魔术的观众，他每一步想干什么我都提前知道。

我踩着灵活的步法，在他剑影里穿梭，快速排序连击使出，剑光像流水一样，一刀一刀砍在他石甲上，每一刀都砍在同一个位置，没一会儿，那块石甲就被我砍得粉碎，露出了里面发出微光的剑胚核心。

"承让了，宗主。"我大喝一声，全身算力凝聚在排序剑剑尖，一招改良版二分查找刺剑式，直直刺向傀儡核心。

噗嗤！

剑尖精准刺入核心，没有任何阻碍，正好刺中那团残魂凝聚的地方。

傀儡瞬间僵住，停止了所有动作，站在那里，一动不动。

我抽回剑，后退几步，防备他最后的反扑。

但是没有反扑。

几秒钟后，傀儡脸上的石甲一块块裂开，哗啦啦掉下来，露出了里面那把完整的长剑，剑身上刻着三个古字：二进制。

然后，那团残魂从剑里面飘出来，变成了一个模糊的老者身影，对着我微微一拱手。

"好后生，我等了三百年，终于等到你了。"老者笑了，笑得很欣慰，"那个边界问题，我错了三百年，你一分钟就改对了，看来创造者留下的神殿，确实该由你这样懂源代码的人进去。我这一剑，还有我一生的感悟，都交给你了。"

话音刚落，一枚拳头大小的记忆水晶从虚空中飘出来，慢慢飞到我面前。

"这是我一生探索二进制大道的感悟，还有我走到第三层记下的神殿地图，都在这里面，你拿去吧。"凌云天的身影越来越淡，"我在这里守了三百年，也该解脱了，希望你能走到最后，拿到创造者留下的README，解决那个内存溢出的大问题，别让比特神域毁了。"

说完，身影彻底消散，只剩下那把二进制剑孤零零地插在地上，剑身发出清脆的嗡鸣，像是在认主。

我走上前，捡起记忆水晶，握在手里，又弯腰拔出那把二进制剑。入手沉重，剑身上二进制铭文闪闪发光，寒气逼人，比我的排序剑材质还要好，至少是上品神器级别。

"没想到吧，凌云天宗主竟然直接把传承给你了，我还以为要打个你死我活呢。"九尾走过来，围着我转了一圈，摸着二进制剑啧啧称奇，"好剑啊，这剑比你那把排序剑好多了，以后你就用这个吧。"

"排序剑是我的第一把剑，舍不得换。"我笑了笑，把二进制剑收进储物袋，"这个可以当副剑，或者回头融了，给你们其中一个重新打一把更好的。"

零音走过来，检查了一下记忆水晶，点点头说："很干净，没有熵魔感染，就是正常的记忆传承，可以吸收。"

叶清语站在我身边，伸出手轻轻放在我手上："累了吧，先坐下休息一会儿，吸收记忆水晶不急，恢复好了再说。"

我点点头，确实有点累，跟一个内核三级的傀儡打了这么久，又改了人家的剑法，算力消耗差不多四成，确实需要休息一下。

我们四个人在大厅边上找了块干净的地方坐下休息，我闭上眼睛，把神念沉入记忆水晶，开始吸收凌云天留下的记忆。

比我想象的清晰得多，凌云天的记忆保存得非常完整，从他年轻时候加入二进制宗，一步步成长为宗主，然后听说创造者神殿的传说，带着弟子出来寻找，一路走到黑风峡谷，在这里被熵魔寄生，最后把内核留在外面，残魂进入神殿当守护者，整个过程清清楚楚。

记忆最后，是他绘制的神殿地图，标注得非常详细：

第一层：编译试炼，就是我们现在所在的大厅，考排序算法。

第二层：调试试炼，考debug能力，要找出一段上古代码里的BUG并修好。

第三层：链接试炼，考链接库依赖处理，要把几个零散的代码片段链接成一个完整的程序。

第四层：就是创造者神殿主殿，存放着README文档和创造者留下的核心权限。

一共四层，凌云天当年走到第三层，卡在链接试炼那里，没能过去，所以只能退回来，在这里当守护者，等待后来者。

我吸收完记忆，睁开眼睛，把地图信息分享给她们三个。

"四层，看来比我们想象的简单，每层一个考验，正好对应程序员开发流程，编译、调试、链接、最后拿到可执行程序，创造者还挺会玩的。"我笑着说。

零音点点头："凌云天当年都走到第三层了，我们有四个人，各展所长，肯定能走到最后。我负责杀毒清BUG，九尾负责找漏洞，清语负责整理代码结构，你负责主编写，正好分工合作。"

"说得对。"九尾伸了个懒腰，红色劲装勾勒出美好的曲线，"休息好了吧？我们走吧，早点拿到README，早点知道真相，我都等不及了。对了，万维联盟那些人估计还在外面找我们呢，他们没有钥匙，就算找到石门也进不来，只能在外面干瞪眼，急死他们。"

我笑了笑，站起身，整理了一下衣服，握紧排序剑："说得对，我们走，下一层看看调试试炼考什么，我倒要看看，上古代码能有什么BUG难住我。"

三个人都站起身，跟在我后面，向着通往第二层的石门走过去。我们四个人，三枚钥匙，各展所长，创造者留下的真相，就在前面等着我们。

石门在我们身后缓缓关上，只留下晶块的光芒还在大厅里静静流淌。

**本章完**
