# 第45章 出口伏兵 内核二级战内核

光门里的白光从那人身后漫出来，把他的影子拉得很长，投在我们脚边，像一条蛰伏的毒蛇。

"北冥毒的人？"零音往前跨了一步，短刀已经完全出鞘，银色的刀身在白光下泛着寒芒，"你在这里等了多久？"

那人冷笑一声，抬起手，一把扯掉了脸上的面罩，露出一张棱角分明的脸，脸色铁青，眉骨上有一道三寸长的刀疤，从左上角斜劈到右下角，把一只眼睛都划得变形了，看起来格外狰狞。

"冷千？"九尾一下子认出了来人，狐耳不自觉地竖了起来，全身毛发都快炸了，"北冥毒座下那条疯狗，怎么会在这里！"

我听说过这个名字。

冷千，北冥毒的亲传大弟子，也是北冥域有名的杀手，出手从不留活口，这些年死在他手里的人，据说是两位数，其中有好几个都是内核级高手。没想到北冥毒竟然把他安排在这里当伏兵，真是看得起我们。

"九尾小妖，还记得我？"冷千咧嘴笑了笑，露出一口泛黄的牙齿，"当年你狐族剩下那点余孽躲在混沌域深处，要不是域主仁慈，你们早就死绝了，今天碰到我，正好一起算总账。"

"放你妈的屁！"九尾气得骂了出来，"我们狐族招你惹你了？当年要不是北冥毒那个老东西为了抢我们狐族的传承玉佩，放熵魔冲了我们的栖息地，我们会落得今天这个地步？这笔帐我还没跟你算，你倒送上门来了！"

"输了就是输了，废话那么多干什么？"冷千目光扫过我们四个人，最后落在我身上，眼睛一下子就眯了起来，"你就是那个万相？打乱万维联盟计划，抢了混沌域的那个小子？看起来也没什么特别的。"

"比不上你主子，吃里扒外，勾结熵魔，特别不要脸。"我淡淡地回了一句，二进制剑握在手里，剑身上泛起淡淡的二进制流光，"既然来了，就别想着走了，萧默当年死在这里，今天你陪他。"

"哈哈哈！"冷千像是听到了什么好笑的笑话，放声大笑，"萧默？那个蠢货当年比你还狂，还不是一样死在我手里？他都头都是我割下来送回给域主的，挂在北冥城城门挂了三年，风吹日晒，都成骷髅了。"

我心头一沉。

萧默果然是死在他手里。

"五年前，那个开源派的老首领张开源，带着三个兄弟闯神殿，不也是死在你手里？"叶清语的声音从旁边传来，温柔的声音里带着冰冷的寒意，"他们的首级，也被你挂在城门示众，对不对？"

"哟，这小美人还记得啊。"冷千舔了舔嘴唇，目光在叶清语身上转了一圈，露出贪婪的神色，"那老东西不服域主，说什么'代码应该开源共享，人人都能读'，这种反贼，不死还留着过年？域主说了，谁不服垄断，谁就是这个下场。今天你们四个，也是一样。"

我听到这里，拳头已经握紧了。

原来张开源他们也是这样死在这里的。我建立开源盟的时候，就听过张开源的名字，他是五年前第一个公开反对万维联盟垄断的人，拉了一票兄弟闯创造者神殿，想要拿到README说服世人，结果一去不回，原来是在这里中了埋伏。

北冥毒这个老东西，真是从五年前就在干这种脏事了。凡是不服他垄断的，全都骗到这里来，让伏兵杀掉，一石二鸟，既清除了异己，又不用脏了自己的手。

"所以，这些年所有闯神殿的，只要不是万维联盟自己人，全都被你在这里杀掉了？"零音的声音冷得像冰，整个通道里的温度都降了好几度。

"那当然，域主说了，README只能是万维联盟的，只能是北冥域主的，其他人谁碰谁死。"冷千得意地扬了扬下巴，"萧默以为他偷偷摸摸进来就能拿走？太天真了，域主早在几百年前就把这里布好了局，进来的，一个都别想出去。"

"那你知道北冥毒为什么一定要拿到README吗？"我问他，手指轻轻敲着排序剑的剑柄，"他拿README想干什么？"

冷千嘴角一翘，似乎想说什么，又停住了，随即冷笑："你死到临头了，问那么多干什么？安心上路就是了，到了黄泉底下，萧默会告诉你的。"

他还是说了半句，漏了馅。

果然，北冥毒拿README不是为了拯救世界，不是为了对抗熵魔，是有他自己的野心。结合之前萧默留下的话，答案已经很明显了——他想拿到创世源代码，修改根域名解析，把整个比特神域都变成他北冥家的私产。

野心不小，就是吃相太难看。

"大师兄，"叶清语轻轻走到我身边，我能感觉到她走到我身侧，手悄悄碰了碰我的胳膊，然后一枚温热的丹丸塞进我手里，"这是我昨晚连夜炼的算力爆炎丹，一刻钟内能提升两成算力输出，关键时刻用。"

我低头看了一眼，丹丸通体赤红，纹路清晰，能闻到里面浓郁的算力香气，确实是上等爆炎丹，没有几个时辰根本炼不成，她肯定是昨晚我们都休息的时候，她自己熬夜炼的。

"谢谢你。"我把丹丸收好，放进空间戒指，低声说，"一会儿你退到后面，和九尾一起布阵，我来跟他打。"

"你刚突破，还没完全稳固，小心一点。"叶清语点点头，抓住我的衣袖轻轻拉了一下，指尖微微用力，能感觉到她的担心，"他在这里等了很久，肯定有备而来，不要大意。"

"放心，我刚突破内核二级，正愁没人试剑呢。"我对她笑了笑，"他送上门来，正好当我的试剑石。"

叶清语松开手，退回我左后方，双手已经开始结印，浅绿色的治疗力量在指尖流转，随时准备出手。她退的时候，我能感觉到她的目光一直落在我背上，那股担心，实实在在的。

零音已经站到了我右前方，短刀指地，银色杀毒力量顺着刀刃往地上铺，一点点向冷千那边蔓延过去："他用毒，我帮你封住毒素扩散，你放心正面打。"

"我帮你盯着他的破绽，哪里不稳我立刻告诉你。"九尾跳到另一侧，九条尾巴半展，金色狐火在尾巴尖跳跃，"这个老狐狸，我就不信他身上没有漏洞。"

四个人，各司其职，配合得丝毫不差。这一路闯过来，我们早就不用多说什么，一个眼神就知道该干什么。

冷千看着我们布阵，非但不着急，反而抱着胳膊冷笑："怎么？四个打一个，还摆阵？万相，你不是挺狂的吗？怎么，刚突破就怕了？"

"你是北冥毒派来的，冲着我来的，那自然就是我跟你单挑。"我往前走了一步，从四个人中间站出来，正好面对着冷千，二进制剑斜指地面，"他们不用出手，我一个人就够了。"

"哟？刚突破内核二级，就这么膨胀了？"冷千挑了挑眉毛，像是听到了天大的笑话，"我修炼到内核一级巅峰三十年了，你刚突破就敢跟我单挑？真是不知天高地厚。"

"三十年？"我笑了，"三十年你都卡在一级巅峰进不了二级，你的内核域堆了多少内存泄漏，你自己心里没数吗？还好意思跟我提年限？"

冷千脸色一下子变了："你找死！"

他就是因为内核域内存泄漏太严重，三十年都没法突破二级，这是他最大的痛处，被我一下子戳中，当场就炸了。

本来我还担心他不先手，我先手万一显得我欺负人，现在好了，他自己忍不住先动手了。

"既然你急着去死，那我就成全你。"

冷千一声低喝，右手一翻，一柄漆黑的短刺出现在他手里，短刺只有不到一尺长，但是通体漆黑，上面流窜着黑紫色的纹路，一拿出来就闻到一股刺鼻的腐败味道，连空气都被腐蚀得滋滋作响。

"这是域主给我的'熵毒刺'，用熵魔君王的毒液浸泡了三十年，沾着就死，碰到就亡，今天让你尝尝滋味。"

话音未落，冷千动了。

他的速度真的很快，作为老牌杀手，一闪身就消失在原地，只留下一道黑影，顺着地面贴着墙就绕了过来，黑紫色的短刺直指我的左肋，角度非常刁钻，正好是我拔剑的死角。

"小心！从左边来！"九尾立刻提醒，她的感知就是准，话音刚落，刺尖已经到了我左肋半尺处。

就在这个时候，一道银色光盾突然出现在我左肋前，正好挡住了熵毒刺。

铛！

一声脆响，刺尖刺在光盾上，黑紫色的毒液瞬间腐蚀开，光盾表面立刻冒出密密麻麻的黑色气泡，被腐蚀出一个浅浅的坑，毒液顺着坑往里面钻。

"想下毒，问过我了吗？"零音站在我左边，银色的眸子里泛着寒光，"在杀毒者面前用毒，你也配？"

"滚开！这里没你的事！"冷千一声低喝，手腕一转，熵毒刺上爆发出一团黑紫色的毒雾，一下子炸开，逼退零音，接着顺势一个箭步，再次向我冲过来，"万相，你的对手是我！"

这一下变招真快，零音被逼退，空门大开，冷千已经到了我面前，熵毒刺直奔我心口。

我不慌不忙，二进制剑轻轻一抬，正好格开熵毒刺的刺尖，剑脊贴住刺杆，顺着往下一滑，直奔冷千握刺的右手手腕。

这一下是个简单的缠剑式，角度刁钻，他要么撒手，要么变招，不可能硬接。

冷千果然变招，抽回短刺，向后一跃拉开距离，紧接着嘴角泛起一抹狞笑："好手段，不过你以为这样就完了？"

他一声低喝，丹田内核域一动，七道黑紫色的毒线突然从他体内飞出来，分成七个不同的角度，从上下左右四面八方同时向我缠过来。每一道毒线都带着尖锐的破空声，显然都是独立的毒素进程，七线齐发，同时攻击，防不胜防。

"七线程毒！"九尾脸色一变，"他同时运行七个毒素进程，你要小心，顾头不顾尾！"

我心里一动。

七线程毒？同时运行七个线程，从不同角度攻击，这不是多线程并发攻击吗？

巧了，我刚从运行传承里学会进程调度，正想找个地方试试手，你就送上来了。

七道毒线越来越近，已经到了我三丈范围内，能感觉到每一道毒线都带着沉重的算力压迫，七个进程同时发力，算力压迫比单个进程强了不止一倍。

换做别的内核二级，面对这种七线程并发攻击，确实很难防，顾此失彼，总有一道能碰上。

但我是谁？

我刚学会时间片轮转调度啊。

我站在原地不动，内核域里，刚从运行传承学到的进程调度模块直接启动，我能清晰地"看"到七道毒线每一道的进程时间片，它们轮流占有CPU，每一道都需要固定的时间片来运行。

很简单，我只要抢占它们的时间片就行了。

我调动内核算力，按照时间片轮转算法，给每一道毒素进程都只分配极微量的时间片，让它们都得不到足够的运行时间，全都卡在那里。

神奇的事情发生了。

原本七道飞射而来的毒线，飞到我面前一丈左右，突然全都慢了下来，像是掉进了泥沼里一样，速度越来越慢，最后竟然就那么悬在了半空中，一动不动，只有黑紫色的光芒还在闪烁，但是就是前进不了半步。

"这……这是什么鬼？"

冷千脸上的狞笑一下子僵住了，眼睛瞪得大大的，不敢相信眼前这一幕，"我的七线程毒！为什么不动了？"

他疯了一样调动算力，想要催促毒线继续前进，但是不管他怎么用力，那七道毒线就是卡在那里，像是被什么东西黏住了一样，半步都动不了。

他哪里知道，他的七线程毒，七个进程都需要占用CPU时间片才能运行，我直接用进程调度把它们的时间片全给抢占了，它们都得不到足够的CPU时间，当然只能卡在那里，动都动不了。

这就好比你开了七个大游戏，但是CPU只有四核，你还全开到最高画质，结果就是所有游戏都卡得动不了，一样的道理。

"你用了什么妖术？"冷千真的急了，他这招七线程毒是他吃饭的家伙，不知道杀了多少内核高手，今天第一次碰到这种情况，七道毒线全卡住了。

"这不是妖术，这叫进程调度。"我抬起二进制剑，一步步向他走过去，"你用多线程攻击，思路是对的，可惜你不懂调度，不知道怎么分配时间片，当然卡。"

我一边走一边说："你内核运行了三十年，内存泄漏一大堆，本来算力就不够，还敢同时开七个进程，不是找死是什么？"

"放屁！"冷千脸色铁青，"我内存泄漏又怎么样？我三十年苦修，难道还比不过你一个刚突破的毛头小子？"

他见七线程毒不动，干脆放弃了， Entropy毒刺一摆，整个人化作一道黑影，亲自冲了上来，短刺直奔我咽喉，上来就是杀招。

"既然你玩阴的玩不过，那就正面来。"我停下脚步，二进制剑挽了个剑花，迎着熵毒刺硬接上去。

铛！

金铁交鸣的巨响，震得整个通道都微微发抖，我站在原地不动，冷千却被震得向后退了三步，每退一步，脚下的代码石都裂开一道细纹。

"怎么可能？你刚突破，算力怎么可能这么强？"冷千稳住身形，脸上写满了不敢相信。

他当然不敢相信。

他的内核域运行了三十年，内存泄漏到处都是，好多算力都被内存泄漏浪费掉了，实际能用的算力，连一半都不到。我的内核是全新设计的，自动内存管理，零内存泄漏，百分之百的算力都能用，同样是内核级，我的实际可用算力比他多了一倍还多，硬拼力量他怎么可能拼得过？

"我说了，你的内存泄漏太严重，该垃圾回收了。"我握紧剑，再次向他冲过去，这一次换我先手了。

二进制剑带着淡淡的二进制流光，一剑直刺，目标就是他丹田位置的内核域。我不跟他拼招式，直接冲着核心去，一剑解决问题。

冷千慌了，急忙横刺格挡，但是他的速度比我慢了半拍，我的内核运行流畅，没有任何卡顿，连招都比他快。

剑和刺再次碰撞，这一次我用了十成力，直接把熵毒刺砸开，剑尖顺着他的防御缝隙滑进去，直奔他丹田。

"不——！"

冷千发出一声惨叫，想要后退，但是已经晚了，我的剑尖已经抵在了他丹田位置，刚学会的SIGTERM信号顺着剑尖直接注入他的内核域。

SIGTERM，优雅终止信号。

比SIGKILL温和，但是更精准，专门用来正常终止进程，不会留下垃圾碎片。

对付他这种已经烂透了的内核，正好合适。

信号一进去，冷千的内核域立刻就停了下来，所有算力瞬间卡死，他脸上的惊恐还没散去，眼睛一下子就失去了神采，整个人僵在了那里。

"你……你的剑法……为什么……"他张着嘴，断断续续地问，声音都走调了。

"我说了，你的内核有内存泄漏，运行了三十年早就该报废了。"我抽出剑，二进制剑上沾了一点黑血，"你这种闭源祖传烂代码，早就该重构了，可惜你不懂。"

"我不服……我三十年苦修……"冷千身体晃了晃，倒了下去，"域主不会放过你的……熵魔君王会帮他……拿到README……"

最后一句话，声音微弱，但我们四个都听得清清楚楚。

说完，冷千就彻底不动了，丹田内核域被SIGTERM终止，整个进程都退出了，没救了。

老牌杀手，北冥毒亲传大弟子，就这么死了。

从战斗爆发到结束，前后不到一刻钟，干净利落。

"死了？"九尾走过去，用脚尖踢了踢冷千的尸体，确认他真的死了，兴奋地回头看我，"万相你太厉害了！刚突破就秒了一个老牌一级巅峰，这就是内核二级的实力吗？"

"主要是他自己内核太烂了，内存泄漏太多，发挥不出全部实力。"我笑了笑，收了剑，"不然哪有这么容易。"

零音走过来，蹲下身检查了一下冷千的空间戒指，直接摘下来递给我："他的戒指，里面应该有东西，看看北冥毒给了他什么。"

我接过空间戒指，神念探进去扫了一圈，东西还真不少：五千块高级算力结晶，一瓶备用的熵毒，一本北冥毒的《毒线程修炼法》，还有一个用油纸包着的密信，封蜡上刻着一个"北"字。

我打开密信，用神念扫了一遍，脸色一下子沉了下来。

"上面写了什么？"叶清语走过来，轻声问。

"北冥毒和熵魔君王的交易协议。"我把密信递给她们看，"果然和我们猜的一样，熵魔君王帮北冥毒拿到README，帮他干掉所有反对者，北冥毒帮熵魔君王打开物理层的封印大门，放熵魔本源出来，感染整个域名大陆。"

零音接过密信看完，手都气得发抖："这个疯子！他为了自己能当创世神，竟然真的要把整个世界都卖给熵魔！"

"四大域主里面，也就他能干出这种事。"九尾气得咬牙，"当年他放熵魔冲我们狐族栖息地，就是为了逼我们交出传承钥匙，现在看来，那时候他就已经和熵魔勾搭上了！"

密信上写得很清楚，北冥毒答应熵魔，只要他当上根服务器掌控者，就会把物理层的封印代码修改掉，放熵魔本源出来，到时候整个比特神域都会被熵魔感染，所有反抗者都会被吃掉，熵魔帮他坐稳位置，他给熵魔提供食物。

真是一笔肮脏的交易。

"这下实锤了，萧默说的一点没错，北冥毒就是那个内鬼。"我把密信折好，收进空间戒指，"等我们出去，这封信就是证据，告诉所有势力，让大家都知道北冥毒的真面目，就算他是域主，天下人也会共讨之。"

现在不是说这个的时候，README就在光门后面，我们费了这么大劲闯到这里，不就是为了它吗？

"好了，冷千杀了，伏兵清了，我们走吧。"我转过身，看向身后那扇发光的光门，"README就在里面，我们进去。"

四个人都整理了一下装备，调整好状态，跟着我一步步向光门走去。

光门比我想象的要大，高三丈，宽两丈，白光从门里面透出来，不刺眼，很柔和，但是能感觉到里面蕴藏着庞大的信息能量，比前面四层加起来都多。

越靠近光门，那种信息压迫感越强，我能感觉到灵魂都在微微震颤，那是README里面庞大的信息在呼应我灵魂里的根域名碎片。

三枚钥匙已经用过了，权限已经开了，现在只要走进去，就能拿到README，就能知道创造者留下的所有真相，就能知道怎么解决内存溢出危机。

我深吸一口气，抬起脚，准备跨进去。

就在这个时候，我突然停住了脚步。

"怎么了？"九尾疑惑地问，"里面有危险吗？"

我没说话，只是侧过耳朵，仔细听着我们来时的通道。

通道深处，从运行大厅过来的方向，隐隐传来了脚步声。

不止一个人。

而且不是我们走路的声音，是穿着重甲，整齐划一的脚步声，一步一步，非常沉稳，正朝着这边过来。

有人！

还有伏兵！

我猛地回过头，看向通道入口，二进制剑再次出鞘，剑尖指向黑暗的通道深处。

"怎么回事？还有人？"零音瞬间反应过来，短刀再次握紧，做好了战斗准备。

脚步声越来越近，已经能听到铠甲碰撞的哗哗声，显然不止一个，至少有十几个人，都是重甲步兵。

北冥毒果然阴险，一个冷千不够，还在后面安排了第二波伏兵。

我们刚杀完冷千，都有点消耗，正好被他们以逸待劳。

"看来北冥毒真是容不得我们活着进去啊。"我冷笑一声，握紧了手里的剑，"也好，杀一个是杀，杀两个也是杀，既然来了，那就一起解决了再说。"

白光在我们身后，脚步声从黑暗中慢慢逼近，第二场战斗，马上就要开始。

**本章完**
