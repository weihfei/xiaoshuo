# 第46章 重甲死士 进程互斥锁

脚步声越来越近，铠甲碰撞的哗哗声也越来越清晰，听得人心里发紧。

"多少人？"我握紧二进制剑，问九尾。

九尾闭着眼睛感知了两秒，睁开眼的时候，眸子里已经泛起了红光："十六个，全都穿着黑钢重甲，最前面那个……内核二级中期！"

我心里一沉。

北冥毒这一手玩得真阴。

冷千在前面光门口堵着我们，后面还有一波伏兵，等我们和冷千拼得两败俱伤，他们再从后面冲上来收人头，这就是标准的螳螂捕蝉黄雀在后。

前后夹击，我们现在背靠光门，退路被堵，真的是进退两难。

"清语，能不能布阵堵一下通道？通道窄，他们展不开。"我转头看向叶清语。

"没问题。"叶清语点点头，指尖飘出十几道浅绿色的光点，瞬间落在通道两侧的石壁上，"我布了个减速困阵，他们进来速度会慢一半，算力流转也会受到影响，能争取一点时间。"

浅绿色的微光闪过，整个通道入口都笼罩在一层淡淡的雾气里，这就是叶清语的困阵，她不仅会治病炼丹，布阵也是一把好手。

"我这边状态没问题。"零音把短刀握在手里，银色的杀毒能量在刀身上流转，"刚才打冷千我没出多少力，前排我来扛。"

九尾活动了一下手腕，狐火在指尖跳动："我消耗有点大，但打起来没问题，真不行我还能自爆狐火跟他们同归于尽。"

"别乱说什么同归于尽。"我拍了拍她肩膀，"我们都能活着出去，拿了README再说。"

其实我心里算得很清楚：对方十六个重甲死士，还有一个二级中期的首领，我们四个刚打完冷千，确实都有不同程度的消耗，真打起来，我们胜算其实不到五成。

但那又怎么样？

从一路打到这里，我们什么时候有过十足的胜算？还不是一步步闯过来了。

北冥毒想在这里把我们全留下，那也要看他有没有这个牙口。

脚步声停了。

十六个身穿黑钢重甲的死士从黑暗的雾气里走了出来，果然如九尾所说，个个都有字节级巅峰实力，穿着厚重的黑钢重甲，手里提着一人高的阔刃重斧，斧刃上泛着幽幽的蓝光，显然喂过剧毒。

十六个人分成两排，正好把整个通道堵得严严实实，连一只老鼠都钻不过去。

一个满脸刀疤的壮汉从人群后面走出来，身高快两米，肩膀比城门还宽，手里提着一把门板一样的大关刀，刀身上刻满了扭曲的毒纹。

他摘了脸上的面罩，露出那张横肉纵横的脸，左眼位置一道刀疤从额头一直划到下巴，把整个脸都劈成了两半，看着就吓人。

"万相，叶清语，九尾，零音。"屠千刀咧开嘴笑了笑，露出一口黄牙，"久仰大名啊，能打死冷千，你们确实有点本事。可惜啊，冷千就是心急，想抢头功，结果把自己脑袋抢没了。"

我握紧剑，盯着他："你就是第二波伏兵？北冥毒倒挺看得起我们，居然派了你这么个大家伙。"

"我？"屠千刀哈哈大笑，声音震得通道顶上的石头都往下掉渣，"北冥域主座下，重甲统领屠千刀，奉命在这里给你们收尸。告诉你们，这叫一箭双雕，冷千消耗你们体力，我上来摘果子，从三年前开始，能打过冷千的，还没人能走出我这一关。"

九尾听到"三年前"三个字，身体一下子绷紧了，狐火瞬间就烧起来了："三年前，我叔叔闯神殿，是不是就是你杀的？"

屠千刀斜着眼睛看了九尾一眼，忽然又笑了："哦？原来你就是那个老狐狸的侄女？不错不错，父债子还，叔债侄还，今天正好一起算，省得我以后再去找你。"

"你！"九尾气得浑身发抖，就要冲上去。

我一把拉住她："先别急，报仇不急这一时，先看他想说什么。"

屠千刀掸了掸刀身上的灰，继续说："北冥域主说了，只要你们把三枚钥匙交出来，再把README拓一份给他，今天我就放你们一条生路，还可以给你们北冥家一个客卿位置，比跟着这个反贼瞎闹强多了。"

"放屁！"九尾忍不住骂了出来，"我们会投靠你这种走狗？"

"不识抬举。"屠千刀脸色冷了下来，"那就别怪我不客气了，三十多个内核级高手都死在这里，你们四个，也不会例外。"

我开启源代码视角，仔细看了一眼这十六个死士，心里顿时明白了八九分。

每个死士的丹田内核域里，都有一把奇怪的锁，十六把锁连在一起，共享同一个互斥变量，同一时间只有一把锁能被拿走，也就是说，同一时间只有一个死士能全力运行算力。

这就是……互斥锁死士阵？

有意思，北冥毒居然能把程序员的并发概念用到武学上，还真有创意。

十六个人共享同一个算力池，用互斥锁保证不会同时抢资源导致算力冲突，每次一个人上去打，打完释放锁，下一个人接着上，这样一来，虽然每次只有一个人出手，但算力是十六个人加在一起的，一斧下去，比普通二级巅峰算力还猛。

这种阵法，确实很邪门，一般人根本破不了。

可惜，他遇到了我。

互斥锁这东西，我写了十年代码，天天跟它打交道，它有什么毛病我能不清楚？

"他们这个阵有点邪门。"零音低声对我说，"每次只有一个人上来，但算力特别猛，我刚才试了一下，挡一斧手臂都麻。"

"我知道。"我点点头，"他们用了互斥锁，十六个人共享算力，轮流上来打，我们一直防守，消耗太快，用不了半个时辰，我们就没劲了。"

叶清语走到我身边，用只有我能听到的声音说："他们挤在一起，阵型密集，我可以用银针干扰他们的算力连接，你找机会杀了屠千刀，只要首领死了，阵应该就能破。"

她说话的时候，气息吹在我耳边，有点痒，我能闻到她身上淡淡的药草香，很清爽。

"不用那么麻烦。"我笑了笑，"这个阵我能破，他们锁设计得有问题，一戳就破。"

话音刚落，屠千刀已经不耐烦了："废话少说，给我杀！一个都别放过！"

第一个重甲死士一声不吭，提着阔刃重斧就冲了上来，重甲踩在地上，整个通道都跟着震动。

果然，只有他一个人全身发光，算力澎湃，其他十五个人都站在原地不动，身上算力暗淡，都在等锁。

一斧劈下来，劲风扑面而来，确实猛，斧还没到，压力已经让我呼吸都有点困难。

零音上前一步，银色光盾瞬间撑开，挡在前面。

嘭！

一声巨响，震得我耳朵嗡嗡响，零音被这一斧震得后退了三步，靠在石壁上才稳住，手臂微微发抖："好大力气，算力真的叠加在一起了！"

第一个死士劈完一斧，退了回去，身上算力立刻暗了下去，第二个死士紧接着上前，又是一斧劈过来，频率刚好卡着冷却时间，一波接一波，根本不给你喘息的机会。

这就是车轮战啊。

十六个人轮流上来，我们一直扛，就算扛得住，体力也耗光了。

零音又硬挡了一斧，又退了两步，脸色有点发白："这样不行，消耗太快了，再挡几斧我就要顶不住了。"

"让我来。"我推开零音，走到前面，"我来破阵。"

二进制剑出鞘，淡淡的二进制流光在剑身上流淌，我盯着冲上来的第三个死士，心里默默数着时间。

一斧劈来，我侧身躲开，剑鞘顺势砸在他斧脊上，把这一斧卸到一边，眼睛死死盯着他内核域里那把锁。

就是现在！

他劈完这一斧，算力回收，锁释放的瞬间——

我一剑刺出，二进制剑精准地点在他丹田位置，一丝代码顺着剑尖直接注入他的内核域。

一段非常简单的代码：

```c
lock(A);
lock(B);
// 等待锁释放，永远等不到
while(1);
```

死锁。

我直接给他的互斥锁加了个循环等待，现在他占着锁A，等着拿锁B，下一个死士占着锁B，等着拿锁A，谁都不让，谁都动不了。

这就是最经典的死锁四个必要条件，我一次性给他凑齐了。

那个死士劈完一剑，正准备退回去，突然就僵在了原地，全身算力瞬间卡住，动都动不了，眼睛瞪得大大的，满脸不敢相信。

"怎么回事？"屠千刀也发现不对劲了，"第一个怎么不动了？"

第二个死士等了半天，没等到锁释放，也不对啊，按照规矩，第一个劈完应该自动释放锁啊。

他不知道，锁已经死了，永远不会释放了。

第二个死士忍不住自己抢锁，结果一抢，直接也陷入了等待，也僵住了。

然后第三个，第四个……

就像多米诺骨牌一样，十六个死士一个接一个全都僵住了，全都站在原地，一动不动，每个人都占着锁，每个人都在等别人放锁，谁都动不了。

整个通道里，瞬间就安静了，只剩下十六个硬邦邦的重甲死士站在那里，像十六尊石像。

屠千刀眼睛都直了："这……这怎么可能？你对他们做了什么？"

我笑了笑，提着剑一步步走过去，从第一个死士身边走过："没什么，就是给你们的锁加了点料。你们北冥家不是喜欢用互斥锁吗？那你应该听说过一句话，**不可重入锁，必死锁**。"

走到第一个死士面前，我举起剑，轻轻一剑刺在他丹田，SIGTERM信号注入，优雅终止。

他身体晃了晃，倒了下去。

我接着走，第二个，第三个……

十六个死士，我一个个走过去，一剑一个，全都干净利落地终结了进程。

全程没有人能还手，全都僵在那里，等着我杀。

屠千刀看着这一切，手都开始发抖了："你……你怎么会破我的互斥锁死士阵？这是我北冥家传绝学，从上古传下来的，从来没有人能破！"

"上古传下来的就了不起啊？"我停下脚步，转过头看他，"你们这锁设计得就不对，没有超时机制，没有死锁检测，连优先级继承都没有，不死锁才怪。回去找你们家域主，让他重新学一遍操作系统再来吧。"

说话间，十六个死士已经全都倒了，整个通道里，就剩下屠千刀一个人站在那里。

九尾眼睛亮了："万相，你太厉害了！这样就破了？"

"本来就不难。"我笑了笑，"不懂并发编程还敢玩锁，不是找死吗？"

屠千刀脸色一阵青一阵白，突然怒吼一声，整个人都爆发出耀眼的算力光芒："好！好得很！毁我死士阵，我要你碎尸万段！"

他双手握刀，整个人的二级中期算力完全爆发，整个通道都被他的算力填满，扑面而来的压力比刚才十六个死士加起来还大。

"万相！让我来！"九尾就要冲上来，"他杀了我叔叔，这个仇我要自己报！"

"你先别急，他交给我，你最后补刀就行。"我拦住九尾，"他现在怒了，出手没章法，我来解决他。"

屠千刀一声暴喝，整个人冲了上来，大关刀劈头盖脸就是一刀，刀风呼啸，整个通道都仿佛要被劈开了。

"多进程劈刺刀！"

一刀接着一刀，一刀比一刀快，每一刀都是一个独立进程，三十六刀劈出来，封死了我所有闪避角度，果然有点东西。

换了别人，面对这么密集的多进程刀法，根本躲不开。

可惜，他遇到了我。

多进程？我正好会进程调度啊。

我脚步移动，二进制剑轻轻颤动，每一刀都被我精准格开，我用优先级调度，直接抢占最高优先级，他每一刀的运行轨迹我都能提前预判，因为他进程调度算法没我好啊。

你用的是先来先服务，我用的是最短作业优先，你怎么可能赢我？

三十六刀劈完，屠千刀一刀都没碰到我衣角，他自己反而喘开了，一脸不敢相信："你……你怎么能躲开我所有刀？"

"因为你调度算法不行啊。"我淡淡说，"先来先服务都什么年代了，还在用，不卡才怪。"

就在这个时候，叶清语突然轻喝一声："就是现在！"

三道银光从她指尖射出，快如闪电，瞬间就打中了屠千刀三个不同位置，正好是三个主要的算力经络。

"啊！"屠千刀身体猛地一僵，一刀劈偏了，歪到了一边，三个经络瞬间麻痹，算力流转一下子就断了。

机会！

我怎么可能放过这么好的机会？

二进制剑不退反进，一步跨到屠千刀面前，剑尖直指他丹田内核，这一次，我没用SIGTERM，用的是SIGKILL。

强制终止，不接受任何反驳，不给任何处理时间。

一剑刺穿！

"不——！"

屠千刀发出一声惨叫，眼睛瞪得大大的，不敢相信："你……你怎么可能……破我……多进程刀法……"

"回去问冷千吧，他在下面等你呢。"我抽出剑，二进制剑上沾了一抹血。

屠千刀身体晃了晃，直挺挺倒了下去。

九尾立刻冲上来，一道狐火直接打在屠千刀身上，火焰瞬间就把他整个人吞噬了："叔叔，侄女今天给你报仇了！你可以安息了！"

狐火熊熊燃烧，很快就把屠千刀烧成了灰烬，只留下一个空间戒指掉在地上。

九尾捡起戒指，递给我，眼睛有点红："谢谢你，万相。"

"跟我还客气什么。"我笑了笑，接过戒指，神念探进去扫了一圈，东西还真不少。

两万块高级算力结晶，一瓶北冥家的毒功心法，还有一张用油纸包着的密令，上面盖着北冥毒的大印。

我打开密令一看，上面字不多，但内容让人心里一紧：

"**万相若死，取首级和钥匙回域主府领赏；万相生，则火速传信，本域主亲自来取。**"

看来北冥毒马上就要到了。

我们必须在他来之前拿到README，不然真被他堵在这里，麻烦就大了。

"大家都没事吧？清语，帮我们看看伤。"我转过头，对叶清语说。

"嗯。"叶清语点点头，拿出一个药瓶，倒出四粒恢复丹，分给我们每个人，"都吃下去，运气调息，我给你们运功顺一下经络。"

她的真气带着淡淡的药香，顺着后背输进来，温暖舒服，刚才打斗造成的经络酸痛，很快就缓解了。

我吃了恢复丹，调息了片刻，算力恢复得差不多了，其实我消耗真不大，大部分时间都是在观察，最后破阵和杀屠千刀也就是几剑的事，消耗真不大。

零音和九尾也都调息完毕，状态恢复得七七八八。

"都好了？"我站起身，看向身后的光门。

白光从光门里透出来，柔和温暖，能感觉到里面庞大的信息波动，那是README在呼应我灵魂里的根域名碎片。

从穿越到这个世界，一路走到这里，我们用了四十六章，闯过了无数关卡，死了不知道多少敌人，玄机子宗主也牺牲了，现在，终于到地方了。

创造者留下的README，就在里面。

这个世界的真相，内存溢出危机的解决方法，全都在里面。

我深吸一口气，握紧了手里的二进制剑。

"走吧，我们进去。"

我率先抬起脚，跨进了光门，白光瞬间把我笼罩，温暖的信息流动漫过全身，说不出的舒服。

叶清语、九尾、零音依次跟了进来。

光门后面，是一个巨大的圆形大厅，天花板上镶嵌着无数发光的晶石，把整个大厅照得如同白昼。

大厅正中央，悬浮着一个巨大的水晶台，水晶台上刻满了密密麻麻的源代码，流淌着淡淡的金色光芒。

水晶台前面，一个古朴的青石台上，放着一卷用晶玉雕成的书卷。

书卷展开一半，封面上面，刻着三个古朴的大字，即使过去了亿万年，依然清晰可见：

**README**

我们四个人都停下了脚步，静静地看着那卷晶玉书卷，没有人说话。

千山万水，我们终于走到这里了。

下一页，就是真相。
