# 第51章 原野列阵 碎片共鸣

我把碎片收进空间戒指，拍了拍排序剑的剑鞘，抬眼看向零音和九尾，嘴角勾起一抹笑："崔雄来得倒是快，正好，省得我们打完仗还要跑去找他。"

零音微微点头，手已经按在了腰间短刀上，银色的短发下，眼神冷得像冰："五千人，大多是北冥毒的旧部，崔雄本人是字节级巅峰，比王虎难对付。"

"难对付又怎么样？"九尾蹦到我身边，粉色的耳朵轻轻晃了晃，小鼻子皱了皱，"我已经感知过了，他们队伍里代码波动很乱，北冥毒刚死，这群人本来就军心不稳，崔雄压得住一时压不住一世。"

我笑了笑，转身往密室出口走，靴子踩在石阶上，发出清脆的回响。

"说的没错，北冥毒都死在我手里了，他一个副将，真当自己能翻天？今天就让万维联盟的人看看，敢踏入我开源盟的地盘，是什么下场。"

走出密室，重新移回假山石，外面天色已经渐渐暗了下来，夕阳把城主府的屋檐染成了金红色。大锤已经得到消息，正站在院子里等我，铜钟一样的身子站在那里，像一座黑铁塔，手里攥着他那柄开山大锤，锤头都快比他身子宽了。

"盟主，近卫团已经在城门口集合完毕，随时可以出发！"大锤瓮声瓮气地报告，嗓子压得很低，听得出来他有点兴奋，自从跟着我出来，这还是第一次遇到正经万维联盟的精锐。

我点点头，抬手拍了拍他宽厚的肩膀："别急，我们不用急着冲，就在城北五里坡列阵，以逸待劳，让他们来。"

"好嘞！"大锤应了一声，转身大步往外走，靴子踩在青石板上，咚咚作响。

零音跟在我身后，轻声开口："要不要我带几个杀毒者先过去，趁他们立脚未稳，先摸掉他们的前锋？"

我停下脚步，回头看了她一眼，她一身银甲在夕阳下泛着冷光，眼神锐利如刀。不得不说，零音永远都是这样，永远想着先下手为强，不给对手任何机会。

"不用，"我摇摇头，"五千人而已，光明正大打垮他们，比偷偷摸摸杀掉崔雄更有用。我们要让整个混沌域都看看，万维联盟的精锐，在我们开源盟面前，就是不堪一击。这样后面的城池才会更心甘情愿地投降。"

九尾从后面追上来，拽住我的袖子，晃了晃："那我呢那我呢？我能干什么？我能闻出来谁身上有熵魔的味道！"

我笑着揉了揉她粉色的头发，手感软乎乎的："你跟着我就行，帮我盯着点，万一崔雄带了什么熵魔感染的死士，你提前告诉我。"

"没问题！"九尾立刻挺起胸脯，骄傲地晃了晃狐狸尾巴，"包在我身上！"

我们出了城主府，翻身上马，一千近卫团已经列队完毕，人人都是重甲长刀，精气神饱满，队列整齐得像一块刀切出来的豆腐。这些人都是跟着我从混沌域一路打出来的老弟兄，个个都练了万相框架，实力比普通万维联盟的士兵强出不止一个档次。

我勒住缰绳，胯下的黑马打了个响鼻，原地刨了刨蹄子。我提了提排序剑，目光扫过面前一千张坚毅的脸，声音不大，但用内核级域名之力送出去，每个人都听得清清楚楚。

"兄弟们，万维联盟的人来了。"

"他们来干什么？他们不让我们好好过日子，不让我们免费学功法，不让我们少交税，他们就是来替欧阳震抢地盘，继续把我们当算力奶牛！"

"今天，我们就在这里告诉他们，混沌域已经不是他们万维联盟的天下了！谁要来碰我们的地盘，谁就要付出代价！"

"杀不杀？！"

最后一个字出口，一千近卫团齐声怒吼，声音像打雷一样，震得路边的树叶哗哗往下掉："杀！杀！杀！"

声浪直冲云霄，连地面都好像微微颤抖。我点点头，拨转马头，马鞭一指北方："出发！"

一千人跟着我，秩序井然地开出城北城门，往五里坡进发。我让大锤把队伍展开成标准的攻击阵型，前锋尖刀，中军主力，后卫接应，安排得妥妥当当。

刚在五里坡列好阵，就看到北方烟尘滚滚，马蹄声震天动地，崔雄的五千人果然已经到了。

对方也很快发现了我们，前锋停止前进，后军慢慢展开，也列成了冲锋阵型。远远就能看到，对方中军大旗上绣着一个大大的"崔"字，黑底金边，被风刮得哗啦啦响。

我催动战马，往前走了几步，离对方大阵大约一百丈站定，域名之力铺开，整个战场都在我的感知范围内。崔雄果然在中军大旗下面，穿着一身黑鳞甲，手里握着一杆三尖两刃刀，骑在一匹高头大马上，脸色阴沉得能滴出水来。

我笑了笑，催动域名之力，声音浩浩荡荡传过去，整个战场都能听得见："对面的崔雄听着！我是开源盟盟主万相！"

"你家主子北冥毒，早在十天前就已经被我斩杀在混沌城下，你现在还带着这么多人来给欧阳震卖命，真以为自己能活得回去？"

"我万相在这里把话撂这，只要你们放下武器投降，我开源盟一视同仁，功法开源，算力共享，原来是什么样，来了还是什么样，我绝不刁难！要是执迷不悟，今天这五里坡，就是你们的埋骨之地！"

这番话喊出去，对方大阵立刻就骚动起来，能听到一阵阵嗡嗡的议论声，不少士兵都停下了动作，往中军方向看，眼神里满是怀疑。

毕竟北冥毒才是他们的老主子，现在老主子死了，他们千里迢迢过来增援，本来就心里没底，被我这么一喊，军心瞬间就散了一半。

崔雄脸色大变，猛地一拍战马，往前冲了几十步，站在阵前，指着我怒吼："万相逆贼！你休要妖言惑众！我家主人神功盖世，怎么可能会死在你手里！你不过是打了点小胜仗，就敢在这里口出狂言！"

"今天崔某奉盟主之命，前来取你狗命，正好为北冥域主报仇！谁要是敢动摇军心，军法处置！"

他抽出三尖两刃刀，对着天空举了举，身后的亲兵也齐声呐喊，好不容易才把骚动压下去。但我能看得出来，大部分士兵脸上还是带着疑云，呐喊声有气无力，根本没什么斗志。

我笑了，笑得很大声："崔雄，你到现在还在骗你的手下。北冥毒的域主核心现在还在我丹田温养，你要不要过来看看？"

"你自己想想，欧阳震要是真有心救青石城，为什么只给你五千人？他就是把你当棋子，扔过来消耗我，就算你打赢了，他没什么损失，打输了，死的也是你，他一点不心疼。你跟着这么个主子，有什么好卖命的？"

这话就像一把刀子，直接扎进崔雄心里，也扎进所有士兵心里。崔雄脸色瞬间变得铁青，他当然知道欧阳震就是这个意思，不然怎么会只派五千人来对付我八万开源盟？但他没办法，军令如山，不来就是死，来了还有一线生机。

"少废话！"崔雄一声怒吼，"万相，你敢不敢跟我单挑！赢了我，这五千人我立刻就走，输了你就束手就擒！"

零音在我身边皱了皱眉，低声说："激将法，他想拖时间，等后续援军。"

我点点头，我当然知道。崔雄现在就是孤注一掷，想跟我单挑，赢了就能翻盘，输了大不了一死，反正对他来说已经没别的出路了。

"单挑？"我哈哈一笑，提着排序剑从阵前走出来，"你一个字节级巅峰，也配跟我单挑？不过我成全你，今天就让你死得明白，让你看看，内核级和字节级，到底差在哪里！"

我一提缰绳，战马往前一蹿，不到十个呼吸，就站在了两军阵前，和崔雄隔着二十丈对视。排序剑在我手里，轻轻挽了个剑花，阳光照在剑身上，反射出冷冽的寒光。

崔雄眼睛一亮，他没想到我真敢答应单挑，催马往前走，三尖两刃刀指着我，吼声如雷："好胆！万相，拿命来！"

他猛地一夹马腹，战马嘶吼着向我冲过来，三尖两刃刀带着呼呼风声，劈头盖脸就砍了下来。刀还没到，凌厉的剑气就已经刮得我脸颊生疼，不愧是北冥毒手下第一猛将，确实有两下子。

但也就那样了。

我站在马背上，看着他冲过来，眼睛里泛起淡淡的二进制流光，崔雄的三尖两刃刀和他体内功法运行的源代码，清清楚楚展现在我面前。

哟，还真是祖传闭源刀法，老代码了，到处都是内存泄漏，运算路径也乱七八糟，排序都排不明白，也好意思拿出来现？

我轻轻侧身，躲开他这一刀，排序剑轻飘飘递出去，正好点在他刀杆上一个最薄弱的节点。那是他刀法源代码里的一个缓存溢出BUG，平时看不出来，一受力就容易出问题。

"铛！"

一声巨响，火花四溅。崔雄只觉得手上一麻，一股精纯的算力顺着刀杆直接冲进来，瞬间冲乱了他体内的功法运行，手臂抖得几乎握不住刀。他脸色一变，心里震惊得无以复加：怎么可能？我这一刀明明势大力沉，他怎么能这么轻松就化开？

他不知道，我只是点了他一个BUG而已。

崔雄咬咬牙，强行稳住手臂，拧腰转胯，三尖两刃刀变劈为刺，直指我心口，招招都是杀招，凶狠无比。他知道今天不能拖，拖得越久对他越不利，必须速战速决。

我不慌不忙，骑着马围着他转圈，排序剑忽快忽慢，每一次出剑都正好点在他功法运行的节点上。他每出一刀，都要被我点歪一点，内力运行都要紊乱一下，打了不到十几个回合，他额头上已经见汗了，呼吸也变得粗重起来。

越是打，崔雄心里越是慌。他发现，不管他出什么招，我都能提前预判，总能恰到好处地找到他的弱点，轻轻一下就让他这一招威力大减。就好像……就好像我把他的刀法看得透透的，比他自己还清楚。

这怎么可能？他这手三尖两刃刀，是北冥毒亲传的闭源刀法，除了他和北冥毒，没人看过源代码！

"你……你怎么会知道我的刀路？"崔雄忍不住低吼，额头上的汗水滴进眼睛里，涩得难受。

我笑了笑，语气轻松："你这刀法代码写得太烂了，BUG比功能还多，一眼就能看出来，还用特意知道？"

"胡说八道！"崔雄怒喝一声，猛地往后一跳，跳出圈外，深吸一口气，全身字节级算力全开，域名之力凝聚在三尖两刃刀上，刀身瞬间泛起黑色的光芒，"看来不使出真本事，你不知道马王爷有几只眼！"

"北冥七杀刀！第一杀——域封！"

崔雄一声怒吼，一刀劈出来，黑色的刀气化作一张巨网，铺天盖地向我压下来，域名之力直接锁死了我周围所有空间，想把我困在里面。这是他压箱底的绝学，用域名封锁对手，然后一步步压缩，最后一刀斩杀。

啧啧，还是用域名当网抓人的思路，闭源功法果然都是这么简单粗暴。

我轻轻摇头，催发自己的内核级域名，瞬间覆盖了整个战场。崔雄那点字节级域名，在我内核级域名面前，就像一滴水掉进了海里，瞬间就被稀释得无影无踪。那张黑色的刀网刚飞到我面前，就被我的域名之力冲击得支离破碎，化作漫天飞散的代码碎片。

"什么？！"崔雄眼睛都瞪圆了，一脸不敢置信，"你已经突破内核级了？！"

他怎么能不惊讶？我斩杀北冥毒的时候才刚到字节级巅峰，这才过去多久？居然就突破内核级了？这怎么可能符合修炼规律？

可惜，他不知道，主角我用的是开源框架，修炼速度就是这么不讲道理。

"现在才知道？太晚了。"

我催动排序剑，体内算力顺着剑刃流淌出去，排序算法全力运转，一道道剑气按照从小到大的顺序排列，形成一道锋利无匹的剑虹，直指崔雄心口。排序剑本来就是我用排序算法重构的，用起来得心应手，威力比普通兵器强出数倍。

崔雄吓得魂飞魄散，急忙横刀招架，"铛"的一声巨响，三尖两刃刀直接被我砍断，排序剑去势不止，径直刺穿了他的胸膛。

我手腕一转，内核级算力顺着剑尖冲进去，直接打乱了他丹田内的核心代码。崔雄眼睛猛地突出，一口鲜血喷出来，洒在我面前的草地上，染红了一大片。

"你……你……"他张了张嘴，想说什么，但是已经说不出来了，身体一歪，从马背上摔了下去，当场没了气息。

我抽回排序剑，剑尖滴血，在战马旁边轻轻蹭了蹭，把血擦干净，抬眼看向对面万维联盟的大阵，声音平静但清晰："崔雄已死，还有谁要来？"

对面大阵死一般的寂静，所有人都惊呆了，刚才还凶神恶煞的五千精锐，瞬间就没了主心骨，一个个站在那里，你看我我看你，都不知道该怎么办了。

过了大概十几秒钟，不知道是谁第一个扔下了武器，"哐当"一声，紧接着就是接二连三的哐当声，大片大片的士兵放下了武器，纷纷跪了下来："我等愿意投降！饶命啊！"

不到半分钟，五千人里面，超过四千人都放下了武器投降，只有崔雄的几百亲兵还站着，但是也没人敢往前冲了。大锤看到这个情况，哈哈一笑，提着大锤就带着前锋冲了过去："兄弟们，投降不杀！反抗的就别怪爷爷锤子不客气了！"

那几百亲兵哪里还敢抵抗，一下子就溃散了，大部分也跟着放下武器投降，少数负隅顽抗的，瞬间就被近卫团砍翻在地，一点浪花都没掀起来。

前后不到半个时辰，五千精锐就彻底解决了，我们这边几乎没什么伤亡，只有十几个轻伤的，还是对方冲锋的时候不小心擦到的。

九尾蹦蹦跳跳跑到我身边，眼睛亮晶晶的："哇，万相你好厉害！三两下就砍了崔雄！比我想象的快多了！"

零音也走过来，脸上冰冷的表情融化了一点点，点了点头："内核级实力果然不一样，压迫感比之前强太多了。"

我笑了笑，收了排序剑："走，回城，我们回去看看那块碎片。现在三块碎片凑齐了，看看能出什么东西。"

我们押着俘虏，打扫完战场，回到青石城的时候，天已经完全黑了。我让人安排好投降的士兵，承诺按照开源盟的规矩待遇，然后就带着九尾和零音，再次来到城主府后花园的假山这里。

移开假山石，推开暗门，我们三个重新走下密室。胡桃木桌子还摆在那里，我从空间戒指里掏出刚找到的那块乳白色碎片，放在桌子上。

碎片还是温温的，散发着淡淡的古老气息。九尾也从脖子上取下一个吊坠，吊坠打开，里面躺着一块完整的乳白色碎片，和我这块大小差不多，气息一模一样。

我又凝神内视，丹田深处，那块根域名碎片也感应到了外面的气息，微微发热，一股温和的信息流顺着经脉流出来，一直流到我的手上。

"现在三块齐了。"我看着桌子上两块，又摸了摸丹田，"我丹田深处一块，这里两块，正好三块，创造者说的三枚代码钥匙，原来就是这么分的。"

九尾凑过来，小鼻子对着两块碎片使劲嗅了嗅，金色的眼睛里满是好奇："对哦，就是这个味道！三块放在一起，气息越来越浓了！你看，是不是发光了？"

我顺着她指的方向看过去，果然，两块碎片原本只是淡淡的微光，现在越来越亮，乳白色的光芒慢慢汇聚在一起，形成了一个小小的光团。我丹田内的根域名碎片也感受到了召唤，一块小小的虚影从丹田飘出来，悬浮在密室空中，慢慢落在光团里面。

三块碎片虚影重合在一起的瞬间，整个密室突然剧烈地震动起来，地面微微摇晃，头顶上的灰尘簌簌往下掉。零音立刻抽刀挡在我身前，眼神警惕地盯着四周："怎么回事？要塌了吗？"

"不是，"我摇摇头，盯着光团，"应该是碎片共鸣，打开通道了。"

震动持续了十几秒钟，然后慢慢停了下来。原本密室后面那堵光秃秃的石墙，突然发出"轰隆隆"的声响，石墙慢慢向两边分开，露出一扇黑沉沉的石拱门，门后面不知道有多深，一股更加古老、更加悠远的气息从里面飘出来，比碎片的气息还要浓郁得多。

石拱门完全打开后，里面隐约能看到淡淡的光芒，一条石阶弯弯曲曲往下延伸，不知道通向哪里。

我们三个都站在那里，看着那扇打开的石门，谁都没说话。过了好一会儿，九尾才小声开口，声音里带着点激动："原来……原来真的有更深的遗迹！我们狐族传说，集齐三块钥匙，就能打开上古创造者遗迹，拿到上古编译器，原来传说都是真的！"

我看着那扇黑沉沉的石门，心里也有点激荡。穿越到这个世界这么久，从一个最低级的比特战士，一步步走到现在，终于摸到了创造者留下的遗迹，拿到上古编译器就在眼前了。有了上古编译器，我就能编译更复杂的代码，就能更快推广开源框架，就能更快推翻万维联盟的垄断。

零音收了短刀，走到我身边，眼神冷静："里面不知道有什么危险，我先进去探路？"

我拉住她，摇摇头："不急，今天已经打了一仗，大家都累了。而且崔雄刚死，城里还要安排俘虏和城防，明天一早我们再进去探索也不迟。"

我顿了顿，看向石门后面，嘴角勾起一抹笑："反正遗迹在这里，跑不了。养足精神，明天我们好好进去看看，创造者到底给我们留下了什么好东西。"

三块代码钥匙已经集齐，大门已经打开，上古遗迹就在眼前。这一趟青石城，来得太值了。

夜色透过假山的缝隙照进来，映在石拱门上，那片古老的黑暗仿佛在静静等待着我们。明天，就是揭开谜底的时候。
