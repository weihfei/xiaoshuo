# 第70章 青云定鼎，剑指Git域

东方天际，第一缕晨曦如同锋利的剑气，撕开了沉沉夜幕，洒落在青云谷这片刚刚经历过血战的土地上。

硝烟还未完全散尽，空气中依旧弥漫着血腥与灵力爆炸后的焦糊气味。昨夜的喊杀声仿佛还在山谷间回荡，此刻却只剩下风吹过残剑断刃的呜咽声。青云谷原本郁郁葱葱的林木，此刻像是被巨大的梳子狠狠梳过一遍，东倒西歪，焦黑的树桩随处可见，地面上布满了大大小小的坑洼，那是高阶法术对轰留下的痕迹。

万相站在青云谷最高的一处山岩上，一袭青袍早已被鲜血染得斑驳，昨夜连续不断的指挥与出手，让他眉宇间也带着几分疲惫，但那双深邃的眼睛，却依旧明亮如晨星，静静地俯瞰着整个山谷。

"首领，统计结果出来了。"

一个清脆冷静的声音在身后响起，零音手持一块晶莹剔透的记忆水晶，一步步走上山岩。她一身银色软甲，多处破损，手臂上还缠着绷带，但背脊挺得笔直，精致的面庞上没有太多表情，只有一种近乎刻板的认真。

万相转过身，点了点头："说吧。"

零音目光扫过水晶表面，指尖灵力微动，一串串清晰的数字浮现在空中："启禀首领，此战我军投入总兵力十五万人，战后统计，战死五千一百二十七人，重伤一万六千四百三十一人，轻伤九千八百七十人，可继续作战者十万余人。阵亡将领包括：左翼偏将黑石，中军都尉铁云，斥候队长风影……"

她一口气念完二十多个阵亡将领的名字，每个名字都准确无误，连他们所属的部队、阵亡的位置都记得分毫不差。在她精准平稳的声音里，昨夜那场惨烈的厮杀仿佛重新浮现——黑石在谷口拦截敌方突袭时被三位元婴围杀，铁云为了保护补给线被自爆的敌将同归于尽，风影带着斥候深入敌后侦查情报，最后只剩下一匹马逃了回来……

念完阵亡名单，零音稍稍停顿，继续说道："敌方投入兵力十三万人，西门封亲自带队五万伏兵，战后统计，被斩杀两万七千余人，俘虏五万两千一百三十七人，其中有降卒意愿者两千八百余人，剩余三千余人不愿投降，已经按军法处置。敌方将领方面，域主西门封被首领当场斩杀，副域主重伤被俘，其他偏将、校尉被斩杀十六人，俘虏九人。"

她顿了顿，补充道："缴获各类功法玉简一万两千余枚，灵石近三百万，兵器甲胄四千余件，另外在西门封的密室中发现了未公开的闭源功法三十七种，还有一份万维帝国皇室和西门封之间的密约。"

整个统计过程，零音没有一丝慌乱，也没有多余的感慨，每一个数字都精准得像是刻在水晶上一样。认识她的人都知道，这位年轻的姑娘就是这样，越是大战之后，越是冷静得可怕，任何杂乱无章的局面到她手里，都能梳理得清清楚楚。

万相静静地听着，听完之后沉默片刻，缓缓说道："伤亡比差不多一比二点五，这个结果，比我预想的要好。"

"是啊……比预想的要好，可……可是那么多兄弟……"

一个粗重的声音响起，巨石迈着沉重的步伐走上山来，他那比常人宽了一倍的肩膀此刻耷拉着，巨大的手掌紧紧攥着，指节发白，原本粗犷豪放的脸上，此刻布满了愧疚与悲痛，一双铜铃大的眼睛里，布满了红血丝。

他昨夜带着前锋营冲在最前面，硬生生撕开了敌方三道防线，全营一千两百人，打完仗剩下不到四百。看着身边跟着自己出来的兄弟一个个倒下去，这个五大三粗的汉子，心里比刀割还要难受。

"首领，我对不起那些死去的兄弟。"巨石瓮声瓮气地说，"如果我不是急于推进，中了西门封的埋伏，咱们也不会死这么多人。都怪我性子太急，太大意了……"

说到最后，他声音都有些沙哑，猛地一拳砸在身边的岩石上，"轰"的一声，坚硬的花岗岩直接被他砸出一个大坑，碎石飞溅。

万相看着他，摇了摇头："战场上，哪有百分之百完美的计策，西门封本来就想用埋伏引我们进去，你就算不上，他照样会设埋伏，统计伤亡就是这样，战死的兄弟，我们会记住他们，他们的家人，我们会好好安置，不会让他们白白牺牲，你不用太自责。"

零音也在一旁说道："巨石将军，根据战场记录，你中埋伏之后沉着稳住阵脚，拼死挡住了敌人两轮反扑，坚持到了主帅主力赶到，你做得没错，要是换了别人，早就崩溃了，你已经做得非常好了。"

巨石抬起头，看了看零音，又看了看万相，粗大的喉咙动了动，最后还是重重叹了口气，一拳砸在自己胸口："俺就是心里难受，跟着俺出来的兄弟，一半都留在这里了，俺……"

"等战后，我们在这里给战死的兄弟们立一块碑，把所有名字都刻上去。"万相望着山谷下方，轻声说道，"每一场胜利，都是用这些兄弟的鲜血堆出来的，我们不会忘记他们，你走吧，先下去休整，接下来还有硬仗要打。"

巨石这才点点头，双手抱拳道："末将领命，"说完，转身大步走了下去，脚步比之前沉稳了很多。

万相望着他的背影，轻轻叹了口气，对零音说道："统计完成，我们下一步，大军什么时候开拔？"

零音低头看了看手里的情报，说道："巨石将军已经整训完成，前锋五万五千人早就准备好了，随时可以开拔，就等主帅一声令下。"

"好，"万相点点头，站直了身体，拔出腰间佩剑，剑指西方，声音传遍了整个营地："既然准备好了，那就拔营出发，目标青云关，前进！"
