# 第8章 宗门大比，一鸣惊人

时间过得真快，转眼三个月过去了，二进制宗一年一度的年度宗门大比，就要开始了。

这是二进制宗传承了几百年的老规矩，每年一次，所有弟子不管新人老人，都可以参加，赢了就能得到丰厚奖励，名次靠前的还能晋升身份，得到更多资源，所以所有年轻弟子都摩拳擦掌，跃跃欲试，都想拿到好名次，得到更多资源，更快进步。

王翔进来也快半年了，从一个刚进门的新人，变成了宗门里人人皆知的天才，修复基础功法，修复顶级心法，补全了残缺几百年的上古传承，早就吸引了所有人的目光，这次大比，很多人都等着看他表现。

报名的时候，执事问他报新秀组还是进阶组，王翔想了想，他才进来半年，还是新人，报新秀组最合适，跟同批次的比，没什么问题。

报完名，王翔回到院子，不少同批次的新人过来聊天，有人就说了："万相，这次新秀组冠军肯定是你的，你都五段了，我们最高才七段，而且你那么会改功法，肯定没人打得过你。"

王翔只是笑了笑，没说什么，冠军不是靠说的，打过才知道，赛前低调点总是没错的。

比赛这天很快就到了，青云山脚下的演武场，人山人海，坐满了来看比赛的弟子，长老宗主都坐在主席台上，气氛热烈得不行，所有人都很兴奋。

新秀组比赛，就是今年新收的五十个新人，抽签对阵，单败淘汰，最后赢到最后的就是冠军。

王翔抽签，第一轮对手是一个三段的新人，来自青岩城本地，上来的时候挺紧张，手心都出汗了，毕竟对手是大名鼎鼎的万相，谁不紧张？

那人一上来，咬咬牙，提着刀就冲上来了，上来就是一招开山劈，拼尽全力。

王翔站在那里，没动，对方一动手，他就看到了对方功法的问题，哪里是弱点，哪里运行不畅，一目了然，根本不用多想，他脚步轻轻错开，躲开刀锋，排序剑轻轻一刺，正好刺中对方握刀的手腕，对方吃痛，刀直接掉在了地上，不用打，直接认输了。

从开始到结束，不到十秒钟。

台下一片惊呼，所有人都没想到，这么快就解决了，太快了！

第二轮，对手是四段，比王翔低一级，上来的时候信心满满，觉得自己等级高，能赢王翔，毕竟王翔才五段，他四段，差一级，说不定能爆冷。

结果呢，还是一样，对方一动手，王翔一眼就看到了对方功法里面的内存泄漏点，一剑过去，直接刺中，对方能量瞬间乱了，没办法再战，直接认输，前后不到一分钟，解决战斗。

就这样，王翔一路赢下去，每一场都是速胜，不管对手等级比他高还是比他低，都是很快解决，从来没有超过一炷香，看得台下观众越来越惊讶，掌声一阵接着一阵，主席台上的长老们也频频点头，玄机子宗主脸上的笑容就没下去过，显然对王翔非常满意。

很快，王翔就打进了新秀组四强，半决赛，对手是赵轩，一个长老的孙子，进来之前就是六段，天赋很好，背景也好，一直就不服王翔，觉得王翔就是会改功法，实战不一定厉害，这次半决赛碰到王翔，正好能打败他，证明自己才是这一届第一天才。

上台之后，赵轩提着剑，站在对面，盯着王翔，语气带着点嚣张：

"万相，我知道你改功法很厉害，但是比赛比的是实战，不是改功法，今天我就要打败你，让大家知道，谁才是这一届真正的第一！"

王翔看着他，挺平静："是不是第一，打过就知道了，准备好了就开始吧。"

裁判一声令下，比赛开始。

赵轩二话不说，一提气，提着剑就冲了上来，一招"雷霆霹雳斩"，力道十足，刀风呼呼，看得出来，确实有两下子，六段就是六段，比之前那些对手强多了。

但那又怎么样？在王翔的源代码视觉面前，赵轩的每一步，每一招，所有的发力路径，所有的弱点，都明明白白摆在那里，他修炼的那本《雷霆刀法》，本来就有内存泄漏，左肩那里每次发力都会有一个微小的卡顿，这个弱点赵轩自己都不知道，王翔一眼就看到了。

王翔脚步轻轻一晃，躲开了赵轩这致命一刀，然后顺势就冲了上去，排序剑轻轻一刺，正好刺中那个卡顿的弱点左肩。

"呃！"

赵轩一声闷哼，左肩瞬间就麻了，一半力气都使不出来，剑都快拿不住了，王翔顺势一脚，踹在他膝盖上，赵轩重心不稳，直接就摔下了台。

整个过程，不到半分钟。

演武场安静了一秒钟，然后爆发出雷鸣般的掌声，所有人都站起来鼓掌，太精彩了！这一剑太准了，正好刺中弱点，直接解决了六段的赵轩，这实力，真的太可怕了。

赵轩爬起来，脸色涨得通红，一句话都说不出来，低着头就下去了，他不服也不行，事实摆在那里，他确实打不过，王翔一眼就找到他的弱点，这本事，没人不服。

就这样，王翔打进了新秀组决赛，对手是周虎，来自一个下属城池的大家族，进来之前就是七段，已经在外门摸爬滚打了两年，这次好不容易考进来，实力非常强劲，夺冠呼声仅次于王翔，是这次决赛最大的热门。

决赛这天，演武场挤得水泄不通，所有人都来看这场巅峰对决，新秀组最强的两个人，到底谁能拿到冠军？

上台之后，周虎对着王翔抱了抱拳，挺客气："早就听说万相兄弟大名，改功法天下第一，今天终于有机会领教了，希望兄弟不吝赐教。"

王翔也抱拳回礼："周兄过奖了，互相学习，开始吧。"

裁判一声令下，决赛开始。

周虎不敢大意，他知道王翔的厉害，所以一上来并没有猛攻，而是稳扎稳打，刀法严密，一刀一刀，慢慢压缩王翔的空间，他等级比王翔高两级，七段对五段，力量占优，他想慢慢消耗，最后找机会一击致命。

但他不知道，他修炼的那本《狂虎刀法》，存在一个很明显的问题，就是连续挥刀三次之后，右腰发力位置因为内存泄漏，能量供应不上，会有一个大约半秒的卡顿，这个问题，周虎自己都不知道，他只是觉得每次三连击之后有点累，从来没想到这是功法本身的BUG，但是王翔一眼就看出来了。

两个人你来我往，打了几十个回合，周虎慢慢觉得不对劲，王翔步伐太灵活了，每次都能提前躲开他的刀，他一刀都没碰到王翔，反而自己力气消耗得很快，比他预想的快多了。

又一次，周虎发动三连击，一刀比一刀重，第三刀劈出来，正好到了那个卡顿点，半秒的卡顿出现了。

就是现在！

王翔眼睛一亮，抓住这个千载难逢的机会，身形一闪，直接冲到近处，排序剑直直刺出，正好刺中那个卡顿的位置，也就是右腰发力点。

一剑刺中，周虎瞬间就感觉右腰一麻，那股麻痹感顺着脊椎直接上去，整个右半边身体都软了，大刀哐当一声掉在了地上，力气一下子就泄了。

王翔上前一步，剑锋一转，就抵住了周虎的脖子，停住了。

"你输了。"王翔平静地说。

周虎愣在那里，半天都没反应过来，过了好一会儿，才叹了口气，低下头："我输了，冠军是你的，我服气。"

说完，捡起刀，走下了台。

演武场彻底爆炸了，掌声欢呼声像是打雷一样，震得人耳朵嗡嗡响，所有人都站起来，拼命鼓掌，王翔太强了！一路连胜，决赛击败七段周虎，拿到了新秀冠军，太牛了！

主席台上，玄机子宗主哈哈大笑，对着身边的长老们说："你们看看，我就说这小子没问题吧，刚来半年就拿到新秀冠军，咱们二进制宗多少年没出这么天才的新人了？"

长老们纷纷点头，赞不绝口，这么年轻，就这么厉害，将来肯定是咱们二进制宗的顶梁柱，没问题。

颁奖的时候，玄机子宗主亲自下来，给王翔颁奖，奖励一百块中纯度比特结晶，一件全新的蓝色内甲，比王翔现在穿的那件好太多了，然后当场宣布，直接晋升王翔为内门弟子，享受内门待遇，每月月俸翻倍，还能分到更好的洞府。

王翔接过奖励，行礼道谢，退了下来。

就这样，年度宗门大比落下帷幕，王翔拿到了新秀组冠军，一鸣惊人，整个二进制宗，谁都知道了这个新人天才，不只是会改功法，实战也一样天下无敌，才进来半年，就击败了所有同批新人，拿到冠军，这份实力，真的没人不服。

晚上，王翔回到自己新分到的内门洞府，比原来那个更大更安静，灵气更浓，他坐在院子里，看着奖励的一百块中纯度比特结晶，还有那件崭新的蓝色内甲，笑了笑，这次大比，收获真不小，资源更多了，修炼速度也能更快了。

现在他已经是内门弟子了，身份不一样了，能接触到更多宗门机密，更多上古传承，能更快提升自己，离自己的目标也更近了一步。

二进制宗年度大比，王翔一鸣惊人，成为了整个宗门年轻一代最耀眼的新星，所有人都觉得，用不了多久，这个年轻人就能成长为宗门的顶尖高手，未来不可限量。

王翔自己心里清楚，这才只是开始，他的路还长着呢，这个世界还有很多秘密等着他去发现，很多问题等着他去解决，他只能一步步，稳稳地往前走。

（第八章完）
