# 第35章 混沌初定 第一城

踏出白桦林的那一刻，我就感觉到了不同。

空气里弥漫着一股淡淡的焦糊味，像是服务器过载烧了主板的气味，周围的空气都在微微颤抖，就像是屏幕信号不稳，时不时闪一下，连影子都跟着歪歪扭扭，一会长一会短，看着很不舒服。脚下的土地，有些地方已经变成了灰黑色，摸上去冰凉刺骨，那是熵魔长期侵蚀留下的痕迹。

"大家小心点，这里数据流不稳定，容易被熵魔偷袭。"零音走在最前面，右手一直按在刀柄上，银色的真气隐隐流转，她的感知比我们所有人都敏锐，"我能感觉到，周围有不少低阶熵魔在盯着我们。"

叶清语轻轻往我身边靠了靠，微凉的手指插进我的指缝，和我紧紧握在一起。她抬头看了我一眼，眼神里有担忧，但更多的是坚定："这里的能量波动好乱，我带了足够的银针和修复符文，真有人被熵魔感染了，我能救。"

我拍拍她的手背，笑着说："放心吧，我们有产权玉牌，域名权限在我手里，熵魔不敢太放肆。"

我举起左手，那枚青色的混沌域产权玉牌在夜色中微微发光，一丝丝暖流从玉牌里涌出来，顺着我的手臂流进丹田，然后我把丹田真气注入玉牌，激活了域名权限。

一瞬间，玉牌爆发出一层淡淡的青色光罩，从我们脚下向四面八方扩散开去。光罩所过之处，原本颤抖扭曲的空气瞬间稳定下来，那些隐约可见的黑色熵魔残影，一碰到光罩就像是雪遇到开水，瞬间蒸发了。

"有用！"赵石眼睛一下子亮了，他伸手摸了摸身边的空气，"刚才还感觉经脉有点发痒，现在舒服多了！"

"产权玉牌就是有这个好处，"我收起玉牌，光罩收敛了一些，只维持着一个基本防护圈，"拿到了域名产权，整个混沌域的根目录就是我们说了算，熵魔属于未授权进程，自然进不来核心防护区。走，我们往中心去，按照地图，上古域名核心在中心盆地，那里地基完好，适合建城。"

一行人排成纵队，跟着我往混沌域中心走。大锤扛着他那柄大锤走在队尾，负责殿后，他每走一步，大锤都在地上顿一下，咚的一声，那些躲在草丛里的小熵魔被震得现行，然后被光罩蒸发，省事得很。偶尔有胆子大的还想跟着，他就发出一声大吼，吼声震得树叶都哗哗掉，那些低阶熵魔吓得跑得比兔子还快。

走了不到十里路，前面一片乱石林里突然传出哗啦一声响，三只成年熵魔冲了出来。这些家伙长得就像是一团黑色的粘液，没有固定形状，流动着朝我们扑过来，所过之处，石头都被它们腐蚀得滋滋冒烟。

"小心！它们能吞噬算力！"零音话音未落，已经拔刀冲了上去，刀光一闪，一道银色刀气劈出去，直接把最前面那只熵魔劈成了两半。

但是预想中的爆炸并没有发生，被劈开的两半熵魔，每一半都很快收缩成型，变成了两只更小的熵魔，继续扑过来。

"不好，它们会分裂！"大锤冲上去，一锤砸在一只熵魔身上，砸得它四分五裂，结果碎片又分裂成更多小熵魔，满地都是，爬得飞快。

转眼间，三只熵魔分裂成了十几只，越来越多，要是再这么打下去，用不了多久我们就要被淹没了。众人都紧张起来，纷纷拔刀戒备。

我站在原地没动，眼睛快速扫过那些熵魔，在我眼里，它们每个都带着相同的特征码，就像是正则表达式能匹配的字符串。我笑了笑，气运丹田，对着那些熵魔轻喝一声："正则匹配，批量清除。"

我抬手一挥，一道青色的数据流从丹田发出，瞬间扫过整个战场。就像是电脑执行了一遍正则替换，所有带着熵魔特征码的进程，一瞬间全部被终止。

十几只熵魔，不管大小，全部定格在原地，然后瞬间化作飞灰，干干净净，一只都没剩下。

整个过程不到三秒钟。

零音收刀站着，愣住了。大锤举着大锤，也愣住了。赵石等人瞪大了眼睛，嘴巴张得能塞进一个拳头。

"这...这就完了？"大锤挠了挠头，"我锤子还没热呢。"

"熵魔分裂繁殖，本质上就是同一个特征码不断复制，用正则一次性全部匹配出来，直接清除，比一个个砍省事多了。"我收了真气，笑着说道。

叶清语走上前，拿出绢布，轻轻帮我擦了擦额头上的汗水，她眼睛亮晶晶的，笑着说："你刚才用正则清除的时候真好看，整个过程行云流水，一点都不拖泥带水。"

我笑着握住她的手："有你在旁边看着，我当然要表现得好看点。"

零音在旁边轻轻哼了一声，转过头去看向前面树林，不过我看到她嘴角微微翘了一下，也没说什么。

继续往前走，又走了大约半个时辰，我们路过一片废墟。这里原来应该是一个小型聚落，现在只剩下断壁残垣，墙上沾满了黑色的熵魔腐蚀痕迹。在废墟入口，一块半截石碑倒在地上，上面还能辨认出字迹。

赵石蹲下来，擦掉石碑上的灰尘，辨认着上面的字，越看脸色越阴沉。他抬起头，对着我们说道："是黑石城残碑，十五年前，这里有一百多户人家，三百多口人，都是受不了万维联盟的算力税，逃到混沌域来开荒的。"

"然后呢？"大锤问道。

"然后？"赵石苦笑一声，"万维联盟说他们私开域名，犯法，派了一队人过来，把能带走的东西都抢走了，放了一把火，还故意引了一堆熵魔过来，把剩下的人都吃了。你看这里——"

他指着石碑底部，那里刻着一行小字，"万维联盟，不得好死"，字迹已经被雨水冲刷得模糊了，但那股怨气，隔着一百多年还能感觉到。

所有人都沉默了，气氛一下子变得沉重起来。这些事，大家都听过，但亲眼看到残碑，感受还是不一样。这些都是活生生的人，就因为不想被吸血，想找个地方安身，结果落得这么个下场。

"万维联盟就是这么干的，"我蹲在石碑前，用手抚摸着那行模糊的字迹，声音低沉，"他们不允许任何人在他们垄断之外开域，谁开，他们就灭谁。今天这石碑，就是他们罪证。等我们在这儿站稳了，就把这块碑立在开源城门口，让所有人都记住，我们为什么要来这里，我们要反抗的是什么。"

"对！立在城门口！让所有人都记住！"赵石咬着牙，狠狠地说道。

众人默默绕过废墟，继续往前走，每个人脸色都不好看，但脚步却比之前更坚定了。

又走了大约一个时辰，天边泛起了鱼肚白，天快亮了。我们转过一片乱石岗，前面豁然开朗，出现了一个巨大的盆地。盆地中心，一块巨大的水晶基岩露出地面，哪怕隔着几里地，都能感觉到那里面蕴含着庞大的算力波动。

"就是那里了！"赵石激动得声音都抖了，他这辈子都在中小势力夹缝里求生，从来没想过能有一块自己的域名，"上古域名核心！真的还在！"

我们加快脚步，走到水晶基岩跟前。这块基岩真大，足有一个篮球场那么大，高出地面两三丈，表面刻满了上古铭文，虽然经过亿万年岁月，很多铭文已经模糊不清，但依旧能感觉到那股厚重的历史感。我伸手摸上去，冰凉的触感顺着指尖传过来，基岩内部，隐隐有数据流在缓缓流动，说明核心其实一直都在运转，只是缺了一块，所以不稳定。

绕着基岩走了一圈，我们找到了那个缺口。在基岩朝北的一面，确实缺了一大块，大约有一人多高，断面黑黢黢的，都是熵魔侵蚀留下的痕迹，黑色的晶屑一碰就往下掉。

"就是这里坏了。"大锤伸手敲了敲缺口周围的晶壁，发出沉闷的响声，"问题挺大的啊，万相盟主，咱们这点人，能修好吗？我听说之前有个大势力来了几百人，修了半年都没修好，最后被熵魔攻破了核心，全灭在这里了。"

赵石也皱着眉："是啊，盟主，修复域名核心得花海量算力，我们二十多个人，就算一起把所有算力都投进去，不知道要修到猴年马月去。万维联盟要是那时候打过来，我们根本挡不住。"

其他人也都沉默了，脸上露出担忧。他们能跟着我来混沌域，已经是把命赌上了，但是真看到这么大的烂摊子，心里还是没底。

我笑了笑，没急着解释，转头看向叶清语。她正伸手轻轻触摸着缺口边缘的黑色晶屑，听到我看她，抬起头，对上我的目光，眼睛弯弯的："我相信你，你肯定有办法。"

就这么一句话，我心里一暖。不管什么时候，她都是这样，永远相信我。我走过去，轻轻帮她理了理被风吹乱的鬓角，她的头发上沾了一点草屑，我拿掉草屑，她微微仰头，对着我笑，阳光从盆地边缘照过来，落在她脸上，好看得让我心跳漏了一拍。

"放心吧，我有办法，而且这个办法，正好需要大家一起出力。"我转过身，对着所有人高声说道，"万维联盟修复核心，都是让顶层高手出大部分算力，底层小角色只能打打下手，所以效率很低，一旦顶层算力不够，就修不下去。"

"但是我们不一样，我们是开源盟，我们讲究的是众人拾柴火焰高，每个人都贡献一点算力，大家一起参与修复，用分布式共识算法，每个人都来验证每一段代码，这样修复起来，比集中式修复快好几倍！"

我掏出早就画好的修复分块图，展开给大家看："我已经把缺口分成了二十四块，正好我们二十四个人，每个人负责一块，大家各自修复自己那一块，然后通过共识算法确认，只要超过半数确认没问题，这块就算修好，最后拼起来就是完整的了。"

所有人都愣住了，他们从来没听过这种修复方法。赵石挠了挠头："盟主，这...这样真能行？每个人修自己的，拼起来不会出错吗？"

"不会，"我肯定地点头，"万相框架我已经写好了分布式共识模块，每个人修好一块，框架会自动同步验证，出错了会马上提示，重新来就是了。比一个人修一整块，效率高多了，而且就算哪个人出点错，也不会影响整个核心。"

零音第一个点头："我相信盟主的办法，我先来，哪一块？"

"我也来！"大锤往前一站，瓮声瓮气地说，"俺力气大，算力足，给俺一块大的！"

"算我一个！"
"我们都跟着盟主干！"

看到零音和大锤都表态了，其他人也纷纷站出来，没人退缩。我攥了攥手里的分块图，鼻子有点发酸，这么多人愿意信我，这份信任，我不能辜负。

我按照每个人的算力大小，分配了修复块，算力大的负责块头大一点的，算力小的负责小块。然后我把万相框架的客户端，通过传讯玉符传到每个人手里，教他们怎么用。这个客户端很小，只需要占用一点点丹田空间，就能连接到核心，参与共识。

"好了，现在大家各就各位，注入真气，开始修复吧！"

一声令下，二十四个人，分别站在自己负责的修复块前，同时运转丹田，把真气注入水晶基岩。万相框架自动启动，分布式网络建立起来，每个人的修复进度，都实时同步到所有人那里。

神奇的事情发生了。

一开始，修复进度走得很慢，每个人都在摸索，但是随着第一块修复完成，超过半数共识通过，那块水晶瞬间恢复了光泽，淡淡的青色光芒从里面透出来。所有人都看到了希望，修复速度一下子快了起来。

一块接一块，修复完成的提示不断在框架里跳出来，每修好一块，核心的算力波动就稳定一分，整个盆地里的空气都跟着清新一分。那些原本躲在远处窥视的熵魔，感受到核心重新变得稳定，吓得纷纷退走，不敢靠近。

一天...

两天...

三天...

我们轮流休息，吃饭都是简单啃点干粮，修复工作一刻没停。叶清语不仅要修复自己那块，还要帮那些真气耗竭的人恢复，忙前忙后，一点怨言都没有。每天晚上我替她按摩肩膀，她都靠在我怀里说不累，看着核心一点点变好，她比谁都开心。

第三天傍晚，最后一块修复完成，共识通过。

轰隆隆——

整个水晶基岩突然震动起来，爆发出一团耀眼的青色光芒，光芒直冲云霄，把半个混沌域的天空都照亮了。我能感觉到，整个混沌域的数据流都在欢呼，都在重新对齐根域名服务器，那些原本扭曲错乱的区域，都慢慢稳定下来。

"成了！"赵石大吼一声，激动得眼泪都出来了，他一把抱住身边的人，"真成了！我们修好核心了！"

所有人都欢呼起来，有的跳起来大喊，有的互相拥抱，还有的人喜极而泣。他们当中很多人，一辈子都被万维联盟压迫，交了一辈子算力税，从来不敢想自己能有一块属于自己的域名，今天，他们做到了。

我站在水晶基岩最高处，从怀里掏出早就准备好的域名铭刻刀，深吸一口气，真气灌注到刀上，在基岩最上方，郑重地刻下四个大字：

**开源城**

刻完最后一笔，我把刻刀收起来，真气催动，四个大字瞬间亮起，青色的光芒凝聚不散，永远留在了水晶基岩上。

从今天起，混沌域有了第一个城池，开源盟有了第一个根据地，比特神域有了第一个开源共享的域名。

欢呼声更大了，二十多个人的吼声，在盆地里久久回荡，惊飞了盆地边缘树林里的所有飞鸟。

等欢呼声渐渐平息下来，我正准备说几句定军心的话，突然，盆地入口方向传来一阵轻微的能量波动，很隐蔽，但零音还是瞬间察觉到了，她手按刀柄，就要冲过去："有人！"

"别冲动，"我拦住她，"应该不是敌人，让他出来吧。"

话音刚落，一个穿着灰色长袍的老人从一棵树后面走出来，他头发胡子都白了，脸上布满了皱纹，但是眼神很亮。我认出来了，就是那个之前给我送地形图，又在流云城给我递纸条的神秘人。

他走到基岩底下，抬头看着我刻的"开源城"三个字，点了点头，赞道："好名字，开源...说得好，这个世界闭源太久了，确实该开源了。"

他从怀里掏出一块令牌，扔给我，我伸手接住，令牌是狐尾形状，上面刻着一个九字，正是九尾狐族的令牌。

"我就是你猜的，一个老开源人，当年跟着九尾狐族一起对抗熵魔，全族都快死光了，我侥幸活下来，躲在混沌域这么多年，就是等着一个能拿到产权，敢来开基的人。"老人看着我，语气郑重，"九尾狐族遗迹在混沌域深处的地下溶洞，等你站稳了脚跟，可以去那里看看，狐族留下了一件好东西，对对抗熵魔很有用。"

我攥紧了狐尾令牌，问道："前辈，你到底是谁？为什么要帮我？"

老人笑了笑，往后退了几步，身影渐渐变得模糊："我是谁不重要，重要的是，当年创造者离开的时候说过，总有一天，会有一个懂源代码的人回来，重新开启这个世界。我只不过是遵守诺言，等着你而已。对了，我在核心深处看到一块奇怪的代码碎片，像是一把钥匙，你以后实力强了，可以去挖出来看看，可能和创造者留下的遗产有关。"

说完，老人的身影彻底消失在了空气中，只留下一阵风吹过，什么都没剩下。

我握着狐尾令牌，若有所思。钥匙...创造者遗产...看来这个混沌域里，藏着不少秘密。那块代码碎片，应该就是大纲说的三枚代码钥匙之一，没想到这么快就有线索了。

"盟主，你看那边！"大锤突然指着盆地边缘的一座小山头上，那里有一个鬼鬼祟祟的身影，看到我们发现他，转身就跑。

零音眼神一冷，就要追上去："是万维联盟的探子！我去杀了他！"

"不用追，"我拉住她，摇了摇头，"让他回去报信吧，告诉欧阳震，我们已经在混沌域站稳脚跟了，让他有本事就来抢，我们接着就是。"

零音收刀站好，点了点头，不再说什么。她就是这样，话不多，但我说什么她都听。

天色完全黑下来了，我们在开源城核心广场，点起了篝火，每个人都拿出自己带的干粮和酒，简单举办了一个开城宴。没有山珍海味，都是普通的干粮和劣酒，但每个人都吃得很开心，脸上都带着对未来的憧憬。

我吃完晚饭，走上水晶基岩的顶端，眺望整个盆地。夜色中，盆地里已经开始有虫鸣，空气清新，再也没有刚进来时那股焦糊味了。

一只温暖的手从后面轻轻搂住我的腰，叶清语靠了上来，把脸贴在我的后背上，轻声说道："终于有家了，真好。"

我转过身，把她搂进怀里，她的头发上有篝火的烟味，还有淡淡的草药香，闻着很安心。零音站在不远处的暗影里，手扶刀柄，警惕地望着盆地入口，她不给我们凑热闹，但她会帮我们守着。

"是啊，有家了。"我看着怀里的叶清语，又看了看黑暗中那个挺拔的身影，心中感慨，"从今天起，这里就是我们的家，万维联盟想要拿回去，就得从我的尸体上踏过去。"

我低头，轻轻吻了吻叶清语的额头，她闭上眼睛，脸上泛起红晕，乖乖靠在我怀里。远处，篝火噼啪作响，人们的说笑声传来，温暖而安稳。

而在盆地外面十里地，那个逃出来的万维联盟探子，气喘吁吁地跑到一片树林里，掏出传讯玉符，对着玉符低声说道："主子，不好了...万相他们真的把核心修好了，还建了城，叫...叫开源城..."

玉符那头，欧阳彻听完，沉默了很久，最后发出一声阴冷的笑："好...好得很...没想到他们真能修成。没关系，让他们先高兴几天，我已经请了万剑宗的三位内核宗师，再过几天，就去混沌域，替我收尸...不，是替我把万相的人头，带回来！"

探子应了一声，收起玉符，他不知道，就在他身后不远的一棵大树后面，一个开源盟的哨兵已经握紧了短刀，悄无声息地摸了上来...

风暴，正在酝酿。但我们已经不再是当年那支无家可归的流浪队伍了。

开源城，立。
