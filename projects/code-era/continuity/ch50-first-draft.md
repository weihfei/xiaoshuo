# 第50章 打通要道 钥匙碎片

三天时间转瞬即逝，整编工作比我预想的还要顺利。

经过黑石那件事，新来的头领们个个都战战兢兢，没人再敢保留编制说二话，五万新编师团顺利组建，各级军官都是从老弟兄里抽调的，军令畅通无阻。

清晨的城主府大厅，我刚喝完一碗叶清语熬的粥，赵凯就拿着军报进来了。

"盟主，三个主力师团都已经出发了，按照您的吩咐，分头去接收周边十二座城池，前锋已经离城三十里了。"

我放下碗筷，接过军报扫了一眼，点点头："好，知道了，继续盯着，有消息立刻报给我。"

赵凯应声退出去，叶清语拿过一块干净巾帕，伸手轻轻帮我擦了擦嘴角，指尖带着淡淡的清香。

"刚熬好了新茶，我给你续上。"她拿起茶壶，给我碗里添满，温热的茶水冒着白汽，"怎么样，现在心里踏实点了吗？"

我看着她温柔的眉眼，笑了笑，握住她的手："有什么不踏实的？北冥毒都死了，周边那些小城，还能翻了天？我就是有点奇怪，十二座城池都送了降表，怎么就三座不降？"

"正好卡在通往东部域的官道上。"我指着铺开的地图，手指点了点三个红点，"青石、黑石、白河，这三座城连成一条线，正好把去二进制宗的路掐断了，你说巧不巧？"

叶清语弯下腰，顺着我的手指看过去，发丝垂下来，轻轻扫过我的手背，有点痒。

"万维联盟干的？"她轻声问道。

"肯定是他们。"我点点头，"欧阳震老狐狸，不可能看着我安安稳稳统一混沌域，肯定提前留了后手。"

话音刚落，门外传来脚步声，零音推门进来，手里拿着一张刚送来的密报，脸色有点冷。

"刚收到的消息，确实是欧阳震安排的。"她走过来把密报递给我，"三座城池的守将都收到了欧阳震的手令，让他们死守，等援军过来，同时掐断所有通往二进制宗的商路。欧阳震还给青云子发了最后通牒，要二进制宗和我们开源盟断绝关系，交出所有和我们有联系的旧部，不然就要荡平青云山。"

我接过密报快速扫了一遍，指尖越捏越紧。

密报后面还附了一张清单，上面写着最近半个月，东部域十几个和开源盟私下有联系的中小势力，全部被万维联盟清洗了。青狼寨三千多人，因为寨主说了一句"开源功法真好用"，被万维联盟的人上门，满门抄斩，寨主的域名核心被挖走，当做战利品送给了北冥毒。还有白水谷，因为收留了几个逃出来的开源盟伤员，整个谷都被抹平了，连老人小孩都没放过。

真狠啊。

我放下密报，呼出一口浊气，心里那股火气蹭一下就上来了。

这就是万维联盟的闭源垄断，谁要是敢说一句开源好，立刻就灭你满门，就是要让所有人都怕他们，都不敢跟我走。

"欧阳震这是在立威。"零音冷冷说道，"他就是要告诉所有人，跟你万相走，就是这个下场。"

就在这时，门口传来轻快的脚步声，九尾蹦蹦跳跳进来了，鼻子嗅了嗅，皱起眉头："怎么这么重的杀气？哦，说到那三座城了？我刚才绕着那一片转了一圈，感觉到不对劲，那三座城里面，有奇怪的代码波动，像是……像是上古遗迹的那种气息，有点像我们狐族守护的那种。"

她走到地图旁边，伸手指了指青石城："尤其是这里，气息最明显，我猜下面可能有东西，和创造者有关的那种。"

创造者……

我下意识摸了摸丹田，那里静静的躺着根域名碎片，暖融融的。来到这个世界这么久，我只知道创造者留下了这个世界，留下了三枚代码钥匙，能打开创造者神殿，看到世界真相。九尾身上有一枚，而我这里有一块碎片，没想到这里又能碰到。

"看来这三座城，我得亲自去一趟了。"我站起身，握紧腰间的排序剑，"欧阳震想掐断我的路，还拿青云子压我，我要是不去，别人还真以为我怕了他万维联盟。"

"我跟你去。"零音立刻说道，手已经按在了刀柄上。

"我也去！"九尾蹦起来，"我帮你找漏洞，看看那遗迹里面到底有什么好东西。"

我笑着点头："好，都去。清语你就留在开源城，这里刚站住脚，离不开你，医疗营还等着你建呢。"

叶清语点点头，伸手帮我理了理衣领，轻声说："好，你放心去，家里有我呢。注意安全，要是不对就先回来，我们等你。"她从怀里掏出一块干净的帕子，塞到我手里，"这个带着，擦汗用。"

我接过帕子，能闻到上面淡淡的清心草香味，捏了捏她的手："放心，我很快回来。"

当天下午，我带着大锤的一千近卫团，和九尾零音一起，轻装出发，直奔青石城。

路比想象的好走，这几天周边的小势力听说盟主亲自来了，都纷纷出来迎接，送粮送水，有的还主动要求派兵跟着一起打青石城。我都谢了，让他们回去安心等着，拿下青石城就给他们开放功法。

三天后，我们已经到了青石城下，在城外五里处扎营。

我站在高坡上，远远就能看到青石城的城墙，城门紧闭，吊桥拉起来了，城头上站满了士兵，弓上弦刀出鞘，严阵以待。守将王虎穿着铁甲，站在城楼最上面，老远就能看到他那张大圆脸。

"万相逆贼！"王虎嗓门很大，域名之力散开，声音顺着风飘过来，"我家盟主大军不日就到，你区区八万乌合之众，也敢称雄混沌域？赶紧开城投降，我家盟主宽宏大量，还能给你留个全尸！不然等大军一到，你就是粉身碎骨！有胆子你就上城来跟我单挑！"

零音站在我左边，一身银甲在阳光下泛着冷光，听到这话，她眼神一寒，手已经按在了刀柄上，就要抽刀出去："我去斩了他。"

我伸手拦住她，笑了笑："不急，他喊这么大声，不就是给他城里的士兵壮胆吗？我满足他，跟他说两句。"

我催动域名之力，声音浩浩荡荡传出去，盖过了王虎的声音："王虎，你在青石城做了三十年城主，我问问你，万维联盟给你多少好处，你要替欧阳震拼命？"

"你收的算力税，十成里面你抽走六成，城里百姓一年辛辛苦苦挖的比特结晶，大半都交了税，孩子都没钱修炼，你告诉他们，这就是万维联盟的正道？"

我顿了顿，声音提高了一度，让城头上每个士兵都能听得清清楚楚："我万相在这里说清楚，只要你们开城投降，三件事：第一，所有低中级功法，全部开放源代码，任何人都可以免费去藏经阁学，不用再花大价钱从万维联盟买；第二，算力税减半，原来十成抽六成，我们只抽两成，剩下的都留给你们自己用；第三，除了王虎一人，其他所有人一概不究，原来的军官该干什么还干什么。"

"你们告诉我，跟着万维联盟，你们吃不饱饭练不成级，跟着我开源盟，人人都能修炼，你们选哪个？"

这番话喊出去，城头上瞬间安静下来，紧接着就是一阵嗡嗡的议论声，我能看到，不少士兵都放下了弓箭，交头接耳，眼神都活泛起来了。

王虎脸色一下子就变了，猛地拔出刀，怒吼道："别听他胡说！他这是妖言惑众！谁敢开城门，我先砍了他！"

但已经晚了。

他话音刚落，城头上就乱了，几个士兵猛地扑上去，从后面抱住了王虎，王虎怒吼着挣扎，一刀砍死了一个，但更多的人涌了上来，直接把他按在地上。

"咔嚓"一声，城门的锁链被砍断，吊桥"吱呀呀"放了下来，几个士兵头目绑着王虎，低着头走出来，远远就跪下了："我等愿意开城投降，欢迎盟主进城！"

大锤在旁边看傻了，挠挠头："盟主，这……这不费一兵一卒就拿下来了？"

我笑了笑，翻身上马："不然呢？得道多助失道寡助，万维联盟失了人心，这座城，本来就不是王虎的了。"

我们骑着马，堂堂正正进了青石城，百姓都夹道欢迎，不少人手里拿着鸡蛋果子，往我们怀里塞，看得出来，他们是真高兴。

进了城主府，我先让人把王虎带下去，核对他这些年贪了多少，把赃款都吐出来，分给城里的百姓，然后安排好城防，就让九尾带着几个探子，去她感觉到异常的地方看看。

九尾转了一圈，回来眼睛亮晶晶的，拽着我的袖子就往后面走："就在后花园假山下面，我感觉到了，气息就是从那里来的！"

我们移开假山，下面果然有一个暗门，推开暗门，一条石阶通往地下密室。

我提着排序剑走在前面，零音和九尾跟在后面，一路往下走了大概十几丈，来到一间不大的密室里。

密室里面空空荡荡，只有一张桌子，上面放着一个盒子，盒子打开，里面躺着一万枚码得整整齐齐的比特结晶，还有一块巴掌大的乳白色碎片，静静地躺在那里，散发着淡淡的古老气息。

我走过去，拿起那块碎片，入手温热，一股熟悉的信息流顺着指尖流进来，和我丹田里面根域名碎片的气息一模一样。

九尾凑过来，鼻子嗅了嗅，点头肯定地说："就是这个气息！和我那块钥匙碎片是一样的！这就是代码钥匙的碎片！"

代码钥匙……

我拿着碎片，心里有点激动。原来创造者把钥匙分成了好几块，散落在世界各地，九尾那里有一块，我这里有一块，这里又找到一块，集齐三块就能打开创造者神殿了。

没想到打青石城，还能有这种意外收获。

就在我们拿到碎片的时候，外面探子匆匆跑进来，递上来一份紧急军报："盟主！万维联盟的援军到了！大约五千人，由北冥毒的副将崔雄带领，已经离青石城不到三十里了！"

我把碎片收进空间戒指，笑了笑，对零音和九尾说："来得正好，王虎我们收了，城池我们拿了，这块碎片也到手了，正好再来一波，送崔雄一块跟着北冥毒一块儿上路。"

我提着排序剑，大步往密室外面走："传令下去，全军整备，出城迎敌！"

夕阳染红了青石城的城墙，我站在城门口，远远就能看到北方天际，滚滚烟尘已经升起来了，万维联盟的援军，真的来了。
