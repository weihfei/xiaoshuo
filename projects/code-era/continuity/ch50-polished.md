# 第50章 青石城头 钥匙碎片

三天时间转瞬即逝，整编工作比我预想的还要顺利。

经过黑石那件事，新来的头领们个个都战战兢兢，没人再敢提保留编制的话，五万新编师团顺利组建，各级军官都是从老弟兄里抽调的，军令畅通无阻。

清晨的城主府大厅，我刚喝完一碗叶清语熬的肉粥，赵凯就拿着军报进来了。

"盟主，三个主力师团都已经出发了，按照您的吩咐，分头去接收周边十二座城池，前锋已经离城三十里了。"

我放下碗筷，接过军报扫了一眼，点点头："好，继续盯着，有消息立刻报。"

赵凯应声退出去。叶清语拿过一块干净巾帕，伸手轻轻帮我擦了擦嘴角，指尖带着淡淡的清心草香气。

"刚熬好了新茶，我给你续上。"她拿起紫砂壶，给我碗里添满，温热的茶水冒着软软的白汽，"怎么样，现在心里踏实点了吗？"

我看着她温柔的眉眼，笑了笑，顺势握住她微凉的手："有什么不踏实的？北冥毒都死了，周边那些小城，还能翻了天？我就是有点奇怪，十二座城池十一座都送了降表，怎么就三座不降？"

"正好卡在通往东部域的官道上。"我指着铺在桌上的地图，手指点了点三个红圈，"青石、黑石、白河，这三座城连成一条线，正好把去二进制宗的路掐断了，你说巧不巧？"

叶清语弯下腰，顺着我的手指看过去，一缕发丝垂下来，轻轻扫过我的手背，有点发痒。

"万维联盟干的？"她轻声问。

"肯定是他们。"我点点头，"欧阳震老狐狸，不可能看着我安安稳稳统一混沌域，肯定提前留了后手。"

话音刚落，门外传来沉稳的脚步声，零音推门进来，手里捏着一张刚送到的密报，脸色比平时更冷。

"刚收到的消息，确实是欧阳震安排的。"她走过来把密报递给我，"三座城池的守将都收到了欧阳震的手令，让他们死守待援，同时掐断所有通往二进制宗的商路。欧阳震还给青云子发了最后通牒，要二进制宗和我们开源盟断绝关系，交出所有和我们有联系的旧部，不然就要荡平青云山。"

我接过密报快速扫了一遍，指尖越捏越紧。

密报后面还附了一张血淋淋的清单：最近半个月，东部域十几个和开源盟私下有联系的中小势力，全部被万维联盟清洗了。青狼寨三千多人，就因为寨主酒后说了一句"开源功法真好用"，被万维联盟的人上门，满门抄斩，寨主的域名核心被硬生生挖走，当做战利品送给了北冥毒。还有白水谷，仅仅因为收留了几个逃出来的开源盟伤员，整个山谷都被抹平，连刚满周岁的孩子都没放过。

真狠。

我放下密报，呼出一口浊气，心里那股火气蹭一下就顶到了天灵盖。

这就是万维联盟的闭源垄断——谁要是敢说一句开源好，立刻就灭你满门，就是要让所有人都怕他们，都不敢跟我走。这种统治，和把所有人都当成算力奶牛有什么区别？

"欧阳震这是在立威。"零音冷冷开口，"他就是要告诉所有人，跟你万相走，就是这个下场。"

就在这时，门口传来轻快的脚步声，九尾蹦蹦跳跳进来了，鼻子习惯性地嗅了嗅，立刻皱起了狐狸耳朵："怎么这么重的杀气？哦，说到那三座城了？我早上绕着那一片飞了一圈，感觉到不对劲，那三座城里面，有奇怪的代码波动，像是……像是上古遗迹的那种气息，跟我们狐族守护的那块碎片味道差不多。"

她走到地图边，纤细的手指点了点青石城的位置："尤其是这里，气息最明显，我猜地下肯定埋着东西，和创造者有关的那种。"

创造者……

我下意识摸了摸丹田，那块根域名碎片静静地躺在那里，依旧是暖融融的。来到这个世界这么久，我只知道创造者留下了这个世界，留下了三枚代码钥匙，集齐就能打开创造者神殿，看到完整的世界真相。九尾身上有一块完整碎片，我灵魂里带着一块，没想到在这里又能碰到一块。

"看来这三座城，我得亲自去一趟了。"我站起身，握紧腰间排序剑的剑柄，"欧阳震想掐断我的路，还拿青云子压我，我要是不去，别人还真以为我万相怕了他万维联盟。"

"我跟你去。"零音立刻接话，手已经按在了腰间短刀的刀柄上。

"我也去我也去！"九尾蹦起来，拽住我袖子晃了晃，"我帮你找漏洞，看看那遗迹里面到底藏了什么好东西。"

我笑着点头："好，都跟我去。清语你就留在开源城，这里刚站住脚，离不开你，医疗营还等着你建起来呢。"

叶清语点点头，伸手帮我理了理被风吹乱的衣领，指尖轻轻碰了碰我的锁骨，声音温柔得像水："好，你放心去，家里有我呢。注意安全，要是不对就先回来，我们都等着你。"她从怀里掏出一块叠得整整齐齐的素色帕子，塞到我手里，"这个带着，擦汗用。"

我接过帕子，上面还带着她身上淡淡的清心草香，捏了捏她的手："放心，我很快回来。"

当天下午，我带着大锤的一千近卫团，和九尾零音一起，轻装出发，直奔青石城。

路比想象的好走，这几天周边的小势力听说盟主亲自来了，都纷纷出来迎接，送粮送水，有的还主动要求派兵跟着一起打青石城。我都谢了，让他们回去安心等着，拿下青石城就给他们开放功法。

三天后，我们已经到了青石城下，在城外五里处扎营。

我站在一处高坡上，远远就能看到青石城青灰色的城墙，城门紧紧闭着，吊桥高高拉起，城头上站满了弓上弦刀出鞘的士兵，严阵以待。守将王虎穿着一身黑铁甲，挺着大肚子站在城楼最上面，老远就能看到他那张油光满面的圆脸。

"万相逆贼！"王虎嗓门很大，催动域名之力把声音送过来，顺着风飘得老远，"我家盟主大军不日就到，你区区八万乌合之众，也敢称雄混沌域？赶紧开城投降，我家盟主宽宏大量，还能给你留个全尸！不然等大军一到，你就是粉身碎骨！有胆子你就上城来跟我单挑！"

零音站在我左手边，一身银甲在阳光下泛着冷光，听到这话，她眼神瞬间寒了下来，手已经按在了刀柄上，就要抽刀出城斩了他："我去杀了他。"

我伸手拦住她，笑了笑："不急，他喊这么大声，不就是给他城里的士兵壮胆吗？我满足他，跟他说两句。"

我催动内核级域名之力，声音浩浩荡荡传出去，一下子就盖过了王虎的声音，城头上每个角落都能听得清清楚楚：

"王虎，你在青石城做了三十年城主，我问问你，万维联盟给了你多少好处，你要替欧阳震拼命？"

"你收的算力税，十成里面你抽走六成，城里百姓一年辛辛苦苦挖的比特结晶，大半都交了税，孩子都没钱修炼，你告诉他们，这就是万维联盟的正道？"

我顿了顿，声音又提高了一度，一字一句砸在城头上每个人耳朵里：

"我万相在这里把话讲清楚，只要你们开城投降，我承诺三件事：第一，所有低中级功法，全部开放源代码，任何人都可以免费去藏经阁学，不用再花大价钱从万维联盟买；第二，算力税减半，原来十成抽六成，我们只抽两成，剩下的都留给你们自己用；第三，除了王虎一人，其他所有人一概不究，原来的军官该干什么还干什么。"

"你们自己说，跟着万维联盟，你们吃不饱饭练不成级，跟着我开源盟，人人都能好好修炼，你们选哪个？"

这番话喊出去，城头上瞬间安静了一秒钟，紧接着就是一阵嗡嗡的议论声，像炸开了马蜂窝。我能远远看到，不少士兵都放下了手里的弓箭，交头接耳，原本紧绷的脸一下子就活泛起来了。

王虎脸色"唰"一下就变成了猪肝色，猛地拔出腰刀，指着下面怒吼："别听他胡说！他这是妖言惑众！谁敢开城门，我先砍了他！"

但已经晚了。

他话音刚落，城头上就乱了，几个膀大腰圆的士兵猛地从后面扑上来，一下子抱住了王虎的胳膊腿。王虎怒吼着挣扎，一刀砍死了一个，但更多的人涌了上来，直接把他按倒在地，刀都被打飞了。

"咔嚓"一声脆响，城门上的锁链被砍断，吊桥"吱呀呀"一声慢慢放下来，几个带头的士兵头目绑着王虎，低着头走出来，隔着老远就跪下了：

"我等愿意开城投降，欢迎盟主进城！"

旁边的大锤看傻了，挠了挠后脑勺，瓮声瓮气地说："盟主，这……这不费一兵一卒就拿下来了？"

我笑了笑，翻身上马，握紧缰绳："不然呢？得道多助失道寡助，万维联盟失了人心，这座城，本来就不是他王虎的了。"

我们骑着马，堂堂正正进了青石城，百姓都夹道欢迎，不少人手里拿着鸡蛋果子，往我们怀里塞，脸上都是藏不住的笑。看得出来，他们是真高兴，被王虎和万维联盟压了三十年，终于能喘口气了。

进了城主府，我先让人把王虎带下去，核对他这些年贪了多少赃款，全部查抄出来，分给城里的百姓，然后安排好城防，就让九尾带着几个灵敏的探子，去找她感觉到异常的地方。

九尾转了小半个时辰，跑回来的时候眼睛亮晶晶的，拽着我的袖子就往后花园走："就在假山下面！我敢肯定，气息就是从那里出来的！"

我们移开千斤重的假山石，下面果然露出一扇黑沉沉的暗门，推开暗门，一股带着霉味的古老空气扑面而来，一条石阶弯弯曲曲通往地下密室。

我提着排序剑走在前面，零音和九尾跟在后面，一路往下走了十几丈，眼前豁然开朗，来到一间不大的方形密室。

密室里面空空荡荡，只有一张落满灰尘的胡桃木桌子，桌子上放着一个紫檀木盒子。打开盒子，里面躺着一万枚码得整整齐齐的标准比特结晶，结晶旁边，一块巴掌大的乳白色碎片静静地躺在那里，散发着淡淡的、悠远的古老气息。

我走过去，伸出手拿起那块碎片，入手温热，一股熟悉的信息流顺着指尖流进来，和我丹田深处根域名碎片的气息隐隐呼应，一模一样。

九尾凑过来，小鼻子皱着嗅了嗅，立刻点头，声音带着点兴奋："就是这个气息！和我那块钥匙碎片完全一样！这就是代码钥匙的碎片没错！"

代码钥匙……

我握着这块温凉的碎片，心里有点激荡。原来创造者早就把钥匙分成了好几块，散落在世界各地，等着有缘人来集齐。九尾那里有一块，我灵魂里带着一块，今天又在这里找到一块，离集齐三枚打开创造者神殿，又近了一步。

没想到打一趟青石城，还能有这种意外收获。

就在这时，密室入口传来急促的脚步声，探子满头大汗跑进来，递上一张紧急军报：

"盟主！万维联盟的援军到了！大约五千人，是北冥毒的副将崔雄带领，已经离青石城不到三十里了！"

我把碎片收进空间戒指，抬手拍了拍剑鞘，笑了一声，对零音和九尾说："来得正好。王虎我们收了，城池我们拿了，碎片也到手了，正好再来一波，送崔雄下去跟北冥毒团聚。"

我提着排序剑，大步往密室外面走，声音稳稳传来："传令下去，全军整备，出城迎敌！"

夕阳染红了青石城的城墙，把我的影子拉得很长。我站在城门楼下，远远就能看到北方天际，滚滚烟尘已经升起来了，万维联盟的援军，真的来了。

这一次，就让他们有来无回。
