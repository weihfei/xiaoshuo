# 《代码纪元》事实数据库

## 世界观事实

### 世界本质
- 比特神域是由源代码构建而成的虚拟世界，运行在地球人类开发的量子计算机之上
- 万物皆代码，生灵=可执行进程，物体=数据对象，地域=内存空间
- 底层是量子比特海，所有代码从这里生成，最终回归这里
- 起源：原本是地球某互联网公司的元宇宙实验项目，项目搁置后世界自主进化了亿万年
- 现存问题：累积了无数BUG、内存泄漏和垃圾进程，演化出了毁灭一切的"熵魔"

### 地理结构（按网络分层）
- **物理层**：量子比特海，最底层，普通生灵无法长期生存，有连接地球的网关入口
- **数据链路层**：蛮荒代码平原，包含黑铁聚落（新手村）、铁锈荒原、碎片区
- **网络层**：域名大陆，被各大势力按域名层级分割统治
  - `com.二进制宗`：东部青云山区，老牌宗门
  - `com.万维联盟`：中央肥沃平原，当前霸主，掌控根服务器入口
  - `com.混沌域`：被遗弃的顶级域名，主角后来拍下开发
  - `org.杀毒者`：极北冰原，杀毒者组织总部
  - `io.开源盟`：主角建立的新势力，核心在混沌域
- **传输层**：云端悬浮群岛，顶级高手修炼圣地，万维联盟总部所在地
- **应用层**：创造者神殿，世界最高处，存放创世README和系统权限，需要三把代码钥匙进入
- **混乱感染区**：被熵魔感染的区域，代码混乱，熵魔滋生，不断扩大

### 势力分布
- **万维联盟**：老牌统治阶级，闭源垄断组织，掌控中央顶级域名+云端根服务器
  - 核心理念：顶级代码由少数人掌控，底层只能使用编译好的二进制文件
  - 统治方式：通过域名分配权收取算力税，吸食底层算力，封锁底层接触源代码
  - 代表人物：万维盟主欧阳震（域主级），欧阳彻（盟主之子，前期反派）
- **二进制宗**：传统宗门，主角初期加入的宗门，占据东部青云山区
  - 核心理念：循序渐进悟道源码，传承数十万年
  - 宗主是穿越者后代，暗中关注主角，对主角有知遇之恩
  - 现任宗主：玄机子（内核级巅峰），核心长老：玄真（内核级三段）、玄静（内核级二段）
- **杀毒者组织**：创造者留下的安全进程，总部在极北冰原
  - 表面使命：清除一切"异常代码"，维护现有秩序
  - 真实使命：等待能重构系统的人出现，帮助完成热重启
  - 代表人物：零音（统领，第三位红颜知己）
- **开源盟**：主角建立的新兴势力，核心理念开源共享，核心根据地混沌域
  - 目前规模：初创阶段，核心成员为主角+三位核心弟子赵凯、钱磊、孙博
  - 未来规划：集齐三枚代码钥匙，探索代码神殿，获取创造者传承，推广开源修炼体系

### 修炼体系
- 比特战士（入门）→ 位元师 → 字节大师 → 内核宗师 → 域主 → 根服务器创世神
- 修炼本质：打磨自己的核心代码，提升算力，优化运行效率
- 原住民修炼方式：只会调用现成API，使用编译好的闭源二进制，无法修改底层
- 主角金手指：能直接看到底层源代码，可以修改优化，用程序员知识解决一切问题
