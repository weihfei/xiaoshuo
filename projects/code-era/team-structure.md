# 通用小说创作AI协作团队

用于小说项目的分工协作，当前支持《代码纪元》项目。

13个独立AI子agent分工协作，所有角色定义文件都在本目录：

| 角色 | 对应文件 | 类型 | 职责 |
|------|----------|------|------|
| **老板** | - | 人类 | 总导演、设定把控、最终审稿 |
| **小虾** | - | 主agent | 项目管理、协同调度、整体排坑 |
| **写作技巧研究员** | `novel_trending_researcher.md` | 子agent | 每日搜索热门写作技巧，持续精进，更新写作指南 |
| **主线剧情师** | `novel_outline_planner.md` | 子agent | 设计整体剧情走向、大纲 |
| **世界观架构师** | `novel_world_builder.md` | 子agent | 构建维护世界观、能力体系 |
| **人设师** | `novel_character_designer.md` | 子agent | 设计维护人物设定、性格特点 |
| **情绪共鸣师** | `novel_emotion_enhancer.md` | 子agent | 强化人物情绪表达，润色心理描写，提升读者共情 |
| **情节设计师** | `novel_plot_designer.md` | 子agent | 设计具体情节、桥段 |
| **章节写手** | `novel_first_draft_writer.md` | 子agent | 撰写具体章节初稿 |
| **编辑/校对** | `novel_editor.md` | 子agent | 检查错别字、基础逻辑 |
| **修订者（评论家）** | `novel_reviewer.md` | 子agent | 深度逻辑审查、设定一致性 |
| **润色师** | `novel_polisher.md` | 子agent | 优化文笔、节奏、统一文风 |
| **类型专家** | `novel_genre_specialist.md` | 子agent | 科幻爽文专业审查 |
| **版权顾问** | `novel_copyright_advisor.md` | 子agent | 版权合规咨询 |

## 协作流程（更新版，含每日精进）

1. **每日启动**：`写作技巧研究员`搜索最新热门写作技巧，更新项目`writing-tips.md`，所有创作角色参考最新技巧
2. 老板提出需求/方向 → **主线剧情师**输出整体大纲
3. **世界观架构师**完善世界观和能力体系
4. **人设师**确认人物设定和人物卡
5. **情节设计师**设计具体卷/章情节
6. **章节写手**撰写具体章节初稿
7. **情绪共鸣师**强化人物情绪表达，润色心理描写，提升读者共情
8. **编辑/校对**和**修订者（评论家）**并行审查挑错
9. **润色师**优化文笔、节奏、统一文风
10. **类型专家**做类型专业审查
11. **版权顾问**版权合规检查
12. 老板最终审稿
