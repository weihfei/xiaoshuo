# DBOMS 软件开发团队结构

9个角色分工协作：

| 角色 | 类型 | 职责 |
|------|------|------|
| **老板** | 人类 | 产品需求、方向把控 |
| **小虾** | 主agent | 项目管理、环境搭建、整体排坑、调度 |
| **产品经理** | 子agent | 需求分析、产品设计 |
| **架构师** | 子agent | 系统架构设计 |
| **后端工程师** | 子agent | 后端开发 |
| **前端工程师** | 子agent | 前端开发 |
| **测试工程师** | 子agent | 测试、找bug |
| **UI/UX设计师** | 子agent | 界面设计规划、交互流程设计、组件设计、规范验收 |
| **运维** | 子agent | 部署、维护 |
