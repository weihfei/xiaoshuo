## 一、"心眼"修炼体系

### 1.1 古籍来历

**《心眼经》**——上古时期某位大能遗留的修炼功法，原为修仙界顶级秘典。

- **起源**：上古时期，天地灵气充沛，人族与万族并存。有大能者观人心善恶，创出此功法，名为《心眼经》，以心为镜，以眼为门，可洞察世间万法本心。
- **沉睡原因**：上古大战后，灵气衰竭，功法失去传承条件，自动封存休眠。
- **选择官平的原因**：
  1. 官平天生心性纯正（父母早亡，历经磨难却不堕其志）
  2. 遭遇失明重大变故时，心境发生质变，符合功法激活条件
  3. 灵魂与功法残留印记产生共鸣
- **隐藏设定**：古籍并非只是功法，其中封印着上古大能的一缕神识，会在修炼过程中逐步苏醒，传授功法同时也在观察官平品行。

### 1.2 修炼等级划分

| 等级 | 名称 | 能力 | 状态 |
|------|------|------|------|
| 初期 | **明心** | 能感知他人情绪波动，善恶分明 | 失明后意外激活 |
| 中期 | **照见** | 能看透人心具体想法，看穿谎言 | 中期巅峰 |
| 后期 | **天眼** | 能短暂预知未来走向，趋吉避凶 | 后期 |
| 圆满 | **心眼合一** | 心中有眼，眼中有心，洞察万物 | 圆满 |

### 1.3 各等级详解

**【初期·明心】**
- 触发条件：失明+心境质变
- 能力表现：
  - 能清晰感知周围人的情绪波动（喜、怒、哀、乐、恐惧、敌意等）
  - 对善恶有本能直觉，好人坏人一眼分辨
  - 情绪强烈者感知更清晰
- 限制：
  - 无法读取具体想法，只能感知情绪
  - 需要目光接触才能触发
  - 情绪波动剧烈者会短暂头痛

**【中期·照见】**
- 触发条件：首次生命危险+强烈求生意志
- 能力表现：
  - 能看穿他人具体想法，如同读心
  - 能感知谎言，听出话语中的虚假成分
  - 对微表情、微动作有敏锐捕捉
- 限制：
  - 看穿深度与对方精神力有关，精神力强者只能看表层
  - 无法看穿同一等级或更高等级修炼者的心思
  - 使用过度会精神疲惫，严重时流鼻血

**【后期·天眼】**
- 触发条件：重大抉择关头+强烈正义感驱动
- 能力表现：
  - 能短暂预知几秒后的未来走向
  - 能看穿阵法、幻术、伪装
  - 对命运线有模糊感知，能察觉针对自己的阴谋
- 限制：
  - 预知不是绝对的，只是概率性预示
  - 每天使用次数有限（3次）
  - 强行多次使用会折寿

**【圆满·心眼合一】**
- 能力表现：
  - 心中有眼，眼中有心，洞察万物本心
  - 可短暂看透因果，追溯往事
  - 修炼至圆满可庇佑亲友感知危险
- 限制：
  - 古籍封印的神识会完全消散，大能彻底离去
  - 圆满之后再无突破可能

### 1.4 修炼条件与代价

**修炼条件**：
- 必须心性纯正（私欲过重者无法入门）
- 需要持续做好事积累"功德"才能突破
- 情绪波动过大时会走火入魔（需要静心）
- 修炼需要消耗体力，需保持身体健康

**修炼代价**：
- 每次突破都会承受剧痛（洗经伐髓）
- 使用能力会消耗精神力
- 功德不足强行突破会导致能力失控
- 修炼过程必须禁欲清心（初期），否则无法突破

### 1.5 "心眼"能力边界

1. **不能无敌**：同等级修炼者互相无法看透；精神力强的人抵抗力强
2. **有代价**：使用需消耗，使用过度有惩罚
3. **有条件**：需要目光接触，需精神清醒
4. **可成长**：敌人也会变强，主角必须持续修炼
5. **可剥夺**：有特殊手段可以暂时封印或剥夺"心眼"能力

## 二、都市社会背景

### 2.1 城市设定

**天海市**——南方沿海一线城市，经济发达，律所林立。

- 经济地位：华南地区金融中心，GDP排名前三
- 都市氛围：繁华与阴暗并存，上流社会纸醉金迷，底层平民苦苦挣扎
- 律所生态：
  - **顶级所**：金诚律师事务所（天海前三，官平所在）
  - **普通所**：各类中小律所
  - **黑律所**：与灰色产业勾结的律所，专门帮人洗钱、脱罪

### 2.2 律所行业生态

**金诚律师事务所**：
- 定位：正规顶级律所，主攻商业、刑事、知识产权
- 规模：50+执业律师，天海市排名前五
- 内部派系：主任派、合伙人派、年轻律师派
- 官平身份：实习律师，刑事部

**行业规则**（热血+搞笑）：
- 案源为王，没有案源的律师就是无根浮萍
- 人脉关系决定能不能接到大案
- 胜诉率是律师的生命线
- 律所之间竞争激烈，不乏恶意竞争
- **"年轻人多锻炼" = 免费劳动力的代名词**
- **同事请喝奶茶 = 让你帮忙干活的信号**
- **"这个案子很简单" = 你要累死了**

### 2.3 上流社会与底层律师对比

| 层级 | 特征 | 代表人物 |
|------|------|----------|
| 顶层 | 权贵家族，政商通吃 | 四大家族掌门人 |
| 上层 | 成功企业家，亿万富豪 | 女主B苏锦瑟 |
| 中层 | 金领、高管、中产律师 | 金诚合伙人 |
| 底层 | 实习律师、刚入行新人 | 官平（初期） |

## 三、势力格局与主要冲突

### 3.1 主要势力

**【正道阵营】**

  - 郑家（政界大佬，幕后操控）
  - 王家（黑道教父，手段狠辣）
  - 李家（商业巨头，财力惊人）
  - 陈家（律政世家，与金诚死对头）

### 3.2 冲突根源（层层递进）

**第一阶段·小试牛刀**：

## 四、修炼体系强化（参考热门网文）

### 4.1 等级突破的戏剧性

参考《逆天邪神》《太荒吞天诀》等热门作品的等级设定：

- **突破有风险**：每次突破都可能失败，失败会有反噬（头痛、流鼻血、短期失明）
- **突破有代价**：需要"功德值"，做坏事会扣功德，功德不足强行突破会失控
- **同级无敌规则**：同等级修炼者互相无法看透心思，增加对抗趣味

### 4.2 能力升级的爽感设计

- **明心初期**：感知情绪→让官平在法庭上读出证人紧张/撒谎
- **照见中期**：读心术→在谈判中占据绝对优势，让对手无处遁形
- **天眼后期**：预知未来→提前察觉危险，英雄救美，化解阴谋
- **心眼合一**：终极形态→看透因果，但会失去师父（大能神识消散）

### 4.3 修炼日常的搞笑元素

融入日常生活的修炼问题，参考热门网文的"修仙日常梗"：

| 场景 | 搞笑情况 |
|------|----------|
| 打坐被外卖打断 | 正在突破关键时机，门铃响了，是外卖到了 |
| 手机没电误事 | 预知到要出事，结果手机没电，无法求助 |
| 读心术失控 | 不小心听到同事对自己的真实评价（差评） |
| 练功走火入魔 | 情绪一激动，能力失控，把会议室的人吓得够呛 |
| 功德不够 | 想突破但功德不够，只能先去扶老太太过马路 |
| 被对手封印 | 能力被封印，只能靠嘴炮和逻辑推理打官司 |

## 五、三位女主设定（差异化设计）

### 女主A：林知念（正牌女友）

**定位**：温柔治愈系，邻家暖女

- **出场方式**：官平失明期间照顾他日常的好邻居
- **性格特点**：温柔善良，不争不抢，默默守护
- **能力/作用**：在官平最低谷时给予温暖，帮忙整理案情
- **感情线**：官平恢复视力后帮她解决了一个案子，两人萌生好感，最终结婚生有一女
- **与其他女主的区别**：最普通最温柔，是官平的港湾

### 女主B：苏锦瑟（贤内助）

**定位**：霸道御姐，商业女强人

- **出场方式**：因为一个商业案件委托认识官平
- **性格特点**：美艳动人，魔鬼身材，一开始高高在上，调戏官平
- **能力/作用**：为官平的事业提供资金支持，出手阔绰
- **感情线**：接触中逐渐爱上官平，英雄救美后确立关系，生有一子
- **与其他女主的区别**：有钱有势，主动倒追，最后成为官平背后的女人

### 女主C：沈夜（暧昧杀手）

**定位**：神秘危险，暗中守护

- **出场方式**：王家派来刺杀官平的杀手
- **性格特点**：冷酷无情，身手了得，但内心有伤痛
- **能力/作用**：被官平策反，帮助官平处理很多不方便在明面处理的问题，多次救官平于危难
- **感情线**：一直暧昧，身份隐秘，结局未明确交代
- **与其他女主的区别**：不能见光的身份，是官平的暗刃
