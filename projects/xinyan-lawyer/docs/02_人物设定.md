## 一、主角人物卡

### 官平

**身份**：天海市金诚律师事务所实习律师（刑事部）

**基本信息**：
- 年龄：24岁
- 身高：178cm
- 体重：70kg
- 学历：政法大学本科毕业，通过司法考试
- 背景：普通家庭，父亲早逝，母亲改嫁，由外婆带大

**外貌特征**：
- **整体**：相貌端正，眉目清朗，属于耐看型。脸型偏瘦，轮廓分明，颧骨微高，显得有些青涩但透着倔强。
- **眼睛**：瞳孔漆黑有神，眼白干净，眼神锐利但不凌厉（心眼修炼后更加深邃，能让人不敢直视）。失明期间眼睛呈灰白色，恢复后瞳孔恢复漆黑，但眼底隐约有金光流转。
- **眉毛**：剑眉，浓密但整齐，眉尾微微上挑。
- **鼻子**：鼻梁挺直，鼻尖微肉，不算精致但很精神。
- **嘴巴**：唇线分明，嘴唇略薄，不笑时显得严肃，笑起来却意外地温暖。
- **身材**：178cm/70kg，体型匀称，偏精瘦。肩膀不宽但直，腰背挺拔，有军人体态的影子（外婆从小要求站如松坐如钟）。
- **皮肤**：肤色偏白，是那种不显老的冷白皮，但脸庞因常年在室内略显苍白。
- **平时穿着**：普通白衬衫或浅蓝衬衫，黑色西裤，黑色皮鞋。偶尔打领带，但领带总是系得有些歪（手残）。袖子有时会挽到小臂。
- **修炼时**：眉头微皱，眼神变得深邃，气质从普通律师变成压迫感十足的强者。

**性格特点**：
- 表面：沉稳内敛，话不多，但关键时刻敢说敢做
- 内在：热血正义，有点小幽默，容易心软
- 对敌人：冷酷无情，绝不手软
- 对朋友：两肋插刀，宁可自己吃亏

**口头禅**：
- "这事我管定了。"
- "法律不是你们玩弄的工具。"
- "（内心OS：又来？）"

**习惯性动作**：
- 思考时会不自觉地眯起眼睛
- 紧张时会摸鼻尖（和《福尔摩斯》学的）
- 胜利后会露出一丝得意的笑

**成长轨迹**：
1. **失明期**：意外失明，获得《心眼经》，修炼"明心"感知情绪
2. **恢复期**：眼睛恢复，但隐藏心眼能力，律所实习
3. **崛起期**：心眼突破"照见"，开始介入大案
4. **巅峰期**：突破"天眼"，与四大家族正面对抗
5. **圆满期**：心眼合一，击败终极BOSS

**金句**：
- "你们以为用钱和权就能践踏法律？"
- "我虽然能看穿人心，但最想看穿的是你们的底线在哪里。"

## 二、三位女主人物卡

### 女主A：林知念

**定位**：正牌女友 · 温柔港湾

**基本信息**：
- 年龄：23岁
- 身高：165cm
- 体重：48kg
- 职业：自由插画师（在家工作）
- 家庭：书香门第，父母是中学教师

**外貌特征**：
- **整体**：清新自然型美女，属于耐看的小家碧玉型。脸型是标准的鹅蛋脸，线条柔和，看起来毫无攻击性，让人自然放松。
- **眼睛**：杏眼，眼角微圆，瞳孔大而清澈，像小鹿一样干净。眼尾微微下垂，笑起来弯成月牙，显得温柔无害。眼神里总是带着淡淡的笑意，没有防备。
- **眉毛**：弯月眉，眉毛细软，颜色浅淡，无需画眉天生好看。
- **鼻子**：鼻梁不算高挺，但鼻尖小巧，微微上翘，有几分娇憨。
- **嘴巴**：嘴唇薄厚适中，颜色浅粉，唇珠明显。不笑时嘴角微微上扬，是天然的微笑唇。
- **皮肤**：皮肤极好，牛奶肌，冷白皮，脸上有淡淡的红晕。
- **头发**：黑长直发，发质软，发量多，发尾微卷。经常扎成松散的马尾，几缕碎发垂在耳边。
- **身材**：165cm/48kg，体型纤细，骨肉匀停。肩膀窄，锁骨明显但不骨感。胸围约B，腰细，臀部微翘，属于健康的瘦。
- **穿着**：休闲舒适为主，棉麻材质。喜欢浅色系：米白、淡蓝、浅粉、鹅黄。夏天穿碎花裙，冬天穿高领毛衣配围巾。
- **标配**：不化妆或淡妆，只涂淡粉色唇膏。戴黑框眼镜（装饰用，度数很低）。

**性格特点**：
- 温柔善良，善解人意
- 不争不抢，但有自己的底线
- 有点小迷糊，生活能力一般
- 内心坚强，在官平最艰难时给予温暖

**与官平的相处模式**：
- 官平失明时每天来照顾：做饭、聊天、读法律书籍给他听
- 官平恢复后继续保持邻居关系，日久生情
- 吵架时官平用读心术能提前知道她为什么生气
- 是官平的"能量充电站"，回家就能放松

**人物小传**：
```
从小就是"别人家的孩子"，听话乖巧。
但内心渴望自由，厌倦了一眼望到头的生活。
遇到官平后，被他的正义感和热血吸引。
在官平失明最无助时悉心照顾，却从不越界。
等官平眼睛恢复，她反而有些不自在——
"你是不是……不需要我了？"
官平用读心术听到了这句话，一把拉住她的手：
"我需要的是你，不是护工。"
```

**结局**：与官平结婚，生有一个女儿（取名"官念"，纪念这段感情）

**出场章节**：第一卷第3章（官平失明后搬到新小区，她是隔壁邻居）

### 女主B：苏锦瑟

**定位**：贤内助 · 商界女强人

**基本信息**：
- 年龄：27岁
- 身高：170cm
- 体重：52kg
- 职业：锦盛集团副总裁（家族企业）
- 家庭：豪门千金，但父母关系冷淡，独立性强

**外貌特征**：
- **整体**：美艳动人型御姐，气场强大的熟女。脸型是精致的倒三角脸（网红级V脸），轮廓分明，高级感十足。五官精致得像画报里走出来的人，让人不敢直视。
- **眼睛**：狐狸眼，眼尾上挑，眼角尖锐，瞳色偏浅（琥珀色）。眼神或妩媚或凌厉，能在媚态和杀气之间切换。睫毛浓密，不需要睫毛膏。自带眼线效果。
- **眉毛**：上挑眉，眉峰明显，眉毛颜色略深于发色，显得英气。
- **鼻子**：鼻梁高挺，鼻尖精致但不尖锐，侧面线条完美。
- **嘴巴**：嘴唇饱满，唇色偏红（无论是否涂口红都显得气色好）。唇形是标准的微笑唇，但不笑时显得高冷。
- **皮肤**：冷白皮，肤质细腻，自带磨皮效果。五官明艳，气场全开。
- **头发**：棕色大波浪长发，发质极好，蓬松有光泽。发量大，约到腰。
- **身材**：170cm/52kg，魔鬼身材。前凸后翘，S型曲线惊人。胸围约E，腰细（60cm），臀围约95。身高170已经很高挑，再加10cm高跟鞋，气场直接两米八。
- **穿着**：高端定制职业套装，或者名媛风礼服。主色：黑、白、红、金。奢侈品堆砌但不显俗气，每一件都穿出价值感。
- **标配**：Dior真我香水，梵克雅宝项链，10cmJimmy Choo高跟鞋。

**性格特点**：
- 外表：高冷美艳，御姐范儿，气场两米八
- 内在：内心柔软，缺乏安全感，渴望真爱
- 习惯：说话直接，喜欢调戏男主
- 弱点：面对真心会害羞，但绝不承认

**与官平的相处模式**：
- 第一次见面就调戏："哟，这律师有点意思，要不要当我男朋友？"
- 官平用读心术发现她其实很紧张
- 官平帮她打赢官司后，开始正眼相看
- 多次"英雄救美"后关系确定
- 她是官平的"资金弹药库"，多次在关键时刻砸钱帮忙

**人物小传**：
```
豪门千金，从小被当继承人培养。
外表光鲜亮丽，其实内心孤独。
谈过几次恋爱，对方要么图钱，要么被她的气场吓跑。
遇到官平时正在打一场商业纠纷案。
官平在法庭上的表现让她眼前一亮：
"这小子……有点东西。"
她开始主动接近官平，却总是用调侃掩饰真心。
直到那次被绑架，官平单枪匹马来救她——
"你是第一个……愿意为我拼命的人。"
```

**结局**：与官平确立关系，生有一子，是官平事业的最大金主

**出场章节**：第二卷第5章（官平接手的第一个商业案子）

### 女主C：沈夜

**定位**：暧昧杀手 · 暗刃守护

**基本信息**：
- 年龄：25岁
- 身高：168cm
- 体重：52kg
- 职业：杀手（表面身份：私人保镖）
- 家庭：孤儿，被王家培养成杀手

**外貌特征**：
- **整体**：冷冽神秘型，身手了得的御姐杀手。脸型偏瘦，线条硬朗，不笑时拒人千里。眼神能杀人，沉默时自带压迫感。
- **眼睛**：丹凤眼，眼型细长，眼尾上挑且尖锐，瞳孔漆黑。眼神冷冽如刀，不带感情。但遇到官平时，眼神深处会有一丝微不可察的波动。
- **眉毛**：剑眉，眉尾锋利，眉毛浓黑，眉骨明显。不化妆，眉毛浓密得自带气场。
- **鼻子**：鼻梁高直，鼻尖略尖，侧面看线条凌厉。
- **嘴巴**：嘴唇薄，唇色偏淡，不笑时抿成一条线。咬字清晰，语调平稳，几乎没有情绪起伏。
- **皮肤**：肤色偏白但偏黄，是那种不太健康的冷白。不是白得发光，而是苍白。常年不见天日的工作环境。
- **头发**：黑色短发，长度及肩，发质硬，干净利落。没有染烫，刘海斜分，露出眉骨。
- **身材**：168cm/52kg，矫健有力型。身体线条流畅，没有一丝赘肉，肌肉紧实但不过分健壮。胸围约C，腰细，腿长比例好。身上有多处旧伤疤（任务留下的勋章）。
- **穿着**：黑色系为主。紧身T恤，工装裤，军靴。或者黑色风衣，墨镜。低调不显眼，方便行动。
- **标配**：左手腕的绷带（遮盖伤疤），腰间的匕首，暗器藏在袖口。

**性格特点**：
- 外表：冷酷寡言，执行任务时心狠手辣
- 内在：内心有伤痛，渴望自由和救赎
- 执念：复仇，为被王家害死的家人报仇
- 变化：遇到官平后开始质疑自己的存在意义

**与官平的相处模式**：
- 最初是王家派来的杀手，目标是杀掉官平
- 官平用"照见"能力读到了她的内心，发现她的悲惨过去
- 官平没有揭穿，反而帮助她复仇
- 她被策反，成为官平背后的暗刃
- 两人之间有暧昧张力，但始终未明确关系

**人物小传**：
```
十二岁那年，王家的人杀了她全家。
她被掳走，培养成杀手。
她不知道自己还能不能算人。
杀人无数，双手沾满鲜血。
直到那次任务，她要杀的人是官平。
她潜入金诚律所，准备动手。
但官平突然说："你父母的仇，你想不想报？"
她震惊："你怎么知道……"
官平："我能看到。你的眼睛里全是恨，但那不是对我的恨。"
她第一次在目标面前流下眼泪。
后来她成了官平最隐秘的刀。
他说："你的手可以沾血，但心不用。"
她知道这辈子还不清他。
但她愿意用命去守护他。
```

**结局**：身份始终在暗处，多次救官平于危难，结局暧昧（开放式结局）

**出场章节**：第二卷第12章（王家派来的杀手，第一次刺杀官平）

## 五、人物设定要点

### 三位女主的差异化设计

| 维度 | 林知念 | 苏锦瑟 | 沈夜 |
|------|--------|--------|------|
| 性格 | 温柔治愈 | 霸道御姐 | 冷酷神秘 |
| 身份 | 邻居普通人 | 商界女强人 | 杀手保镖 |
| 外貌 | 清新鹅蛋脸 | 美艳V字脸 | 冷冽削瘦 |
| 眼睛 | 杏眼温柔无辜 | 狐狸眼妩媚凌厉 | 丹凤眼冷冽如刀 |
| 身材 | 纤细邻家型 | 魔鬼S型 | 矫健有力型 |
| 感情 | 日久生情 | 主动倒追 | 暗中守护 |
| 作用 | 温暖港湾 | 资金支持 | 暗中护航 |
| 结局 | 结婚生女 | 结婚生男 | 暧昧不明 |

### 配角设计要点

- **周正**：严师出高徒，但也会护短
- **林小溪**：情报担当，搞笑担当
- **陈浩**：有背景的反派，让官平前期有对手
- **四大家主**：每个都有独特外貌和性格，不是脸谱化"有钱人"
- **外婆**：精神支柱，代表官平的来处
- **老张**：底层小人物代表，官平守护的初心
