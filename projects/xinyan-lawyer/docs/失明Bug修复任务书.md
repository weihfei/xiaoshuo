# 《心眼律师》前10章失明Bug修复任务书

> 执行方：工作室章节写手
> 版本：v1.0 | 日期：2026-04-29
> 说明：官平失明后，"闭眼/睁眼/望/看到"等视觉词需替换为感知类表达

---

## 修改原则

| 原表达 | 替换为 |
|--------|--------|
| 闭上眼睛/闭眼 | 沉心静气 / 屏息凝神 / 定了定神 |
| 猛地睁开眼 | 意识猛地一振 / 精神一凛 |
| 慢慢睁开眼睛 | 慢慢回过神来 |
| 睁开眼睛——虽然看不见 | 直接写动作，如"转向声音的方向" |
| 望着XX | 用明心感知 / 侧耳倾听 / 将感知投向XX |
| 失明后"看到" | 感知到 / 意识到 / 判断出 |
