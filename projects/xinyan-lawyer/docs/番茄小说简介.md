# 《心眼律师》作品简介

---

## 简介

天海市，一场车祸，他失去光明，却意外激活了上古秘法——

《心眼经》！

失明又如何？他能"看"透人心，"听"出谎言，"感"知真相。

法庭之上，当对手还在编织谎言，他早已读出破绽。
敌人以为他是软柿子，却不知瞎子的心比任何人都看得清楚。

他是官平，一个身负灭门血仇的实习律师。

**金手指**：心眼——失明后觉醒的超级感知能力，能读心、感知情绪、扫描隐藏信息

**三个女主**：
- 林知念——温柔邻居，失明时不离不弃，心灵最纯净的存在
- 苏锦瑟——霸道女总裁，御姐外表下的脆弱只对他一人展露
- 沈夜——神秘杀手，冷若冰霜却在他面前藏不住情绪波动

**三条主线**：
- 案件线——劳动纠纷、商业大案、刑事要案，案案精彩
- 身世线——二十年前官家灭门惨案，幕后黑手逐渐浮出水面
- 势力线——四大家族盘踞天海，陈家、王家、郑家、李家，个个来头不小

**核心爽点**：
- 法庭读心打脸——"我眼睛看不见，但你的心，我一览无余"
- 弱势群体逆袭——农民工讨薪、小民对抗权贵，看他如何用法律武器逆风翻盘
- 暧昧三角——霸道御姐疯狂撩，他却只对邻家女孩心动

失明不是终点，而是觉醒的开始。

当心眼洞开，世界在他耳中清晰无比。

在这座吃人的城市，他要用法律+金手指，杀出一条血路！
